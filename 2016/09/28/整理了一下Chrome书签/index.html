整理了一下Chrome的书签

发表于 2016-09-28 更新于 2016-10-06 分类于 分享 本文字数： 805 阅读时长 ≈ 3 分钟

今天整理了一下Chrome书签，发现近几年竟然存下了数十个分类、数百个书签，着实还有点震惊。虽然已经远离广告行业，但是还是能看得出我当年还是挺用功学习的。但一个个理下来，现在对我来说比较有收藏意义的已经少之又少。花了近一个小时温习、重新归类以后，目前只剩下10来个分类，数十个标签而已。有时候整理东西，丢弃东西，确实能给人带来一些愉悦感，如同搬家时丢东西一样，感觉整个人都轻松了许多。整理下来书签栏干净而又有条理，顺便温习了一下自己以前关注多的文章或内容，不亦乐乎？

目前分类：

  • 学习&资料
    • 有意思的网站 一些发现的比较有意思的创业项目
    • 设计相关 关于UI、交互设计的优秀作品或者网站
    • 技术学习 主要是Python相关网站和资料
    • 网络课程 有趣的网络课程，比如PPT制作、做菜、音乐等
    • APP开发&上线相关 了解APP开发上线的一些知识补充
    • Hexo 本网站用的主题，一些设置或者应用拓展说明
    • Mac 提高Mac使用效率，一些应用或者技巧收集
    • 数据可视化 关注数据可视化领域的产品更新
    • 产品 关于做产品经理的一些好文或经验分享
    • 股票 关于量化交易的一些信息收集，但是还没有用上
    • 网站分析资料 前三年工作所关注的内容，未清理，情怀
  • 关注订阅 一些知名的博客或媒体
  • 在线工具 在某些方面曾经帮我提高效率的小工具
  • 常用平台 近几个月用的比较多的在线平台
  • Adchina 之前公司的系统地址，还保留着
  • 当前工作 目前公司内部的网站或者系统

其中常用平台 这里的一些给大家分享一下，因为目前我用的Mac，APP Store上很多软件又非常贵，于是乎我收集了一些在线的应用，无须安装软件又能高效解决问题。作为我最常用的几个（职责：PM），不管你是做产品经理、还是项目经理、或者是策划，相信都会对你有帮助，推荐给大家：

名称 描述 推荐指数
墨刀 移动APP原型设计，PM必备，功能强大也非常易用 ⭐️⭐️⭐️⭐️⭐️
百度脑图 脑图在线制作、保存、导出，操作便捷 ⭐️⭐️⭐️⭐️⭐️
Processon 在线流程图制作，还下什么Visio，就是导出结果会略丑 ⭐️⭐️⭐️⭐️⭐️
雁阵 在线甘特图，项目计划/进度管理，要啥project，要啥EXCEL ⭐️⭐️⭐️⭐️⭐️
麦克CRM 在线信息表单，设计表单发布，收集数据 ⭐️⭐️⭐️⭐️
Worktile 在线协作、任务分配管理平台，相比于Teambition和tower等系列，这个使用起来体验最佳 ⭐️⭐️⭐️⭐️

希望这些能对你有帮助，如若有其他好的产品或者平台，也希望能推荐给我，在此谢过！
