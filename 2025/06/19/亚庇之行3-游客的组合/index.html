亚庇之行3-游客的组合

发表于 2025-06-19 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

这次出去玩注意到一个很有意思的现象，就是游客的组合呈现出一个很特别的比例。据我观察，这次去亚庇游客大致的比例如下：

  • 家庭组合，大约占5%
  • 夫妻/情侣组合，大约5%
  • 剩下的90%，为【小姐姐 x N】，其中N∈[1,5]
  • 男 x N，只看到一组

这几组人的外在情绪表现也很有特点：

  • 家庭组合，老人带着小孩跑，年轻夫妻两个出去玩项目，还算和谐

  • 夫妻/情侣组合，女方比较享受过程，打扮地漂漂亮亮拍照；男方脸上普遍写着“疲惫”二字

  • 小姐姐组合，活力四射，仿佛这个世界上的美景就是为他们准备的。她们好像有花不完的精力，可以在一个地方凹两个小时造型，就为了那一张满意的照片。在他们脸上看不到焦虑，有的只是“享受当下”的洒脱。

  • 男生组合，是我看到印象最深的那一组。那天正值落日时分，远方的岛屿、天上的云以及缓缓落下的红日构成了一幅绝美的风景，周围的人大部分都在拍照拍个不停，唯独这几位，端坐在岸边的石头上，静静地观赏，享受这难得的瞬间。那一刻，我看到了我心中旅游该有的样子：不是为了特种兵一般段时间内把各种景点刷一遍、东西全吃一遍、照片拍一堆留作纪念，仿佛单位时间内这些东西刷的越多，这趟旅行的性价比越高，连玩乐享受都匆匆忙忙地充满算计…真正的旅游应该是抛开平日里的那些紧张、压力或者消极的情绪，融入这片大自然馈赠与你的安宁。那个时候脑子里就是东坡先生的这句：

    惟江上之清风，

    与山间之明月，

    耳得之而为声，

    目遇之而成色，

    取之无禁，

    用之不竭，

    是造物者之无尽藏也，

    而吾与子之所共适

  • 平时都懒得贴照片，今天贴两张纪念一下
