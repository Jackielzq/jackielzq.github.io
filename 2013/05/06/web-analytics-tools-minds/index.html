网站分析的工具与思想

发表于 2013-05-06 分类于 网站分析 Waline： 本文字数： 1.6k 阅读时长 ≈ 6 分钟

俗话说“工欲善其事，必先利其器”，在网上看到许多这样的问题：“网站分析需要掌握哪些工具？用哪些工具比较好？”，这些问题就类似于“我要学编程 ，学哪种语言比较好？”。以前我也问过这种问题，结果被一位朋友批评说：“哪个工具不重要，最重要的是思想！”。毕业时在找工作的时候也困惑过，因为看到职位描述，总觉得为什么这个工作要求要懂这么多东西，会那么多工具?一时间感觉有点迷茫而又亚历山大。但是实际上有很多是用不上的，有很多是不常用的，所以掌握好最常用最基础的部分是至关重要的，而进阶的部分虽然说在某些时候能大大提高你的效率，但是这一部分通常来说要难很多，同时也会耗费更多的时间。时间是很宝贵的，所以还是尽量集中在更重要的事情上把。

现在有一个普遍认同的说法：“分析师的水平=10%的工具+90%的人”，人的作用是至关重要的，也就是工具之外的“思想”。到现在为止我越来越认同这个说法了，更大了看，每个公司都是由许多人组成，每个职位，你要面对的都是人。没有不靠谱的公司，只有不靠谱的人。就像金庸笔下的独孤求败，“三十岁前所用，误伤义士不祥，乃弃之深谷。重剑无锋，大巧不工。四十岁前恃之横行天下。 四十岁后，不滞于物，草木竹石均可为剑。自此精修，渐进至无剑胜有剑之境。”近几年大名鼎鼎的捷克流浪汉摄影家提奇Miroslav Tichy和我最喜欢看的一档节目《荒野求生》中的贝尔.格里尔斯（人称贝爷），都属于现代的这号人物（如下图），一个拿着垃圾堆里拼凑出来的相机完成了许多精美的摄影创作，另外一个几乎赤手空拳，凭着丰富的经验经受住了各种严酷环境下的生存挑战。

Miroslav Tichy 米罗斯洛夫-蒂奇

[caption id=”attachment_205” align=”aligncenter” width=”557”] 贝尔.格里尔斯[/caption]

说到《荒野求生》这个节目我突然觉得我有好多话可以说，这里概括成三点吧：一是我从小到大对野外的自然环境都非常的好奇，它极大得满足了我的好奇心；二是他在中途随机应变、灵活引用现有资源的思想让我非常佩服，同时也有一些感悟；三是现在不管看到别人吃什么奇葩的东西，都不会再有恶心的感觉了。

现在搜索引擎和微博等资源方便了信息的传播和共享，但是同时也有一个问题：对于初学者来说，许多内容不好判断。为什么这么说呢，因为我所知道的许多大牛通常都是很低调的，他们一般不在公开场合露面，也不写博客不发微博。反而许多“号称”为专家的人借着这种潮流大行其道，比如说微博微信刚出来的时候，许多人就各种开讲座大谈特谈“微博的盈利模式”，各种忽悠。要真如你们讲的一样微博还要耗到现在跟阿里勾搭上了么。而这个时候新人们就很难去辨别这些信息的价值了，因为看起来一套一套的，好像确实有道理，但是实际上看了等于是浪费时间。当然，还是有很多非常有用的阅读资料的，还是需要关注一些靠谱的人，因为他们转发或者收藏的东西，通常也是靠谱的。

那么对于每个人来说，10%+90%的比率都是不同的。首先应该对自己定个位，在这个行业里面，我是排在哪一个阶层的，这个阶层最重要的需要掌握的东西是什么？同样是10%，大牛的总数是1000，你是100，这是不能比的。比如工具，他们的10%是EXCEL、SPSS、VISIO、SQL、PPT等其他更多高级工具的熟练或者精通，而你却是EXCEL常用技巧的熟练使用。思想那方面就更难衡量了，因为这个没有办法量化。这两者都是需要跟着时间一起继续扩充的，而最好的办法就是实践。这也是为什么说经验比学历重要了，因为他们是最实用的。这里分享一首诗吧，初中时候做奥数的时候看到的，现在还记得–“埋头苦干是第一，熟能生出百巧来。勤能补拙是良训，一分辛劳一分才。”（猜猜谁是作者）

最后回到我们主题-网站分析上来讲，工具的实用是为了满足需求，而不是为了学习这门工具。当你需要什么功能的时候，尽可能精确的总结成几个关键词，然后问问百度或者google吧，我感觉99%以上都会找到答案的。我的EXCEL和HTML、CSS、Javascript知识的丰富和补充就是这么来的。开始只大概地学一下基础的东西，知道原理以后，其他具体的写法、功能、函数名称什么的都可以上网找到，这些功能如果常用的话，用个几次就熟练了。至于思想，则确实是需要沉淀的，到现在为止我还是有些浮躁，许多书看了也没有网上别人说的感悟这么多，可能是领悟能力有限吧。形象思维比较适合我，一抽象起来就犯困。但是看书是绝对没有坏处的，这就跟听音乐一样，以前你听不懂，但是某一刻你突然明白这首歌说的什么，仿佛它说的就是你的心声，这，也许就是一种领悟吧。
