Wordpress设置首行缩进

发表于 2014-04-24 | 分类于 wordpress |

做为强迫症患者，怎么能容忍发布的文章首行未缩进？如果你还在发布文章的时候在每段段首狂敲空格或者Tab键的话，那别人看你的空格键也许会把你当做高端的劲舞选手。所以还是动用一分钟设置一下把（当然，前提是你的主题没有这功能)…

  1. 打开wordpress后台，选择外观-编辑

    2. 找到style.css进行编辑

    3. CTRL+F (Mac为command+F)查找 “.p” 也就是段落的样式名称，在下面添加相应代码，效果如下：

css p { line-height: 24px; margin-bottom: 15px; font-family: 'Lato', sans-serif; font-size: 16px; text-indent:2em; #这段为加上的代码 }

完成，目前你所有的文章首行都是默认缩进了~
