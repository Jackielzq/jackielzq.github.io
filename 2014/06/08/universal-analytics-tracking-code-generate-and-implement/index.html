Universal Analytics基础配置(二) 代码的生成与部署

发表于 2014-06-08 更新于 2016-09-21 分类于 Universal Analytics

上篇文章中，我们已经设置好Account，接下来来到Property(网站资源)板块，选择Tracking info(监测代码) - Tracking code，你会看到如下形式的代码：
&lt;script&gt;

(function(i,s,o,g,r,a,m){i[‘GoogleAnalyticsObject’]=r;i[r]=i[r]||function(){

(i[r].q=i[r].q||[]).push(arguments)},i[r].l=1*new Date();a=s.createElement(o),

m=s.getElementsByTagName(o)[0];a.async=1;a.src=g;m.parentNode.insertBefore(a,m)

})(window,document,’script’,’//www.google-analytics.com/analytics.js’,’ga’);

 

ga(‘create’, ‘UA-28253637-2’, ‘jackie.ms’);

ga(‘send’, ‘pageview’);

</script>

打开网站后台(我这里是Wordpress),找到仪表盘 - 外观 - 编辑，在界面右侧选择header.php,找到</header>标签，在这个标签前面粘贴上这段代码，保存修改，整个账号生成-配置-部署监测的过程就完成了。
