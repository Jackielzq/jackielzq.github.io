Mac端使用Homebrew Cask 快速安装软件

发表于 2014-03-21 更新于 2016-09-21 分类于 mac Waline： 本文字数： 671 阅读时长 ≈ 2 分钟

在Mac端安装软件，用到Home brew Cask这个神器，像程序员一般装软件，高端大气，安装常用软件只需要敲一行代码就搞定了，不用担心各种乱七八糟的下载网站给你带来的不爽。 安装步骤： 一、安装Xcode，在AppStore里直接搜索安装即可 二、安装Homebrew - 打开控制台(Terminal)，在命令行里输入

ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

敲回车，坐等十几秒就OK了

三、安装Homebrew Cask，继续在Terminal里输入 
brew tap phinze/homebrew-cask && brew install brew-cask

OK,安装成功，然后就可以轻松下载各种软件了,但是如果你不确定这个软件是否存在的话，可以先搜一下，像这样：

如果搜到了结果，那么接下来再输入一行命令就可以直接安装了：

brew cask install xunlei

下面列举几个比较常用的软件：

- Chrome，Google浏览器

brew cask install chrome

- 迅雷，这个自然不用多说，大家都懂的

brew cask install xunlei

- QQ，默认安装的是mac版，很贴心

brew cask install qq

-虾米，音乐不可少，之前的版本bug比较多，但是最近更新了，很好用

brew cask install xiami

-Alfred，关于这个神器，详见知乎这个帖子借助 Alfred 2.0 的 Workflows 功能可以做哪些好玩的事情？

brew cask install alfred

-Dropbox 云盘，简洁好用，比国内的各种网盘给的空间要小，但是体验很好，很干净！

brew cask install dropbox

-Movist Mac上没有暴风等播放视频的软件，但是电源也得看啊，Movist绝对是首选！

brew cask install movist-fork

-Wunderlist 任务记事本，简单高效，界面相当漂亮

brew cask install wunderlist

-Evernote 记录笔记，收集资料，比浏览器的书签要靠谱

brew cask install evernote

-Transmit FTP工具，类似于Windows上的FlashXP

brew cask install transmit

-Unarchiver 文件解压工具，默认系统是不自带解压工具的

brew cask install the-unarchiver

**-Mou  **Markdown 文字编辑器，普通青年瞬间变文艺青年，详情请戳这里

brew cask install mou

-Sublimetext 最好用的代码编辑器

brew cask install sublime-text

-ColorSchemer 提色器，迅速提取你所需要的颜色RGB值，做PPT的话，精准

brew cask install colorschemer-studio

另外还有Brewcask不支持的应用，在这里一并介绍了：

-搜狐视频 -爱奇艺 -微信Mac版 -MemoryDiag 查看机器内存使用情况 -Cleanapp 卸载软件工具 确实比Windows方便很多把，谁用谁知道
