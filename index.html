蔡澜先生

发表于 2025-06-29 分类于 记录 本文字数： 652 阅读时长 ≈ 2 分钟

前几天，香港的“四大才子“最后一个，蔡澜先生走了。之前对他不算太了解，只是刷到过有关他访谈的几个视频，也去吃过他开的港餐店。虽然了解不多，但是仅从他有限的言语中，我便喜欢上了这个有趣的老头子，颇有当年苏东坡的味道，但相比之下，他过得应该比苏东坡要开心圆满得多吧。毕竟前者经历了众多波折，很多时候是在苦中作乐，而蔡澜先生则更加顺风顺水，单纯快乐地活着，活成了许多人理想中的样子。

去翻了翻关于他的语录：

1.人生的意义，就在于吃吃喝喝。 （相比于余华的活着，吃吃喝喝这四个字可太有意思了）

2.压力最大的敌对头，是好玩，什么东西都把它变成好玩，压力自然减少。

3.平稳的人生，一定闷。我受不了闷，我决定活得有趣。（这也是我的人生态度）

4.除了肉体的痛苦，所有所谓的痛苦都只是你想出来的。（许多人都沉溺于自己想象中的痛苦无法自拔，真的应该学学）

5.一个人来到这个世界，要爱最可爱的，最好听的，最好看的，最好吃的，有趣地活着。

6.想做人豁达，首先得在年轻时拼命，什么都做，什么都学，不埋怨。学习多了，就有信心。

7.尽量地学习，尽量地经历，尽量地旅游，尽量地吃好东西，人生就比较美好一点，就这么简单。

8.人生需要有一技之长，让自己相信可以靠这个为生。对自己有自信，确定任何事情都一定要做好；多学一点，自信就会一直增加。

……

如果是某些所谓的专家或者德行不匹配的人说出这句话，我会感觉鸡汤味很浓。但是像他，像余华这样的人说出来，你会觉得非常的对味，言行合一的人确实很有人格魅力，哪怕他是个“反派”。

当然，我印象最深，并且我常拿来教育我们公司年轻单身而且怂得不敢追女生的小同事的，还是那句——“丑的照杀！”

相比于前些年吴孟达“达叔”的离开让大家惋惜，蔡澜先生的离开，我没有那种悲伤的情绪，反而更多的是羡慕，羡慕他一生如此潇洒~

前几年有两档节目比较火《中国有嘻哈》和《乐队的夏天》，那时也恰好看了这两个节目，看得还挺有味道。这几年这两个圈子也发生了许多变化，我发现这两个群体还真的有许多共同点：

1、原本大部分都不挣钱，节目火了以后身价大涨，赚得盆满钵满

2、都挺有个性，相当一部分选手觉得自己就是最屌的，觉得别人不如自己

3、嘴里都喜欢喊些口号，不管信不信，反正就喜欢喊，比如“homie”、“respect”、“dope”，摇滚的喜欢“rock N roll”等等

4、大部分唱功都不行，甚至可以用“稀烂”来形容，好歹也是要唱歌的啊，说唱两个字里好歹有一个是“唱啊”，结果五音不全的都上来了，连基本的音准都做不到，丢人丢到姥姥家去了

5、嘴里喊着要“real”，结果一上节目看到金主了，一个个比谁都客气，点头哈腰敬酒，反正挣钱嘛，不寒碜

6、平均学历偏低，写词功底太差，真正有生活阅历的少，像玩说唱的许多没有丰富的生活经历，歌词里总是陈词滥调，整体无非就是《我很强and你不行》《车子房子票子马子》，还有一些生活里满是法律禁止的东西，他们能写，但是没法播，所以你指望这些人能写点什么东西出来

自己常听的歌单，仍然还是10多年前的歌，这么些年来，曲库基本没有多少新增。我相信许多人也是和我一样的情况，因为确实国内乐坛很难再重现2000年左右那种神仙打架，金曲频出的黄金时代了。

我曲库里大部分新增的，都是纯音乐指弹曲，我觉得它更为纯粹，自己更能体会到它要表达的东西；相比之下，那些乱七八糟的所谓“音乐”，我只觉得它们吵闹。

山姆-信任危机造就的品牌

发表于 2025-06-24 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

最近山姆超市相关的新闻挺多的，这种会员制超市，一年收一个会员200多会费，光会员费一年就能收20亿，另其他超市羡慕不已。我周围也有挺多人是它的会员，包括小区业主群也经常有人组织山姆团购。他们有挺多明星产品确实挺不错的，选品比较严格，部分产品我也很喜欢，它算是我比较有好感的一个品牌吧。

我觉得它和胖东来挺像，它们的成功，真的得靠同行的衬托。同行超市经常出现什么过期食品啊、或者一模一样的东西卖的比便利店比网上贵，或者出现其他各种各样的状况，导致大家出门购物，在绝大多数场合，都得防着卖方一手。称东西有没足斤足两、买的牛奶有没有过期、买的肉有没有掺淋巴、买的品牌日用品有没有比网上贵很多…只要你一不留神，就会被套路，当冤大头吃哑巴亏。

而其实你要让客户喜欢你真的很简单，前人早就总结好了那几个字：“物美价廉“、“货真价实”，都是这么简单的道理。但是现在大部分企业就是做不到，究竟是什么原因？因为那些人眼中只有怎么更多地节约成本，怎么把同样的东西卖出更多的利润，并不是以客户为中心的，所以才能干出那么多下作的事情。但我觉得背后更多的原因，就是“监管不力”“制法不严”，违法的成本太低，处罚太轻，导致很多人干出这种昧良心的事情都感觉不到一丝“铤而走险”的感觉，反而觉得你不这么干就是“蠢”。就比如在印度，一群人QJ一个女人，结果法官判嫌疑人给受害者道歉，这样的法律管辖的环境，XQ案件只会多不会少。

法律法规的纵容，造就了这样的一个商业环境，而这样的环境也使得消费者对大部分企业失去了信任，而在这样的环境中依然选择坚持初心，做好产品服务的企业被同行衬托地光鲜亮丽。也许山姆的老板开始也是战战兢兢地不知道能不能竞争得过对手，结果没想到对手直接把客户往自己怀里送…

男人都得会点修理技能

发表于 2025-06-23 更新于 2025-06-22 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

最近这个周末由于下午就没带娃出去玩，在家干了几个事情: 疏通管道、洗空调，感慨了一下，作为家庭里的主要劳动力，真的需要具备一定的修理技能。

去年厕所门锁坏了，买了个锁50，上门换锁的说要100，我不服，网上搜了个教程，自己把锁给装好了

上个月，房间里吸顶灯坏了，由于以前没接过，本来想找个维修工，但是他一个报价直接比我买的灯芯还贵，还嘲讽我“嫌贵自己装吧”。我这个人最受不了嘲讽，硬是抖音搜了个教程，自己也装好了

家里厨房的管道老是几个月就一堵，还返水，自己尝试买了管道疏通剂，顺了几次，效果也还不错，毕竟这种事情，维修工一上门就是200。这周末实在是不行了，所有招都用上没办法请了业主群邻居推荐的师傅，花了150，带机器把管道里的油垢打了个底朝天

洗空调这个事情做了好多年了，我自己本来缺乏这样的意识，我爸也是，家里的空调十几年都没洗过，秉着“能用就行”的原则，懒得动它。但是我老婆爱干净，弥补了这一缺陷，每隔几个月就要求我洗一次空调。这次不洗不知道，一拆出来，客厅空调的过滤网满满一层黑色的霉菌，甚是恐怖。经过这次一清洗，家里四台空调感觉都脱胎换骨，整个家都感觉干净了。

其实这些事情请人也不贵，也不是舍不得这些钱。只是因为之前遇到的某些维修工真的素质太差，直接跟抢钱差不多，我只是不想被这些人拿捏。某次通管道通到一半，地面上已经被弄得到处是污垢了，他临时说要加钱，开个一个机器，开一次100，一次不通开2次要200，我直接让他出去了。这种人就跟在手术台上的医生问你，现在你这切开一半了，你这个结石拿不拿出来，不拿我就缝回去，拿的话一克500，拿不拿？

换一个角度想，其实自己多掌握点这种家庭常用技能也挺有用的。小时候看到爸爸会换灯泡修水管修电表就觉得特别厉害，对老爸很崇拜。以后希望儿子在心里也能种下一颗这样的种子：希望自己以后可以像爸爸一样这么厉害！

亚庇之行4-经济与生态

发表于 2025-06-22 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

旅游业让许多处在偏远地区的人增加了大量的收入，也让从未到达过该类地方的游客人生有了不一样的体验。本来这是一件很好的事情，但是总有那么一小波人，因为素质太差，仅为了自己一时的便利，对当地生态造成了极大的破坏，也让后来的人体验大打折扣。

我个人可以拍着胸脯说，我从来没有做过这种事情，垃圾肯定都会找到垃圾桶再扔。如果没有遇到，那我就会一直捏着，直到遇见为止，所以我认为我有资格来说这些事。我一直对这种乱扔垃圾的行为都是深恶痛绝，即使是关系不错的朋友，如果发现TA做了这种事情并且丝毫没有觉得愧疚，那么TA在我心里，他已经被列入待定区了。（当然，被我说动迷途知返的还能给机会）

这次出行有几个场景让我觉得很失望：

1、某天的浮潜安排，第二个浮潜点是看珊瑚，导游说那里的珊瑚品种繁多、颜色艳丽，特别漂亮。但是我去看了之后，只有大量的“残骸”，珊瑚几乎已经死尽，连周围的鱼都懒得过来这里觅食。看完我大失所望，没几分钟就回船上去了。

2、一个拍照特别好看的小岛，沙质松软细腻，走起来特别舒服。但是在休息处稍微往前走100米，隔壁的沙滩上，堆着满是人类留下的塑料垃圾。记得贝爷有一期节目，在一个荒岛求生，捡到了许多塑料垃圾并对其求生起到了很大的作用，那时的他感慨“不知道高兴还是不高兴，不管有人或者没人的地方，一定会有人类留下的垃圾，对于我求生而言是好事，但是对于整个环境来说不是“。

3、回到酒店，酒店自带一个环形沙滩，并把它围起来供游客浮潜，鱼确实也很多很好看，但是隔几步就是一块垃圾飘过来，周而复始，有许多可以看到是非常新鲜的包装，显然是酒店住客刚扔的，我不知道他们为什么能这么心安理得，那时我心里想的是，这个国家的法律能不能严一点，跟新加坡一样，把这些人吊起来抽几鞭子…

另外，还想到某一次国庆开车回老家，平时6小时的路程，扎扎实实用了19个小时才到家，回去的这趟路程只觉得累，并没有太多在意。但返程的时候看到路上真是触目惊心：高速隔离带、应急车道附近，几十上百公里绵延不绝全是垃圾，各种包装袋塑料袋、烟头等等。当时我真的是很想骂娘，在路上堵着吃点东西产生点垃圾很正常，所以每次我都会备着塑料袋，去服务区或者到了地方再扔。但是这么多的人，打开车窗随手就是那么一扔，毫无愧疚感，有些甚至觉得自己还挺潇洒。他们很多还是拖家带口，上有老下有小，直接在自己小孩面前做出这种事情，相信小朋友在学校也学过别随地乱扔垃圾这种事情吧，这样给自己小孩树立了一个什么样的榜样？

中国这些年来环保取得了很不错的成绩，城市乃至乡村的面貌都焕然一新，比以前干净了很多。我所看到的是大众普遍的环保意识提高了，加上数以万计年长的清洁工们每天早晚不停歇的劳动换来的。但是，很多人在保护自己的生存环境上是可以的，但是一到了外面的世界，又回归了本性，“反正污染也有人治理”，在这方面，还有很大的进步空间。

亚庇之行3-游客的组合

发表于 2025-06-19 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

这次出去玩注意到一个很有意思的现象，就是游客的组合呈现出一个很特别的比例。据我观察，这次去亚庇游客大致的比例如下：

  • 家庭组合，大约占5%
  • 夫妻/情侣组合，大约5%
  • 剩下的90%，为【小姐姐 x N】，其中N∈[1,5]
  • 男 x N，只看到一组

这几组人的外在情绪表现也很有特点：

  • 家庭组合，老人带着小孩跑，年轻夫妻两个出去玩项目，还算和谐

  • 免费，这个是其次，但是对于提供这个服务的github来说，我给于最大的尊重

  • 标签，通过用户自己+系统自动的方式，会给每个用户打上一系列标签（得益于当时在广告行业以及在阿里接触的人群标签）

  • 广场，其实就是抄了微信的朋友圈
  • 活动机制，这是因为我们的投资人给了我们一些线下场馆的资源，使得我们可以将线上的人群引流到线下开展活动，比如说爬山、骑行、徒步、打台球等等，基于用户是否打上此类标签做优先推荐，同时如果是室内活动，我们还可以通过引流进行分成

前段时间这边一位同事的一个状态，让我突然有点恍神，那一瞬间让我想到了前公司的另外一位同事。首先这位同事是这样的，通过自己的数学知识和编程手段，成功解决了一个会影响我们产品功能的问题，那一刻他非常开心，就像高考时解对了最后一道那题那样开心。但那时我高兴不起来，因为它产生的额外问题会影响客户的正常使用，我得去考虑怎么把这个影响降到最低，后面得做什么样的调整。但是对于他来说很简单，我解开了这道题，证明我很厉害，很有成就感，就行了。

刚毕业的时候我其实也是那样，先不管公司的业务，也不管客户的业务和目标，也不管某个问题是否有什么坑，这个问题给到我了，我针对这个“题目”解出来了，就很有成就感，觉得自己应该会得到表扬。但实际上并不是那样，领导和客户看中的都是结果，你把一个螺丝钉拧得再好看，它也不会让这个产品卖出多贵的价格，多赚多少钱。这个同事也是我当时的想法，于是我给他泼了一盆冷水，告诉他，解决这个问题仅仅是开始，而且领导也会认为，解决这个问题是你必须要做的，并不是“加分项“。做的事情最终都能转化为客户在业务上的价值，这才是这个事情最大的意义。

另外一位同事，是前司的一位博士，学历高专业能力强，这点毋庸置疑。为什么单独拿她出来说呢，因为当时我们做的项目，因为她学历最高，领导先让她来主导，这个时候弊端就出现了，她选择了一条非常学术、非常理想化的路线来做这个事情，并且把做这个事情的人员门槛都拉得非常高。导致的结果就是，花了大量的时间做一些“无意义”的事情。聪明的学会了，对业务没具体帮助；笨一点的直接学不会，进退两难。

这两位都是非常典型的“学生思维”，论做题，他们绝对是佼佼者，单打独斗绝对没有问题。但是如果是做一个商业化的产品，这种思维会成为一种枷锁，使得人过度沉溺于细节而忘却初心。

因此奉劝一下各位看官们，尤其是工作年限尚短的朋友，尽早摒弃掉这种“学生思维”，多想想公司和客户的业务目标、想想领导关心的指标，你的工作如何与这些指标挂上钩，你在职场上一定会比别的同龄人走的更快。

停止内耗

发表于 2025-04-08 更新于 2025-04-09 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

内耗真的一件非常蠢的事情，当你在一个组织中，不管是公司亦或是家庭，不管是因为某个人还是什么情况，当发生了持续内耗的情况时，这个组织的风险已经很大了。

之前在的某司，本来公司的愿景很好，老板也愿意投入资源，奈何被“骗”得很惨，招了一些不靠谱的高管，高管带了一堆不靠谱的人，在公司业务上毫无建树，然而在搞事情上却到处是这些人的身影。立一些没有意义的项，做出来屎一样的产品，然后被骂，被关闭项目，然后再找个油头再干别的项目，继续浪费公司的资源。其实这样问题也不是特别严重，严重的是，别的组别的部门做的好好的东西，他们再去做一些类似或者一竞争的，再加之在上级领导面前碎嘴，导致领导把做事的这些人的活转给了那些不靠谱的群体。本来是一个团队做好能够非常好对接的东西，硬是分成了两个团队两个项目，接下来的扯皮、对接事情更是层出不穷，耗费了大量的人力物力，最后还把项目整得四不像…

在家庭里，本来各司其职，互相理解包容，一个家庭会是非常和谐。然而如果出现了内耗型成员，那么整个家庭的氛围都会被这一个人搅得鸡犬不宁。有些事情不太了解，为什么有的人就是不喜欢和谐的场面，非要去搞一些事情出来，才显得这个家庭热闹，好像自己才有了存在感，才被大家关注。也许是原生家庭成长环境的问题，而且一个人的性格一旦形成，真的很难改变。不要指望去改变谁，但是真的需要找到一个好的办法和这种人相处，怎么样才能保持一个相对和谐的状态。

内耗真的是太费心力了，把时间拿出来，去做些有意义的事情吧，而不是来折磨人。

过程 - 结果 - 目标

发表于 2025-03-30 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

这些年工作的一些感悟，我觉得很重要的一点，就是你在做一件事情关注的是什么，这点也许是最重要的。

刚毕业时，觉得自己什么都不会，埋头苦干，有什么需要就学什么，想把自己本事练会，就能得到器重，到哪都能吃上饭

工作了一些年头后，发现很多事情不在于你有多聪明多努力，而是结果更重要。只要结果是好的，那么这件事情就算做好了，你管这个人到底专业不专业，人品好不好，他能谈下单子，你管他到底是怎么做的？

再到后来，发现很多时候，自己努力了，但是结果往往不受自己控制，但是还能昂首挺胸地继续前进，这就是因为有一个好的目标。目标明确了，你会想尽办法去达成，会有一种不到最后不罢休的勇气与毅力，而往往，有了这个目标以后，这件事情的成功概率是最大的！

回想了下自己的经历：

1、高中考上大学时，没有目标，随便填了个大学与专业，惶惶不可终日，耽误了最好的青春年华

2、大学毕业时没有目标，只是被动地乱投简历，随便找了个不知道是否喜欢是否可以坚持的工作，虽然很努力学习，有收获，投入产出比太低

3、工作了一段时间后，不知道自己想做什么，但是起码知道了自己不想做什么，在这个过程中跳来跳去，换了很多工作，导致没有在一个领域深耕，错过了许多更好的机会

4、直到最近这两年，受老板提点，整日给我们灌输“目标”二字，而很多事情在围绕这两个字的基础上，虽然历经波折，但是往往都有比预期更好的效果

所以啊，人一定要有目标，不管是事业、金钱、爱情甚至是玩，都需要有清晰的目标，哪怕看起来有些难以实现或者荒诞，但是当你分解成若干个小目标一个个实现离最终目标靠近时，好像一切都不是那么遥不可及了，难的事情也没那么难了。

突然想到初中的一句课文“天下事有难易乎？为之，则难者亦易矣；不为，则易者亦难矣”。关键是行动起来，这个道理老生常谈，但真正做到的人太少。

多问问自己为什么

发表于 2025-03-21 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

一直以来我都认为我是个情绪很稳定的人，有时稳定到我都觉得自己很薄情寡义。有的人觉得我很理性，碰到事情总会不慌不忙的去处理，去想办法解决，而不是肆意发泄情绪。我不知道我是性格如此还是说因为经历地多了对很多事情变得麻木。但是最近，我感觉情绪开始变得不稳定了，我告诉我自己，说时候要做些改变了。

我的经验告诉我，如果对现在的生活不是很满意，如果你不开心，一定不要闷着生气，抱怨，因为抱怨是没有任何用的。一定要想想，原因是什么？如果要改变，需要做什么？

比如最近工作上的事情让我很烦躁，具体表现在：能者多劳，多做多错，加量不加价

因为之前在求职的时候无意中暴露了自己这个也会那个也会，导致老板一有什么不知道该谁去做的时候，总会想到我，让我去，这无形中让我多了不少工作

因为做的多，对我本职工作的进度造成了影响，同时因为某些分外的事情不太熟悉，导致前期难免会出一些问题，这就导致了，老板就会看到你出了一些问题，甚至会觉得“你本职工作做得也不咋地，让你干点别的也不会”，纯吃力不讨好

因为某些其他的活给别的同事干不好，老板一怒之下把他裁掉了，然后一甩手就说全部交接给我。看起来好像是对我的赏赐？现在我一个人又多干了一个人的活，还TM不给我加工资

总结原因：

1、不太会拒绝

性格有些软弱，碰到强势的领导，虽然有时会跟TA硬刚，但是发现每次硬刚除了挨更多的骂之外对事情并没有什么卵用，于是乎我很多时候就用“哦”“嗯”代替。有些事情无形中就扔给你了

2、喜欢卖弄小聪明

时不时展露一些自己这也懂那也懂（都是自己的积累，但不是为你这家公司特意准备的），显得总比其他的同事干得好，这样变相把对方的活给“抢“了，结果呢，挨了对方挨的骂，接了对方的活，还拿不到对方的一毛钱工资

3、不太主动去争取

有时感觉自己吃亏了就吃亏了，但是这样不行，没有人会因此同情你，反而会继续在你身上踩一脚，世界就是这么现实!

举个简单的例子，现在我们同事都是理工男，性格比较腼腆的那种，之前吃了很多哑巴亏，而越是这样，老板就越肆无忌惮。我来之前，他们都是要自己带电脑的，连纸笔包括餐巾纸都要自己买，我在想，凭什么？虽然没多少钱，但是我率先开口，于是乎我的电脑是老板给的，办公室里有了公费的纸币卫生纸。

所以，权利一定要去争取！你要敢去跟人要东西，去提要求，别不好意思！宁愿让别人讨厌你，别不要让别人觉得你的便宜好占！

接下来，我要去多争取一些东西，不然，持续性地让自己不开心，这个工作也铁定长久不下去。

科技进步“造福全人类？”

发表于 2025-03-19 分类于 记录 本文字数： 阅读时长 ≈ NaN:aN

ChatGTP横空出世，到前段时间的Deepseek以及炒的火热的机器人，以及现在的各种无人工厂，我们可以看到科技进步确实带来了生产力的明显提升。然而历史的经验告诉我们，几次工业革命带来的生产力的飞跃，造就的其实是更大的贫富差距，工厂主效率高成本低了，更加赚钱；而原本在产线上的工人则面临着失业与生存危机。

我们应该清晰地认识到这一点，而不是一味地相信科技进步能带给你个人一个美好的未来。就如同前段时间上海的那些二代们，听说许多韩国人来上海旅游，扎堆地把豪车开出来炫耀，旁边的打工人和外卖小哥看着那些韩国人一脸没见过世面的样子，心里居然还能生出来优越感，“看你们这些棒子，没见识，我们中国比你们想象的有钱！”。但是实际上，那帮二代的豪车跟你一毛钱关系没有，改变不了你任何命运，甚至造成的拥堵还会影响你这单外卖能不能准时送达。

认清自己，很重要；知道什么和自己有关很重要。当然，面对这些新鲜事物的出现，不要盲目沉溺在对人类社会这么大维度的乐观中，而是更应该关心，对自己有何影响，如何调整自己，或抓住机会，或未雨绸缪。毕竟，企业家们可是苦打工人久矣。
