人工智能词汇表

发表于 2024-03-07 更新于 2024-03-25 分类于 人工智能
综合性词汇
词汇 概念解释
AI /Artificial Intelligience /人工智能 让机器实现像人一样的技术的统称
ML/Machine Learning /机器学习 让机器从已有的数据中去寻找规律并对新来的数据做预测的方法
DL/Deep Learning 深度学习 机器学习的方法，最重要的一个分支，模拟人脑多层神经元结构的一种方法，约等于神经网络。深度学习大致就是用人类的数学知识与计算机算法构建起来的整体架构，再结合尽可能多的训练数据以及计算机的大规模运算能力去调节内部参数，尽可能逼近问题目标的半理论、半经验的建模方式。中间隐藏层数较多的才叫深度学习，层数较少的只能叫神经网络，不能叫“深度”
神经网络 模仿生物神经网络的结构和功能的计算系统，输入信号经过多层神经网络处理（加权求和、偏置），并通过激活函数来决定是不是继续传给下一层，最终得到输出
监督学习-Supervised learning 机器学习的类别，监督学习就是训练的数据有题目也有答案，机器从问题和答案中找规律，下次新的问题来了，根据规律或者从中学习到的经验/公式，通过“套公式”得出对应的结果
非监督学习-Unsupervised learning | UL 有题目没答案，让他自己学规律，比如给一堆猫狗的照片，让他按共同特征归类，哪些是猫哪些是狗(机器只知道他们像，并不知道他们叫阿猫还是阿狗)，非监督学习没有明确目的，不需要给数据打标签，也无法量化效果
强化学习 通过正反馈/回报来告诉机器应该采取什么行为（典型的就是打游戏，一次次尝试，机器总会朝着分高或者胜利的方向往下走，比如AlphaGo下围棋）
GAN生成对抗网络 通过同时训练“警察”与“小偷”，实现“道高一尺魔高一丈”的效果，最终用于生成以假乱真的内容

机器学习经典算法

发表于 2024-03-07 更新于 2024-03-25 分类于 人工智能
主流监督学习算法
算法 类型 简介
朴素贝叶斯 分类 贝叶斯分类法是基于贝叶斯定定理的统计学分类方法。它通过预测一个给定的元组属于一个特定类的概率，来进行分类。朴素贝叶斯分类法假定一个属性值在给定类的影响独立于其他属性的 —— 类条件独立性。
决策树 分类 决策树是一种简单但广泛使用的分类器，它通过训练数据构建决策树，对未知的数据进行分类。
随机森林 分类 随机森林是由很多决策树构成的，不同决策树之间没有关联。当我们进行分类任务时，新的输入样本进入，就让森林中的每一棵决策树分别进行判断和分类，每个决策树会得到一个自己的分类结果，决策树的分类结果中哪一个分类最多，那么随机森林就会把这个结果当做最终的结果。
SVM 分类 支持向量机把分类问题转化为寻找分类平面的问题，并通过最大化分类边界点距离分类平面的距离来实现分类。
逻辑回归 分类 逻辑回归是用于处理因变量为分类变量的回归问题，常见的是二分类或二项分布问题，也可以处理多分类问题，它实际上是属于一种分类方法。
线性回归 回归 线性回归是处理回归任务最常用的算法之一。该算法的形式十分简单，它期望使用一个超平面拟合数据集（只有两个变量的时候就是一条直线）。主要解决二分类问题，用来表示某件事情发生的可能性。
回归树 回归 回归树（决策树的一种）通过将数据集重复分割为不同的分支而实现分层学习，分割的标准是最大化每一次分离的信息增益。这种分支结构让回归树很自然地学习到非线性关系。
K邻近 分类+回归 通过搜索K个最相似的实例（邻居）的整个训练集并总结那些K个实例的输出变量，对新数据点进行预测。
Adaboosting 分类+回归 Adaboost目的就是从训练数据中学习一系列的弱分类器或基本分类器，然后将这些弱分类器组合成一个强分类器。
神经网络 分类+回归 它从信息处理角度对人脑神经元网络进行抽象， 建立某种简单模型，按不同的连接方式组成不同的网络。

又一个“消费降级”

发表于 2023-12-08 更新于 2023-12-15 分类于 纪录

眼镜的暴利是众所周知的“秘密”，几十块钱的成本动不动卖上千。记得我在上大学前，眼镜基本都是100左右吧，是在小县城的一个老头店里配的，感觉挺好用，一戴就是好多年。然而上大学以后偶然一次和同学的交流发现，他们动不动就是800-1000的眼镜，那时第一个感觉就是，自己太没有见识了，一个月生活费还不够买一副眼镜。至于后来听到别的学院有钱学生每周逛街花1W，一件羽绒服5K块的时候，更是诧异的不行，第一次感觉到贫富差距是如此之大。在小县城里，稍微富裕一点的同学顶多也就是多两件美特斯邦威/乙纯而已，那就算是名牌了，NIKE/Adidas是奢侈品，能拥有的也是屈指可数。

工作以后，稍微有一点存款的时候，我有段时间还真有点“报复性消费”，总想买好的买牌子货，也许某些方面是想证明自己，也许只是填补少年时代的那颗自卑的心。但是再过了一段时间，也许是自己成长了吧，觉得对外界物质的东西没那么看重了，更多在意的是自己的体验与内心的感受。于是开始走平价路线，觉得好的东西应该首先是好用的，其次应该是价廉的。第一份工作广告出身的我，后面听到某些人大谈“把XX卖的贵是本事，我们做的是高端品牌”等字眼甚至还会嗤之以鼻。你能用低的成本低的价格给用户提供好的产品，同时还能挣到钱，给员工提供好的待遇，那才是一个品牌最大的本事。整天在那吹自己打造什么什么品牌的，不谈自己产品好在哪里价格好在哪里的，在我这都是骗子。

去年买的眼镜700多，戴了一年眼镜腿开叉，马上又要更换了，这次自己换了个思路，找网上配一个看看。这次的体验确实令我非常难忘，京东的某个眼镜店，给到他以前量过的参数，度数、瞳距等，89包邮，5天后送到家，戴上的体验并无两样，某种时候甚至感觉比700的还要好（也许是心理作用吧，就这如果坏了，坏一副我买两幅备用，一点不心疼）。至此，这次消费降级让我做了个决定，以后买眼镜都这样买，太值了!

病因 医生级别 态度 挂号难度 等待时间 花费 治疗效果

甲亢 主任医师 好 高 长 700+ 良好

新时代的信息不对称

发表于 2023-10-30 分类于 纪录

都说互联网出现极大提升了信息传输的效率，让人不出门便知天下事。这个年代如果和年纪小的朋友说我们父辈当年从隔壁县进一顿钢材转手利用差价赚一两年的工资钱，他们的确是很难会相信的。毕竟现在到处都在比“最低价”，销售渠道和品牌都卷的厉害，除了拥有绝对话语权的王者，其他的都在你争我夺剩下的市场。

很多年前，我们获得信息的方式顶多是电视、报纸、杂志以及身边的人，信息差非常明显。web1.0四大门户和各类垂直网站起来的时候，我们通过他们单方面的来获得信息，虽说是来自媒体，其实背后也是个人，他们说啥是啥，但是有一点好处就是，信息量是有限的，大家还看得过来，这个时候的信息差在减小，但是依然存在，因为信息源也很有限。但是自从进入web2.0以后，开始互动了，开始由个人发表不同的内容的时候，大家就已经看不过来了，都是选择性的去看自己愿意相信的内容。这个时候，信息差就不再是“我知道，你不知道”了，而是“好多人都知道，但是我不知道该信谁”的问题。网络上充斥着谣言以及营销性质的内容，铺天盖地，更可恶的是，这些营销性质的内容往往有着更大的流量，他们甚至舍得花钱去刷，导致真正需要信息的人被蒙蔽了双眼。

大众点评初期，很多人还是认真去点评每个店铺的服务、口味等等，还具有参考性。但是现在，你去了某一个不熟悉的地方想找一个好吃的店，如果你看点评/小红书/抖音等平台等热门或者高分推荐，八成是要踩坑。因为你搜到的热门内容，许多都是有团队来运作的，内容质量“高”而且有煽动性，没经验的人一看觉得不错可能就信了，结果过去了发现网络和现实的差距实在太大……

天下熙熙，皆为利来；天下攘攘，皆为利往。工具或者技术的进步可以帮助人去减少一定的信息差，但是在利益面前，人为的“信息差”却又无处不在。他们充斥在各大平台上，误导着一群又一群过来的人。原本向善的平台，希望能为大家服务的平台，“屠龙者终成恶龙”，变成了滋养这些内容的润土。

当然，还是存在许多的优秀内容的，并没有说一棒子打死。但是劣币驱逐良币在各个角落都存在，需要个人自己的经验和判断力，以及平台的共同努力，当大家对一个平台不信任时，它差不多也凉凉了。

门庭若市的儿科门诊

发表于 2023-10-30 更新于 2023-10-29 分类于 纪录

开始只觉得周围好像多了很多咳嗽的小朋友，没想到这么快也轮到了我1岁半的儿子。当他开始咳嗽发烧时，我们并没有多在意，就把家里原来屯着新冠用的药吃了点，烧是容易退，但是会反复，而且咳嗽也不见任何好转。没办法带去医院检查一下，结果看到现场的场景惊呆了，相比于其他科室，儿科+急诊基本上全是小朋友，咳嗽的咳嗽，挂水的挂水。很多年没有见到过这种场面了，应该说对于我印象中的儿科来讲，这是头一次。

治疗的经历比较坎坷，庆幸的是我这段时间不用上班，医院跑了不知道多少回。而这次经历，让我对部分医生产生了质疑：

  • 第一次验血，医生a（副主任医师）说支原体弱阳性，开了阿奇霉素吃3停4，毫无用处
  • 第二次验血，医生b（副主任医师）推翻第一个医生的结论，说验支原体毫无用处，改打三天头孢试试（真的是试试），期间问了问题，很不耐烦不愿解答
  • 打完针三天，症状无好转，回医生b复诊，说继续再加针加药再验血，还要宝宝住院七天观察，尝试询问又被怼回去，认为我们质疑他专业性。我把单子拿了，没交钱直接出了医院
  • 老婆从邻居宝妈群得知医生c（主治医师）口碑不错，运气好挂到了号再去看，医生很有耐心看了之前的病例，知道部分药有剩，开了点新的药以及中药，一共花了40
  • 吃了一天药后，症状有所好转，状态也好了起来，准备过几天再挂他的号复查

看到这个医生，我就想起了我高中的老师们。这个医生态度好有耐心开的药也便宜效果也好，结果一直是主治医师，没有副主任更没有主任，挂号费也便宜，但是那几个让我花了上千还不知道病因的，挂号费贵搞半天也不知道是什么原因态度也差。高中我最敬佩的两个老师，教书教的好一直教书，而那些整天醉醺醺脸红晚上打牌的老师，一个个往上升，什么教务处主任副校长乃至教育局。这个现象说明了很多问题，这个社会病了，而且病得不轻。

掌握权力的人没有被好好监督的话，他们就是“人上人”，容不得质疑，这才是最可怕的地方。

35“危机”

发表于 2023-10-16 更新于 2023-10-17 分类于 纪录

不记得是从哪年开始，貌似是从阿里开始，其他公司也纷纷效仿，陆续地或”裁“或”优化“掉许多35岁以上的员工，公关水平低一点的叫”优化组织结构“，水平高一点的就自称”为社会输送人才“。把这个动作还说出了一点社会责任感，很是佩服。

回到自己身上，目前也在经历这个所谓的”35危机“，其中一点很明显就是，当你重新开始找工作的时候，你会感受到这个社会的”恶意“，你会收到很多”已读不回“，你会收到很多的”不匹配“，你会收到各式各样的拒绝你的理由，但他们不会说你年龄大。

对此我也表示理解，因为之前听到一个说法”当你工作到30几岁的时候，找工作如果还需要自己去投简历，说明你确实挺失败“。虽然很残酷，但想想好像也是那么回事。年轻的时候尽量去折腾了，缺少对自己未来的规划，这边搞搞那边尝试一下，晃晃几年过去了，没有积累到足够的“人脉”，也没有成为某方面的“专家”，还是那个“兴趣广泛，无一精湛”，且还多了房贷和娃的中年男人。如果这个时候还不好好反省，那确实应该受点“毒打”。

但是平心而论，经历的多，我个人觉得收获还是挺大。从开始的懵懵懂懂去搞数据分析，到后面义无反顾地从分析角色头铁跑去做产品经理，在阿里好好的又头脑发热去创业，再到后面做一点杂七杂八的事情，再到后来被忽悠去做自己不喜欢不擅长的售前客户工作，再到后来觉醒决定一门心思的去好好钻研深耕一个岗位一个行业，其实都是在不断摸索的过程。我认为道路虽然曲折，但是前途未必不光明。因为做过的角色多，更清楚在不同的视角看问题，知道从执行层到管理层再到上下游再到投资人是怎么看待公司怎么看待业务的，知道什么才是关键的问题，知道怎么去做团队和产品的管理。目前这个阶段，起码我自己感觉，应该是最可以在岗位上发光发热的时候，也是可以价值最大化的时候。
