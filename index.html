一次顺风车经历

发表于 2025-09-02 分类于 记录 Waline： 本文字数： 504 阅读时长 ≈ 2 分钟

前段时间周末去隔壁市找同学聚会，鉴于之前自己开车挺累的经验，于是打算这次坐车去，但奇怪的是这两个市挨得挺近，坐高铁居然如此麻烦…于是乎上滴滴约了一个顺风车。讲道理，如果要考虑坐的舒服的话，顺风车还是别考虑了，来回的两趟，这两个司机的行为出奇地一致…

首先，现在平台上真正的顺风车基本你是遇不到了，都是营运车辆，在我有限的两次体验中，有如下经历：

1、不载满客肯定不会走，就算绕远一点也要接满再出发

2、平台你勾选了【不出高速费】选项，单子也会被这些营运车抢掉，并且马上打电话给你，出不出高速费，一人XX块，不出那你乘客自己取消订单吧

3、上车就给你二维码，扫了高速费以后再发动

4、路上堵车，疯狂走应急车道加塞，我这次来回两个司机稍微堵一点就往应急车道跑，好像是他们家开的一样

5、开车怠速，玩手机刷抖音，我真的服气，被我大声制止

6、有近道不走，偷偷提前下高速，耽误时间，就为了省几个乘客一起10多块钱的高速费

以后除了群里的顺风车，平台上的顺风车还是能别坐别坐，早点安排买票，体验会好很多。

令我更觉得又好气又好笑的是，在车上大声打电话影响到他人且看起来脾气不太好的两位大姐，在面对司机怠速开车刷抖音的时候，居然选择了一声不吭？在这种这么危险的事情面前，她们的容忍度居然如此之高…罢了罢了，也许知道，这次命真在司机手上，骂不得，骂不得…

前两天又跟老板吵得面红耳赤，感觉这种状况是停不下来了。多buff加持的老板真的是最可怕生物（单身+大龄+女性+事业心强+当过领导），通常都是性格强势，听不得任何与她想法不同的意见。这种人除了用比她更大的声音跟她吵，把她镇住，基本无解，你越是不跟她吵她越觉得自己正确，得寸进尺…吵完之后发现自己的问题了，也不会认，就转移一下话题就想把这个问题盖过去，丝毫不考虑之前自己那副德性是否给对方造成什么样的心理创伤。

向上管理

发表于 2025-08-21 更新于 2025-08-22 分类于 记录 Waline： 本文字数： 701 阅读时长 ≈ 3 分钟

最近比较“同情”某一个客户，从与他以及他领导或者其他相关部门同事的沟通过程可以了解到，跟其他兄弟公司对比，他承担了最大的压力，做了最多的事情，考核的结果虽然不是特别好，但是他所在公司却是最赚钱的，其他公司指标确实更漂亮，但是不赚钱…从公司经营的角度来讲，其实他是做得更优秀的，就是在面临更复杂更困难的情况下，反而拿了更好的成绩，这一点，我以及周围的同事还是挺佩服的。

但是有一点，在跟上级汇报时，他太实诚了，从来不掩饰自己的问题，正视问题也愿意去解决，正常情况下大家都会认为他应该会得到晋升。然而实际情况就是，领导看他能吃苦，就让他继续待在吃苦的岗位上；而其他汇报做得漂亮的，容易得到老板的重视以及提拔，相比之下其余几个业绩没那么好的，反而升得快。

虽然之前一直都有人在说“向上管理”这个事，我相信绝大多数他这种人也是懂得这个道理。包括我在内，我觉得我和他是一类人，所以比较惺惺相惜吧，然而我们这一类人，就多年一直在一个承担各种责任的角色上继续做着“牛马”。我们也知道所谓的“向上管理”，会让自己升的更快，但是骨子里的那点“任性”或者“傲气”却一直在阻止我们这么做，仍然相信，把事情做好了比什么都重要，而自然而然会得到好的回报。这就跟佛家所说的“因果“有点像吧，这种也是“害人“的玩意，用来安慰那些善良的人的。事实上，我们周围很多的“恶人”，没有得到“恶报”，反而继续逍遥，这些例子数不胜数；相反的，许多好人，也并没有得到“好报”，只是一味地在坚持自己的“善念”。毕竟大部分好人其实做好事是不求回报的，而“恶人”不一样，目的性强，利益驱动，损人利己的事情更是不在话下。

我想“性格决定命运”其实说的也是这事吧，但这个世界上的各种“美好”确实是由千千万万善良的人来维系的，一个好的公司也一定是一群踏实肯干的人在背后支撑才能做起来，如果不是到万不得已，相信绝大多数人在“善恶”之间，还是愿意去选择那个“善”

长不大的我们

发表于 2025-08-12 分类于 记录 Waline： 本文字数： 522 阅读时长 ≈ 2 分钟

最近和同学晚上会一边开黑打dota一边聊天，仿佛回到了10几年前那个无忧无虑的年纪，大家没日没夜地打dota的日子。虽然那个时光已经不复返，但是熟悉的感觉还在，虽然开黑的同学里大家干着不同的工作，有的在公司或者政府机关做到了比较高的位置，但是一到开黑的语音里，感觉大家还是那一帮爱玩的高中生。

突然觉得，我们这代人似乎大部分都还没“长大”，回想起自己的长辈们，比如我的爸妈、舅舅舅妈、叔叔婶婶等，当年给我的感觉，他们在25-26岁就已经显得相当成熟了，到了35岁也有了印象中“中年人”的样子：有城府，说话慢条斯理，对人客客气气没有锋芒。而我们这帮人呢，还是那样口无遮拦地喷队友“菜鸡”，除了正式场合，还是Tshirt/短裤/休闲裤/运动鞋，几十年不变的装束…

之前经常被我爸妈吐槽说，你要穿衬衫西裤皮鞋；30几岁的人了吃饭还老被老妈说端好碗别剩、吃完筷子放桌上、别抖腿…；出去玩和95后打成一片，甚至被他们说我“幼稚”；没有以前中年人的爱好：KTV、抽烟、喝酒、打牌、钓鱼、泡茶…

我喜欢自己穿得最舒服的样子；我喜欢妈妈在旁边唠叨我的样子；我喜欢在某些场合去说一些“幼稚”的话；我也有我喜欢的爱好，哪怕许多人说不成熟…

《明朝那些事儿》的结尾里说道：我之所以写徐霞客，是想告诉你：所谓百年功名、千秋霸业、万古流芳，与一件事情相比，其实算不了什么。这件事情就是——用你喜欢的方式度过一生。

长不大就长不大吧，其实，如果有选择的话，谁会愿意去“长大”呢？

《浪浪山小妖怪》

发表于 2025-08-11 更新于 2025-08-12 分类于 记录 Waline： 本文字数： 350 阅读时长 ≈ 1 分钟

周末去看了个电影《浪浪山小妖怪》，之所以选这个是因为几年前有几集小短片在网络上爆火，当时对这个短片就很有好感。这次看完，有一种很熟悉很亲切的感觉，熟悉并不是因为之前看过这个短片，而是感觉熟悉的中国动画，它回来了。

我们这个年代人许多国产动画的回忆都集中在《黑猫警长》《舒克和贝塔》《天书奇谈》《邋遢大王》《葫芦娃》等等，而《浪浪山小妖怪》给我的是仿佛回到了那个年代的感觉，水墨的背景，二维的人物，加上了许多中国式的幽默，还有当前社会众多小人物的缩影，许多梗恰到好处，故事改编得也很出彩，在我看来，完全是可以打9.8分的水平，推荐一看，我也很期待它的下一部。

同事问我，如果要和《哪吒2》来比，你觉得哪个好？说实话，相比之下我更喜欢这部。哪吒当然也很出彩，但是出彩在制作，但是从故事性来说，我觉得这部，更贴近生活，从这几个小妖怪身上可以看到自己以及身边人的影子。相比于拯救世界的超级英雄，我更喜欢平凡的伟大。

学琴这么久，目前我手上有两把吉他，都是拿火这个品牌。第一把是学完入门吉他课程后，意识到当时那个“老师”（实际就是个琴贩子）卖给我的其实是一把烧火棍后，决定换一把趁手的兵器。

恰逢当时同学的表弟在上海，在一个吉他店打工，让他给我挑了一把性价比还可以的，当时这个牌子才出来，想着支持国产，1K多块钱买把面单，也没想太多了就入手了，到现在已经好多年，陪我从上海走到广州，我亲手给它换的弦的价值可能也已经超过它本身了，带着它听了押尾光太郎、岸部真明、GIN、Andy Mckee、Luca等的现场演出，它见证了我的进步与目前的停滞不前，我也见证了它从光鲜亮丽到现在的“伤痕累累”。我这个人很念旧，一旦时间久了，有了感情了，就很难割舍，所以哪怕它再旧，但是和我有着这种多年的默契，我也一直把它带在身边，在闲暇时候和它一起放松心情。

到后来，这个品牌出了碳纤维吉他，对于我这种不太会保养琴的人来说也算是一种便利，自带拾音器效果器，还有APP提供一些基础的功能，卖我5K块应该也算是大冤种了吧。但是没办法，我这个人念旧，和这个牌子有感情，只要弹得还行就一般不会刻意计较，这把琴也跟了我有3年，目前用着也还算顺手。

但是说到软件，我只想用“稀烂”两个字来形容。内置调音器这些是基本的就不说了，鼓点、节奏、效果器等内容，乱七八糟，不支持搜索不支持收藏，曲库评自己心情来更新，感觉是使了大力气，做了一堆事情，看起来也很漂亮，但是用起来一坨屎。貌似也好久没有维护了，可能他们都以为做的很完美不需要更新了吧。

一开始给我的感觉是做琴还挺用心，颜值高用起来也不错，但是这个软件一出来，让我感觉自己像个冤大头…建议这位老总，把你们负责软件的产品总监先裁了，去找个懂一点的吧，我估计那人自己都不弹琴，都不知道练琴的人需要什么东西，这样做出来的东西能让人满意么。

国内工厂生死现状

发表于 2025-07-08 分类于 记录 Waline： 本文字数： 605 阅读时长 ≈ 2 分钟

今天开了一整天的会，和客户工厂的管理者聊挺多，可能是国内许多工厂当前现状，仅拿客户这一家为例：

  • 行业萎缩，极度内卷，到处抢单，许多亏钱也要做
  • 各部门之间存在严重信息差，产供销脱节严重
  • 明知没有订单，依然为了保产线产能开工，大量成品及物料堆积，呆滞风险极大
  • 行情好的时候销售掌握了话语权，其他部门无条件满足，当行情不好的时候，问题暴露严重：
    • 不考虑物料包材通用性，大量采购堆积浪费
    • 有单就接，型号多，换线成本高，小小产线成品SKU居然能有上万个
    • 不赚钱型号多未及时优化，管理难度倍增
  • 同时，每个部门都有各自的KPI，各自有冲突但没有一个角色统一协调做出牺牲
    • 销售只管卖，不管你后面供应链是否复杂，成本如何，只要能完成销售KPI就接
    • 制造部/产线要保证效率，希望保证出勤率保住员工工资不流失，不愿意减少产能，不愿意关闭自动化线（关闭自动化产线使用手工线会影响单位生产成本）
    • 夹在中间的供应链部门最为憋屈，面对销售，在没有预测的情况下只能大胆备货保证交付；面对产线明明不需要这么多货却依然为了所谓出勤率要开工这么久生产这么多，最后库存高周转慢呆滞多，全是他们的锅

要保证一个部门的KPI，那么必然会影响其他部门目标的达成，谁也不愿意自己背个差的绩效影响工资奖金，就这么耗着，慢性死亡…..

如果老板意识不到这个问题，那么距离工厂的生死存亡，可能真的近在咫尺。

还有一个心酸的点，产线负责人指出来，当前工厂工人普遍工资偏低，而且普遍26天x10小时的工作时间，如果工厂要减产能，改为22天x8小时（劳动法规定的上限），离职率将飙升……

所以说，经济下行到头了吗？开始复苏了吗？我觉得还远远没到…

4、原创性很重要，二次剪辑也要讲究技巧，审核通过率高

5、量不需要太大，一天一更就可以了，多了反而分散流量

当年刚毕业的时候在广告公司做广告与网站流量分析，在那个时候受同事影响，决定学习自己搭建网站，以更好地去完成手上的工作。最开始是用wordpress搭建，有一撇没一撇的随便更新点内容，顺便自学一下html、css、js等在自己手上练手，并且在网站上布上监测代码以验证自己的某些猜想，那个时候还没想着自己写点什么，毕竟我从小到大最恨的就是语文

后来一段时间心血来潮，买了域名，买了服务器，决定好好地更新一下，奈何肚子里没有墨水，不知道写点啥，中间荒废了一段时间。但是成本还在，域名和服务器的钱一样要交，导致后来想重新拾起来的时候，因为续费不及时问题差点所有内容都被清除

再后来，写了一段时间公众号，但我这个人不太会拿捏文字的尺度，导致经常性地，写了半天的东西抨击一下社会不良现象，针砭时弊的内容，无一例外全部被下架，审核都过不了。一气之下直接放弃了公众号，这种被平台肆意践踏、不自由的感觉让我感觉极其恶心，于是我决定，一定要找一个“自由”的平台，不要各种乱七八糟的备案，没有平台各种主观且无理由的拒绝，于是找到了Github+Hexo+Dropbox这个组合，爱不释手。Dropbox多端同步保留我的原始文件，hexo用于生成静态网站并提供高质量的主题，github用于存放网站内容，还有什么比这个更贴心的呢。

总结一下原因就是：

  • 自由，不受各种管控，无需看人脸色，想说什么说什么
  • 免费，这个是其次，但是对于提供这个服务的github来说，我给于最大的尊重
  • “永久”留存，域名和服务器不续费，你的东西就找不到了。但有了上面的组合，我相信，只要我自己不主动删除，它存在的时间可能比我都长

15年被忽悠从阿里出来“创业”做社交产品，因为整体投融资氛围较好，所以也还算比较容易地拿了第一笔投资，当时正值陌陌美国上市价值40亿美金，而探探等一系列社交软件也如雨后春笋一般冒出。记得我有一段时间每天早上上班第一件事就是打开App Store看下有多少同类产品出来，那一段时间的统计结果我记得很清楚：平均每天16个APP上线。没错，当时就是那么疯狂，也是那么残酷。同时还有一个数字，就是死亡率99%+，很不幸，当时我们的项目也在这之列，不过也都是过去的事情了。

说到产品本身，记得当时陌陌探探都还是左滑右滑聊天，然后就是各种与约炮有关的营销活动，铺天盖地，它自己官方肯定不承认，但是凭着这个点，他们一举拿下了陌生人社交的半壁江山。而当时我的设计稍微有所不同，比他们多了点功能：

  • 标签，通过用户自己+系统自动的方式，会给每个用户打上一系列标签（得益于当时在广告行业以及在阿里接触的人群标签）
  • 匹配机制，我设计了一个数学模型，基于年龄差（差的小的分高）、距离、性别（异性分高）、兴趣标签（包含同类兴趣标签越多的越高）这些因素，每个人都会与池子里的用户进行匹配并计算得分，然后优先推荐匹配度高的用户撮合沟通
  • 广场，其实就是抄了微信的朋友圈
  • 活动机制，这是因为我们的投资人给了我们一些线下场馆的资源，使得我们可以将线上的人群引流到线下开展活动，比如说爬山、骑行、徒步、打台球等等，基于用户是否打上此类标签做优先推荐，同时如果是室内活动，我们还可以通过引流进行分成

在当时我们看来，比陌陌探探多了一个匹配机制和广场，其实可以更好地促成陌生人社交，而后失败的经历告诉我们：1、简单才是硬道理，初期不要把产品做太复杂 2、不要去揣测人性

尤其是第二点对我们的教育意义更大，因为从后面与大部分用户（尤其是女性用户）沟通下来，结合我们平台的数据发现，其实约炮才是我们此类平台重度用户留存的根本原因。我们试图去把交友以及活动这个事情往好的方面发展，为茫茫孤独的青年找到与自己志趣相投的伙伴，但是用户回馈给我们的其实就是这几个字“我不管，我就要约炮”，我相信做过社交平台的朋友应该可以深刻地理解这个事情。

虽然我们当时也积累数十万的用户，但是这类同质化平台的核心竞争力其实就是资本，看谁能烧钱抢占市场，谁就赢了，就和当年的“百团大战”、打车平台混战一样。由于我们对股权以及对赌协议这个事情出现了重大失误，导致后续的资金无法跟进，最后这个项目黄了，很是可惜。看了一下，Soul这个平台第一版上线的时候，我们才倒闭没几个月。

规则是文明社会的产物，是其得以稳定发展极为重要的因素。作为一个社会中的个体，当然默认也要去遵守各种规则，至少，在我们接受的教育，不管是家庭教育还是学校教育，都教我们要“守规则”。当然有些时候直接变味了，变成了“听话”，某些掌握权力者把自己的话当“规则”，因而从小“听话”也被认为是素质优良的一种表现。

但是到了社会这座大学，你会发现，很多时候守规则变成了一种“束缚“。虽然很多时候我们都告诉自己，一定要恪守底线，自己认为不对的事情坚决不做，但是当你看到许多不守规则的人，反而走上了捷径的时候，心里的那种不平衡感是很强烈的。尤其是见得越来越多的时候，你会开始怀疑自己，是不是也要走一下这种“捷径”。

打游戏的时候，你恪守底线，坚持用自己的手法来与对方竞技，发现对面一堆开挂的，你拼尽全力也动不得对方分毫，这个时候有几种选择：1、离开这个游戏，远离这种环境 2、举报，发现永远举报不完，惩罚力度太小，重来太简单 3、打不过就加入。试问你会选哪种？

我相信很多人一开始也是这样的，觉得应该做正确的事，然而由于破坏规则人因此享受到了便利，影响了他人的同时却得不到应有的惩罚，慢慢地也变成这样的人，试问谁一开始就喜欢走应急车道，谁一玩游戏就想着开挂？

就和现在某些荒谬的案件一样，如果好人得不到保护，坏人得不到制裁，那么，坏人只会越来越多，而手握规则的人则都是整个风气变坏的帮凶。

“学生思维”

发表于 2025-05-04 更新于 2025-05-05 分类于 记录 Waline： 本文字数： 758 阅读时长 ≈ 3 分钟

前段时间这边一位同事的一个状态，让我突然有点恍神，那一瞬间让我想到了前公司的另外一位同事。首先这位同事是这样的，通过自己的数学知识和编程手段，成功解决了一个会影响我们产品功能的问题，那一刻他非常开心，就像高考时解对了最后一道那题那样开心。但那时我高兴不起来，因为它产生的额外问题会影响客户的正常使用，我得去考虑怎么把这个影响降到最低，后面得做什么样的调整。但是对于他来说很简单，我解开了这道题，证明我很厉害，很有成就感，就行了。

刚毕业的时候我其实也是那样，先不管公司的业务，也不管客户的业务和目标，也不管某个问题是否有什么坑，这个问题给到我了，我针对这个“题目”解出来了，就很有成就感，觉得自己应该会得到表扬。但实际上并不是那样，领导和客户看中的都是结果，你把一个螺丝钉拧得再好看，它也不会让这个产品卖出多贵的价格，多赚多少钱。这个同事也是我当时的想法，于是我给他泼了一盆冷水，告诉他，解决这个问题仅仅是开始，而且领导也会认为，解决这个问题是你必须要做的，并不是“加分项“。做的事情最终都能转化为客户在业务上的价值，这才是这个事情最大的意义。

另外一位同事，是前司的一位博士，学历高专业能力强，这点毋庸置疑。为什么单独拿她出来说呢，因为当时我们做的项目，因为她学历最高，领导先让她来主导，这个时候弊端就出现了，她选择了一条非常学术、非常理想化的路线来做这个事情，并且把做这个事情的人员门槛都拉得非常高。导致的结果就是，花了大量的时间做一些“无意义”的事情。聪明的学会了，对业务没具体帮助；笨一点的直接学不会，进退两难。

这两位都是非常典型的“学生思维”，论做题，他们绝对是佼佼者，单打独斗绝对没有问题。但是如果是做一个商业化的产品，这种思维会成为一种枷锁，使得人过度沉溺于细节而忘却初心。

因此奉劝一下各位看官们，尤其是工作年限尚短的朋友，尽早摒弃掉这种“学生思维”，多想想公司和客户的业务目标、想想领导关心的指标，你的工作如何与这些指标挂上钩，你在职场上一定会比别的同龄人走的更快。

这几天京东来了大动作，宣布要进军外卖行业，并且要给所有的外卖员上社保。国内我比较有好感的两个企业家，一个是雷军，一个就是刘强东。他们两个都算是草根出身，两个特点很鲜明：一个重视用户，为自己的客户提供性价比高的产品；一个重视员工，尤其是底层员工，让他们更有尊严更有安全感地活着。

最北与最南

发现一个现象，刷到海南的短视频，基本上周围的群众一溜全是东北口音。我身在珠海，但是周围听到东北口音的概率也远远大于广州。给我的感觉就是，东北人一大票的都在往南边，尤其是靠海的地方转移，其中年纪偏大的比例更多。好家伙，一下从一年一半冬天直接到了一年一半夏天。

很多年前就有个说法“投资不过山海关”，东北的公务员们在全国人民的口碑跟黑社会没什么区别了已经，各种吃拿卡要把想去投资的人都整得够够的。而那些通过权力交换财富的人呢，一部分把自己的子女和财产送往国外，还有一部分去了海南，买豪宅养老，好不自在。同时过去的还有一批过去开烧烤店的，拉帮结伙，宰客，缺斤少两的事情也没少干，把原来海南本地淳朴的民风都带出了一点“匪气“。把这里变成了东北人的“第二故乡”。

当然，许多人远离家乡并不都是出去“享福”，更多的是因为自己的家乡容不下自己而被迫外出。在外地见过许多东北人，都是传说中的仗义、靠谱、热心，他们都是因为家乡没有背景关系没有合适自己的岗位才选择出来。而作为一个湖南人，其实大部分外出的湖南人（包括其他江西四川等省份）也一样，湖南教育资源很好，人才足够，但是没有一个好的土壤，没有好的公司提供足够多的岗位让他们留下来，所以才选择外出。网上流行的梗“湖南人的毕业礼物，就是一张去往广东的车票”，乍一看挺搞笑，实际道出了大家的心酸与无奈。
