Sublime快速格式化json

发表于 2024-03-25 分类于 记录

平常爬取数据大部分是json格式的，但是经常是一大版不带逗号糊过来的字符堆，不便阅读。还好Sublime自带的插件很好的解决了这一问题。步骤如下：

  1. 安装Package Control（如果尚未安装）： Sublime Text的Package Control是一个非常重要的插件管理器，它可以让你轻松地安装、更新和删除Sublime Text的插件。如果你还没有安装Package Control，可以通过以下步骤进行安装：

    • 打开Sublime Text。
    • 按下Ctrl+``（在Windows/Linux上）或Cmd+``（在Mac上）打开控制台。
    • 复制并粘贴以下命令到控制台中，然后按回车键执行：
    • 重启Sublime Text以完成安装。

  2. 安装Pretty JSON插件：

    • 按下Ctrl+Shift+P（在Windows/Linux上）或Cmd+Shift+P（在Mac上）打开命令面板。
    • 输入Install Package并选择它。
    • 等待Package Control加载所有可用的插件。
    • 在搜索框中输入Pretty JSON并选择它进行安装。

  3. 格式化JSON数据：

    • 打开包含JSON数据的文件。
    • 确保Pretty JSON插件已经安装成功。
    • 选中你想要格式化的JSON数据，或者不选任何内容以格式化整个文件。
    • 按下Ctrl+Alt+J（在Windows/Linux上）或Cmd+Alt+J（在Mac上）来格式化JSON数据。
