不用下载安装云音乐客户端，也能尽享高品质音乐，这就是github的魅力。感谢darknessomi提供的酷炫命令行版解决方案。

来看下命令行下的界面吧：

安装方法

只贴Mac的，其他方式的自己去搜文档了：

前提

  • pip

当然前提你需要装一个pip,这是一个包管理软件，通过他可以很轻易地安装许多或者模块

https://pypi.python.org/pypi/pip参照链接中的文档安装就好了

  • brew

打开命令行，输入

系统会自动完成安装

安装Musicbox

通过刚才安装的brew 和pip安装Musicbox，mpg123是依赖的播放器，也必须安装

$ pip(3) install NetEase-MusicBox

如果报错，则需要向下图一样，在命令行前面加入sudo（授权）

$ brew install mpg123

使用

安装完成，可以来爽一把了

打开终端，输入musicbox

$ musicbox
