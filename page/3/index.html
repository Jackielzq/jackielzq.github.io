暴雨突袭

发表于 2016-10-03 | 分类于 纪录 |

国庆假期怎么也要约朋友一起出来吃吃喝喝，才不枉此节。吃完晚饭后慢慢悠悠地在街上漫步消食，天空中没有一点即将下雨的迹象，然而一场突如其来的暴雨打破了上海的宁静，也打破了整个过节的气氛。

坐着地铁上玩着手机的我一刷朋友圈，发现好多人被大雨所困，心想着还好我机智地上了地铁，不至于回不去，一个小时的车程下来应该也差不多雨小一些能一截一截小跑着蹭回去。然而这次的雨量比我预想的要大好多倍。当我走到地铁口看到熙熙攘攘的人群时，就知道这个雨比我想的严重，严重得就像电视剧里我们看起来很假的那种，几位壮汉拿着巨型水枪左右甩动的情形，一阵接一阵。

这个地区稍有偏僻，人不多，周围也没有商店，因此也没有其他繁华地带地铁口一样，一下雨卖伞的人蜂拥而至。地铁口等待雨停的人群彰显着人生百态：有的一脸茫然无助等待雨停；有的悠然自得，三三两两坐着聊天打牌；有的焦急地掏出手机打电话发微信求助；还有的干脆挽起裤腿，帅气地消失在了雨中……

财商欠费 - 失败的“黄牛”

发表于 2016-10-01 | 分类于 纪录 |

一直都想看一场五月天的演唱会，毕竟除了周杰伦以外，我唱的最多的就是他们的歌了。年华易逝，这个年轻的摇滚乐团以不再年轻了，97年成立，到现在已经是一个将近20年的团队了，年纪比目前的互联网巨头还要大。而他们的成员也是要奔四的人了，说一句悲伤一点的话真是“唱一场少一场”。

说来也奇怪，貌似音乐是有一种让人年轻的力量，时隔20年，他们的长相还真是看不出来多大变化，教我吉他的某一个老师也是，看穿着看外表跟我差不多大，结果比我大了一轮，应该不是我显老……

两个月前开始抢票，如我所预想的一样：几秒之内售罄。随后就是漫天扩散的黄牛票，各种加价炒作。而就在大家以为无票可抢，不买黄牛票就看不了的情况下，几个票务公司半个月前突然又放出了大量的票（当然，数量不显示，不知具体有多少）。好家伙，得赶紧下手买票啊！吸取上次失败的教训，这次手一定要快，要狠，要准。选好价位以后，下单-支付，一气呵成，就那么简单买到了票，感觉还有点小激动。

所谓的客户忠诚度

发表于 2016-09-21 |

对于做广告、做营销的人，“客户忠诚度”这个词应该都耳熟能详。虽然已不直接做这块相关的工作，但由于之前工作遗留的一些习惯，有时还会想想这其中的问题。如果给我自己身上打上“忠实客户“这个标签，我应该挂上什么品牌或者产品，于是首先想到的就是那么几个：微信、支付宝、海底捞。
