发表于 2023-05-10 分类于 纪录 Waline： 本文字数： 538 阅读时长 ≈ 2 分钟

今天是这段时间身体状况最好的一天，也是最近最忙的一天。貌似稍微清闲一点我的身体都不太答应，只要我稍微摸摸鱼搞搞自己的东西，老是很容易就腰酸背痛，但是只要是工作上的事情来了，明显自己会变得更加的专注，之前的一些不舒服的情况好像也在刻意地回避。身体在用实际行动告诉我，”好好打工，别偷懒“

命令 说明 备注
echo “abc” 输出 “abc”
echo < “abc” 输出 “abc” 标准输出
echo “abc” > abc.txt 在abc.txt文件中写入“abc”字符
cat abc.txt 输出文件abc.txt的内容
cat abc.txt > xyz.txt 把abc的内容完全写入到xyz（覆盖）
cat abc.txt >> xyz.txt 把abc的内容添加到xyz中（叠加，在abc原始内容后）
cat abc.txt I wc 输出行数、字(单词)数、字符数
cat abc.txt I wc I cat xyz.txt 竖线”I”作为管道，左边为输入，右边为输出
sort abc.txt 按字母顺序输出（行）
uniq abc.txt 文件内去重
grep aa abc.txt 在abc.txt中查找含有aa的行 global regular expression print
grep -i aa abc.txt 在abc.txt中查找含有aa的行，不区分大小写 case insensitive
grep -R aa /user/Documents/abc 在指定目录下查找相应内容aa，显示详细内容 R：recursive递归
grep -Rl aa /user/Documents/abc 在指定目录下查找相应内容aa，只显示文字不显示详情 files with matchs
sed ‘s/abc/xyz’ xyz.txt 在xyz.txt中查找第一个abc并将其替换为xyz ‘s’为sed固定参数
sed ‘s/abc/xyz/g’ xyz.txt 在xyz.txt中查找所有abc并将其替换为xyz ‘g’：global

最近一个视频莫名地火了起来，视频中男主角用自己不太标准的普通话说了一段“内心独白”，暂且不说这段视频产生的原因，它火了起来是因为一大片的网友以此大做文章，制作大量的表情、模仿视频等，甚至还给其配上了和弦。一时间微博、微信、论坛中各式各样的“蓝瘦香菇”充斥着整个互联网，好一片全民共欢乐的盛景！

暴雨突袭

发表于 2016-10-03 更新于 2016-12-21 分类于 纪录 Waline： 本文字数： 937 阅读时长 ≈ 3 分钟

国庆假期怎么也要约朋友一起出来吃吃喝喝，才不枉此节。吃完晚饭后慢慢悠悠地在街上漫步消食，天空中没有一点即将下雨的迹象，然而一场突如其来的暴雨打破了上海的宁静，也打破了整个过节的气氛。

坐着地铁上玩着手机的我一刷朋友圈，发现好多人被大雨所困，心想着还好我机智地上了地铁，不至于回不去，一个小时的车程下来应该也差不多雨小一些能一截一截小跑着蹭回去。然而这次的雨量比我预想的要大好多倍。当我走到地铁口看到熙熙攘攘的人群时，就知道这个雨比我想的严重，严重得就像电视剧里我们看起来很假的那种，几位壮汉拿着巨型水枪左右甩动的情形，一阵接一阵。

这个地区稍有偏僻，人不多，周围也没有商店，因此也没有其他繁华地带地铁口一样，一下雨卖伞的人蜂拥而至。地铁口等待雨停的人群彰显着人生百态：有的一脸茫然无助等待雨停；有的悠然自得，三三两两坐着聊天打牌；有的焦急地掏出手机打电话发微信求助；还有的干脆挽起裤腿，帅气地消失在了雨中……

一直都想看一场五月天的演唱会，毕竟除了周杰伦以外，我唱的最多的就是他们的歌了。年华易逝，这个年轻的摇滚乐团以不再年轻了，97年成立，到现在已经是一个将近20年的团队了，年纪比目前的互联网巨头还要大。而他们的成员也是要奔四的人了，说一句悲伤一点的话真是“唱一场少一场”。

说来也奇怪，貌似音乐是有一种让人年轻的力量，时隔20年，他们的长相还真是看不出来多大变化，教我吉他的某一个老师也是，看穿着看外表跟我差不多大，结果比我大了一轮，应该不是我显老……

两个月前开始抢票，如我所预想的一样：几秒之内售罄。随后就是漫天扩散的黄牛票，各种加价炒作。而就在大家以为无票可抢，不买黄牛票就看不了的情况下，几个票务公司半个月前突然又放出了大量的票（当然，数量不显示，不知具体有多少）。好家伙，得赶紧下手买票啊！吸取上次失败的教训，这次手一定要快，要狠，要准。选好价位以后，下单-支付，一气呵成，就那么简单买到了票，感觉还有点小激动。
