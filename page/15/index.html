互联网广告的“衰”与“兴”

发表于 2013-03-21 更新于 2016-09-21 分类于 观点 本文字数： 2.1k 阅读时长 ≈ 8 分钟

今天在知乎上看到有人问这样的问题：“现在的互联网广告有用吗？是在浪费钱吗？”。在这之前我也有想过这个问题，据我所看到的数据来说，从有互联网广告开始到现在，效果可以说是在逐渐下降的，但是最近这两年却又有回暖之势。

为 什么这么说呢？我们从杨致远创立的雅虎开始讲，虽然雅虎目前除了雅虎日本，其他国家或都已经在没落了，但却丝毫不能否认他在互联网历史中举足轻重的作用： 创立了门户网站的模式，内容免费，利用广告获得收入，同时也最先推出了搜索引擎。那个时候真可谓是风生水起，广告的效果也因为互联网的兴起而得到了很好的 效果，展示广告普遍都有1%的点击率。然而回到现在，普通的展示广告大部分都只有0.1%的水平（当然，部分位置醒目尺寸大的除外）。因为现在的网站太多 了，各类门户网站也太多了，广告也是眼花缭乱，让人下意识地对广告产生了抵触情绪，对于熟悉的网站甚至能够做到非常熟练的避开各种各样的广告，所以这两年 不免让少数广告人喊出“互联网广告已死”这种口号。

发表于 2013-03-21 更新于 2016-09-21 分类于 纪录 本文字数： 2k 阅读时长 ≈ 7 分钟

前段时间有看过关于腾讯张小龙谈产品的内容受益良多，对他最后那一句“我所说的都是错的”印象颇深。最近微博上看了一些消息，自己又听到感觉到一些 东西，觉得这句话虽然有些绝对，但是还是很有认同感。所以我说，我们所知道的不一定是对的。毕竟“不对”和“错”还是有一定区别的。

我们通常只是在用自己的思维、自己的眼界来思考事情，或者干脆不假思索一味地接受来自外界传播而来的东西。但是实际上也许这些东西对我们来说，确实 是正确的，但是对于其他人来说，也许我们所认为的真理、事实，在他们眼里完全是谬论。所以为什么说读万卷书不如行万里路呢，因为行万里路你能接触到交流道 各个阶层的人，可以清楚的知道各种人之间的区别。

也许你在这里大谈特谈微博微信营销，但是中国13亿人，有一半以上可能都不知道微博微信是什么。你们在这里讨论哪款手机什么配置、哪款APP可以约 炮怎么炒作怎么火，但是他们却只关心手机信号好不好，话费偏不便宜，wap快不快，有没有好的小游戏可以玩，待机时间长不长，有没有自带《月亮之上》铃 声……回到产品方面，我觉得目前做产品的确实应该跳出自己局限的思维圈，跟各个收入阶层的人都多了解一下“需求”，特别是针对于每一个产品的TA。

广告主与媒体的博弈–广告数据的真实性

发表于 2013-03-21 更新于 2016-09-21 分类于 观点 本文字数： 1.7k 阅读时长 ≈ 6 分钟

相信这是许多广告主和代理商一直以来非常头痛的问题：我花了这么多钱投广告，到底达到了什么样的效果？数据是真实可信的吗？有多少被浪费掉？如果有 水分，那么水分大概占到多少？媒体是用什么手段作弊？怎么通过技术手段滤掉这一部分数据？我觉得，这是一场永远不会结束的博弈，不会有永远的赢家（当然， 除非政府出台强硬措施，不过就目前来看还尚早）。

先来说一下互联网广告的数据监测原理吧，创意公司制作好素材发给媒体，由媒体投放时添加相应的监测代码，这种代码通常是非常简单的形式：展示代码 （也叫曝光代码），通常是一个普通的1×1像素的图片请求，这个请求发向监测服务器，表示这个时候广告被展示了一次；点击代码通常是一个带301重定向的 链接，打个比方来说，张三要去李四家，但是他爸不放心，不知道他到底去没去或者去了几次，就让他每次先到李四的邻居王五家打个招呼，最后再去李四家，着同 样以来张三爸爸最后通过王五就知道张三到底去了几次了。同理，这个链接先通过了监测的服务器计数，然后再跳转到指定的页面。最后由我们看到的数就是经过服 务器整理后的数据–曝光和点击。

Do Not Track–精准广告的“威胁”

发表于 2013-03-21 更新于 2016-09-21 分类于 观点 本文字数： 1.8k 阅读时长 ≈ 7 分钟

Do Not Track，简称”DNT”,能够防止第三方网站跟踪用户活动，是万维网联盟W3C提出的最新解决网络隐私问题的草案之一。根据该草案，用户在使用支持该 功能的浏览器上网时，可以借此发送数字信号，告知企业停止搜集有关他们的在线活动信息，从而保护个人隐私。接受“禁止追踪”协议的网站不可以追踪用户活 动，就像电话推销员不可以向登记过“禁止推销”的人推销一样。微软首当其冲地执行了这一协议，在IE10中默认启用这个设置，想象依靠cookie做精准 广告投放的机构或代理商会遇到什么样的挑战–这显然是砸他们的饭碗了。所以这个消息一出来得到了美国广告界的强烈抗议，好不容易要进入工业革命了突然又要 你回到石器时代，这是怎么样的一种心情。

微软此举相信主要是为了得到更多用户的支持，对抗强大的对手google。光有IE浏览器这么多的用户但是互联网的广告费大部分都给google挣 去了，自己只能眼巴巴得看着，还要给你继续做嫁衣，而现在你自己给自己做了个叫Chrome的衣服，我连衣服都快没的做了，当然会不爽。但是他这么做确实 有个正当的理由，“保护用户隐私”。那么这么做到底对用户来说有真正的区别吗？我好奇地点开了有关文章看了看用户的评论，发现大部分都没有理解这个“跟 踪”的定义，其中还有大部分是互联网从业人士，这让我比较意外。他们所骂的都是“现在网站广告好恶心”，“弹窗弹的好烦，关也关不掉”，“广告去死！”， “最烦贴片广告了，又臭又长”等等说的都是对某些形式广告的深恶痛绝，但这些和cookie基本没有半毛钱关系。毕竟用户习惯的养成是需要时间的，我们总 是习惯电视里插入又臭又长的广告（因为你不看也得看，再不想看也能听见），但是对于互联网来说，我可以选择我看的东西，怎么可以有广告？我想说的是，就算 不跟踪，你也一样得看广告，但是你看的内容就和你期待的会不一样了。
