实用命令行笔记二 - 文件操作

发表于 2016-11-03 | 分类于 分享 ， 命令行 |

对文件的操作：复制、移动、重命名、粘贴

命令 说明 备注
cp abc.txt xyz.txt 复制abc.txt文件，并重命名为xyz.txt copy
cp dir1/file.txt dir2/ 复制dir1文件夹内的文件到dir2文件夹内
cp * xxx/ 复制当前目录下所有文件到xxx文件夹下
cp m* xxx/ 复制m开头的所有文件到xxx文件夹下
mv abc.txt dir1/ 移动单个文件到指定文件夹下
mv abc.txt xyz.txt dir1/ 移动多个文件到dir1文件夹下 move
mv abc.txt xyz.txt abc.txt重命名为xyz.txt
rm abc.txt 删除文件
rm -r dir1 删除文件夹以及其所有文件 r：recursive递归

实用命令行笔记一 - 文件目录及相关操作

曾几何时，看到电影屏幕里黑客们在一个漆黑而神秘的屏幕上敲打着我们看不懂的代码，是不是感觉好酷？其实，你也可以。花一点时间学习基础的用法，大大提升效率，实在是居家旅行必(zhuang)备(bi)神器。

ls -l 列出文件详细 lists all contents of a directory in long format
ls -t 以时间顺序排列，由近至远 order files and directories by the time they were last modified.
pwd 打印工作目录 print working directory
cd xxx 工作目录转移至xxx cd : change directory
cd .. 返回上一级目录
cd ~ 返回跟目录
cd ../../xxx 返回上上级目录的xxx文件夹
mkdir xxx 建立一个xxx为名的文件夹 make directory
touch xxx.txt 建立一个xxx.txt的文件 为什么用touch

令人反感的“蓝瘦香菇”

发表于 2016-10-19 | 分类于 纪录 |

最近一个视频莫名地火了起来，视频中男主角用自己不太标准的普通话说了一段“内心独白”，暂且不说这段视频产生的原因，它火了起来是因为一大片的网友以此大做文章，制作大量的表情、模仿视频等，甚至还给其配上了和弦。一时间微博、微信、论坛中各式各样的“蓝瘦香菇”充斥着整个互联网，好一片全民共欢乐的盛景！

暴雨突袭

发表于 2016-10-03 | 分类于 纪录 |

国庆假期怎么也要约朋友一起出来吃吃喝喝，才不枉此节。吃完晚饭后慢慢悠悠地在街上漫步消食，天空中没有一点即将下雨的迹象，然而一场突如其来的暴雨打破了上海的宁静，也打破了整个过节的气氛。

坐着地铁上玩着手机的我一刷朋友圈，发现好多人被大雨所困，心想着还好我机智地上了地铁，不至于回不去，一个小时的车程下来应该也差不多雨小一些能一截一截小跑着蹭回去。然而这次的雨量比我预想的要大好多倍。当我走到地铁口看到熙熙攘攘的人群时，就知道这个雨比我想的严重，严重得就像电视剧里我们看起来很假的那种，几位壮汉拿着巨型水枪左右甩动的情形，一阵接一阵。

这个地区稍有偏僻，人不多，周围也没有商店，因此也没有其他繁华地带地铁口一样，一下雨卖伞的人蜂拥而至。地铁口等待雨停的人群彰显着人生百态：有的一脸茫然无助等待雨停；有的悠然自得，三三两两坐着聊天打牌；有的焦急地掏出手机打电话发微信求助；还有的干脆挽起裤腿，帅气地消失在了雨中……

财商欠费 - 失败的“黄牛”

发表于 2016-10-01 | 分类于 纪录 |

一直都想看一场五月天的演唱会，毕竟除了周杰伦以外，我唱的最多的就是他们的歌了。年华易逝，这个年轻的摇滚乐团以不再年轻了，97年成立，到现在已经是一个将近20年的团队了，年纪比目前的互联网巨头还要大。而他们的成员也是要奔四的人了，说一句悲伤一点的话真是“唱一场少一场”。

说来也奇怪，貌似音乐是有一种让人年轻的力量，时隔20年，他们的长相还真是看不出来多大变化，教我吉他的某一个老师也是，看穿着看外表跟我差不多大，结果比我大了一轮，应该不是我显老……

两个月前开始抢票，如我所预想的一样：几秒之内售罄。随后就是漫天扩散的黄牛票，各种加价炒作。而就在大家以为无票可抢，不买黄牛票就看不了的情况下，几个票务公司半个月前突然又放出了大量的票（当然，数量不显示，不知具体有多少）。好家伙，得赶紧下手买票啊！吸取上次失败的教训，这次手一定要快，要狠，要准。选好价位以后，下单-支付，一气呵成，就那么简单买到了票，感觉还有点小激动。
