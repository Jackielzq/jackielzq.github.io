不同背景同学的聚会

发表于 2023-10-16 更新于 2023-10-19 分类于 纪录 本文字数： 990 阅读时长 ≈ 4 分钟

国庆和一批同学聚了一下，越来越觉得，随着时间的推进，大家的距离真的越来越远了。这些同学或朋友，都是初高中就认识了，我简单做个对比

同学 家庭背景 工作 工作地点 住房 家庭状况
A 父母公务员，已退休 公务员10余年，稳步上升 老家市区 别墅 儿女双全
B 父亲公务员，母亲个体户 编外人员-高速公路收费-考公务员林业局-调进市区工作 老家市区 县里市区有房 儿女双全
C 家里数个亲戚在烟草局 在外工作两年，自觉甚苦，接受现实，回家接受安排进原来看不上的烟草局，现已调入市烟草局，前途一片光明 老家市区 多套房 儿女双全
D 家里有亲戚很富 在北京工作几年，通过亲戚进入投资公司，但不堪重负，选择回老家县城，进入烟草局，目前工作稳定，半修仙状态 老家县城 老家别墅 儿女双全
E 父母皆教师，老婆也是教师 毕业后一直当教师，稳定，但工资不高 老家县城 独栋自建房，市区有学区房 二女
F 家中做小生意，不愁吃穿 大学毕业后未找到合适工作，在外干过几年杂工，无法忍受，遂回家半玩半考，考了三年成功上岸，在银行管信息系统，稳定 老家县城 老家独栋，市区有房产 儿女双全
G 父亲曾在乡政府，后政府变卖国有资产，失业被迫创业，母亲全职家庭主妇 国内各大城市四处奔波，各种不服，誓要自己掌握命运。放弃过湖南卫视与国税局的安排，换过许多工作，目前刚失业，待业在家 外地城市 一线城市小户型，还欠几十年100多w贷款 一子

首先声明下，G就是我自己，看起来好像是最没着落的一个，在外漂泊，想好好看看这个世界，在刚毕业那几年，他们都很羡慕我，有胆量，敢闯，有见识，做公务员的一直抱怨每天无聊透顶，电脑不能装qq微信，每天喝茶看报纸见不到几个活人，然后每天就是看看窗外今天进出了几辆车，作为今天的工作内容……

而现在不一样了，这次聚会我感觉到明显的不一样的就是几点：

  • 原来大家不喝酒的，现在这些公务员还有烟草人员，基本烟酒都来，还喜欢劝酒
  • 他们变得“自信”了，觉得自己已经“高人一等”了，是个“官”
  • 我拿出尿酸报告说不能喝酒，对方劝酒失败，我感觉出来他们有一种“官威”被冒犯到感觉
  • 除了公务员和烟草局的香单位，其他的各有各的苦
  • 越“努力”的人，好像就越苦；被安排的，越走越顺（上一辈还是老练）

单从工作稳定与收入角度来说，确实挺羡慕他们。当年毕业时作为学校最好的学生，年少轻狂誓要靠自己闯出一片天空，拒绝各种好意与帮助，回头想想真的是挺“蠢”。到现在十几年后来看，慢慢懂得了一些社会运行的法则，开始寻求一些改变，但是这个“悟”的过程，相比之下代价就有点大。

不过过去的已经过去了，我们一直都是乐观向前的那个在路上的少年，从未停歇。前面的路很长，许多人注定会渐行渐远，但是同时也会有新的伙伴进入你的世界，专心做好自己的事情，就不会孤单。

学校不教的东西

发表于 2023-08-29 分类于 纪录 本文字数： 870 阅读时长 ≈ 3 分钟

很可悲，学校的教育到了大学，很多还是填鸭式的方式。不集中精力搞学术，非要去评比，搞综合性大学，搞一堆自己从来没有的专业，做不擅长的事情，教材用别人的，老师也没有行业相关经验与资源，大家心照不宣玩了四年，出去什么都不会，处处面壁。学校也不会教你社会各类企业的特点，各个工种需要什么能力，有哪些权利可以使用。是的，没有，都没有，我只能说，这些学校远离社会，身在象牙塔高层不知人间几何。每个高校都在喊“产学研结合”，结合了吗，除了增加一点对外露脸的渠道，多了些赚钱的方式，其他都是停留在表面而已。

回首自身，环顾四周。大家在经历某些事情的时候，都是闷着头到处找教程，自己摸索。比如什么生育津贴、比如失业金领取、比如异地就医、公积金转移等等，对于体制内的人来说如数家珍，但是对于大部分社会企业的员工，一头雾水。之前很多时候大家的印象就是，公务员钱少很累、国企都是马屁精没意思、外企资本主义剥削……导致很多人就出去“闯”，这样的结果，有少数确实闯出来了。但是大多数经不起各类风险，因为给到的保障太少了，反观之，在做公务员的，每天很清闲，可以和同事“白头到老”，国企的表面工资不高，但是福利奖金各方面给你顶到满，企业每年几百几千个亿营收没有对手，结果为了数万员工的福祉，楞是能做成亏损。没错，这就是社会主义的好处。而某些外企的就更惨了，生娃奖励10W，假期中外的都休，什么公积金比例很多也给你按最高的交，生怕惹怒了中国的员工。

像我们民企多好，为了社会经济的进步，以顾劳动法，已加班为荣，不光PUA自己，还PUA别人，卷得飞起。员工什么金都按最低标准交，多出来的还得自己凑发票，账上资金有问题的话，资金断裂融不到钱的，首先也是这些企业。但是他们还是很坚强的在生存，起码还提供一些工作岗位让一些人能活下来。疫情过后，本以为是万物复苏，结果却是后遗症发作期。亲身经历的某些行业，发现仅剩国企还活着，民企暴雷的暴雷，裁员的裁员，苦不堪言，经济危机的同时还要面对所谓的35岁危机“为社会输送人才”，不得不说，这些人都是我国的栋梁。

  • 建议学校开设下列相关教育内容（不完全）：

  • 劳动法对员工的保护以及常见案例

  • 可以领取的津贴类型与方式

  • 某某工种具体做什么工作

  • 什么样的公司福利比较好

  • 什么坑尽量不要踩

  • 什么样的公司有背景的可以去，普通人别想

  • ……

如此观之，对于张雪峰，只有敬佩。只恨当年他还没有出来，我们就草草毕业。

一个公司的没落

发表于 2023-08-24 分类于 纪录 本文字数： 500 阅读时长 ≈ 2 分钟

据不完全统计：一个公司的没落有几个特点

  • 一批一批地裁员，想走就走，甚至不管你重不重要
  • 员工毫无生气，办公室安安静静
  • 养着一些不产生价值而高薪的人
  • 电梯开始控制时间
  • 开始抓考勤
  • 卫生间纸都开始限制
  • 食堂的饭菜变得难吃

本以为进的是做机器人，高科技行业。到头来还是地产行业，被地产爸爸养着。行业一开始雪崩，自身难保，何况还是嗷嗷待哺的“子女”们。这次经历其实也看清楚了很多事，其中最重要的一点，要成事，一定由“人”决定的。你招了什么样的人，尤其是管理层的人，你的公司就会变成什么样。一个高科技行业，被一群房地产“大佬”连基本概念都不懂人，瞎指挥做机器人，等于是没读书的老头子教一帮大学人学数学。然后这帮吃惯了灰色食品的人，走到哪里都秉承着能吃多肥吃多肥的原则，贪污、拉小山头、进关系户、对内摆架子、对领导画大饼。虽然确实有挺多有能力的人在下面努力的想把事情做好，想把公司从这个环境中“解救”出来。但是没有权利在身，更多时候也是只能看着，最后要走要砍人的时候，那些蛀虫吃得香香的，把下面的人，也不管能力强不强，都干掉。反正能多吃一会儿是一会儿，自己也不懂，也不指望公司能好，不好了去下一家继续吃。

期待一个干净的，干实事，真想创造价值的环境，然后，能跟着它一起成长，然后若干年后能够欣慰地退休。目前自己还不够自己创造这个环境，眼光向外看看。

给西瓜的赞美

发表于 2023-07-22 更新于 2023-07-23 分类于 纪录 本文字数： 464 阅读时长 ≈ 2 分钟

西瓜是我们印象中再平常不过的水果，但是今天忍不住要单独赞美一下。它不像其他的水果，是有目的性质的，比如吃苹果橙子提高免疫力，吃香蕉火龙果是为了通便。很多时候，我们看到水果，如果没有特别的想去吃它，那么说明它其实并没有那么自己喜欢。西瓜在我的眼中，就是最纯粹的，大自然送给夏天最好的礼物。它吃了能解渴，更重要的，它吃了能让人快乐！古人言“日啖荔枝三百颗，不辞长作岭南人”，我一点都不能共情。荔枝这个水果有反噬性，就像“七伤拳”，伤害越高，自己越残，我单次吃过10颗感觉就上火，极容易口腔溃疡。而西瓜就是一个无私奉献的水果，滋润你，让你感到甜蜜。这种炎热的夏天，来上一份冰镇西瓜，那种“爽”的感觉不言而喻。

我就喜欢这种，简单单纯的，不要掺杂太多利益性的东西，我也喜欢这样的朋友。有的朋友，平时跟你称兄道弟，但是你要去照顾他生意买点他家的东西时，卖的和给任何陌生人一样贵，甚至比他们还贵，因为他觉得他做生意，这个钱就该挣。而其他有的好朋友，平时没有说这些客套话，当你说要跟TA买东西的时候，TA甚至会不挣钱的给你，还给你额外赠送，这种朋友我要格外珍惜，因为TA真拿我当朋友。

还是那句话：看一个人，不要看他说了什么，而要看他做了什么。我喜欢吃西瓜，我也喜欢这样的人。
