新时代的信息不对称

发表于 2023-10-30 分类于 纪录 Changyan： 本文字数： 786 阅读时长 ≈ 3 分钟

都说互联网出现极大提升了信息传输的效率，让人不出门便知天下事。这个年代如果和年纪小的朋友说我们父辈当年从隔壁县进一顿钢材转手利用差价赚一两年的工资钱，他们的确是很难会相信的。毕竟现在到处都在比“最低价”，销售渠道和品牌都卷的厉害，除了拥有绝对话语权的王者，其他的都在你争我夺剩下的市场。

很多年前，我们获得信息的方式顶多是电视、报纸、杂志以及身边的人，信息差非常明显。web1.0四大门户和各类垂直网站起来的时候，我们通过他们单方面的来获得信息，虽说是来自媒体，其实背后也是个人，他们说啥是啥，但是有一点好处就是，信息量是有限的，大家还看得过来，这个时候的信息差在减小，但是依然存在，因为信息源也很有限。但是自从进入web2.0以后，开始互动了，开始由个人发表不同的内容的时候，大家就已经看不过来了，都是选择性的去看自己愿意相信的内容。这个时候，信息差就不再是“我知道，你不知道”了，而是“好多人都知道，但是我不知道该信谁”的问题。网络上充斥着谣言以及营销性质的内容，铺天盖地，更可恶的是，这些营销性质的内容往往有着更大的流量，他们甚至舍得花钱去刷，导致真正需要信息的人被蒙蔽了双眼。

大众点评初期，很多人还是认真去点评每个店铺的服务、口味等等，还具有参考性。但是现在，你去了某一个不熟悉的地方想找一个好吃的店，如果你看点评/小红书/抖音等平台等热门或者高分推荐，八成是要踩坑。因为你搜到的热门内容，许多都是有团队来运作的，内容质量“高”而且有煽动性，没经验的人一看觉得不错可能就信了，结果过去了发现网络和现实的差距实在太大……

天下熙熙，皆为利来；天下攘攘，皆为利往。工具或者技术的进步可以帮助人去减少一定的信息差，但是在利益面前，人为的“信息差”却又无处不在。他们充斥在各大平台上，误导着一群又一群过来的人。原本向善的平台，希望能为大家服务的平台，“屠龙者终成恶龙”，变成了滋养这些内容的润土。

当然，还是存在许多的优秀内容的，并没有说一棒子打死。但是劣币驱逐良币在各个角落都存在，需要个人自己的经验和判断力，以及平台的共同努力，当大家对一个平台不信任时，它差不多也凉凉了。

门庭若市的儿科门诊

发表于 2023-10-30 更新于 2023-10-29 分类于 纪录 Changyan： 本文字数： 702 阅读时长 ≈ 3 分钟

开始只觉得周围好像多了很多咳嗽的小朋友，没想到这么快也轮到了我1岁半的儿子。当他开始咳嗽发烧时，我们并没有多在意，就把家里原来屯着新冠用的药吃了点，烧是容易退，但是会反复，而且咳嗽也不见任何好转。没办法带去医院检查一下，结果看到现场的场景惊呆了，相比于其他科室，儿科+急诊基本上全是小朋友，咳嗽的咳嗽，挂水的挂水。很多年没有见到过这种场面了，应该说对于我印象中的儿科来讲，这是头一次。

治疗的经历比较坎坷，庆幸的是我这段时间不用上班，医院跑了不知道多少回。而这次经历，让我对部分医生产生了质疑：

  • 第一次验血，医生a（副主任医师）说支原体弱阳性，开了阿奇霉素吃3停4，毫无用处
  • 第二次验血，医生b（副主任医师）推翻第一个医生的结论，说验支原体毫无用处，改打三天头孢试试（真的是试试），期间问了问题，很不耐烦不愿解答
  • 打完针三天，症状无好转，回医生b复诊，说继续再加针加药再验血，还要宝宝住院七天观察，尝试询问又被怼回去，认为我们质疑他专业性。我把单子拿了，没交钱直接出了医院
  • 老婆从邻居宝妈群得知医生c（主治医师）口碑不错，运气好挂到了号再去看，医生很有耐心看了之前的病例，知道部分药有剩，开了点新的药以及中药，一共花了40
  • 吃了一天药后，症状有所好转，状态也好了起来，准备过几天再挂他的号复查

看到这个医生，我就想起了我高中的老师们。这个医生态度好有耐心开的药也便宜效果也好，结果一直是主治医师，没有副主任更没有主任，挂号费也便宜，但是那几个让我花了上千还不知道病因的，挂号费贵搞半天也不知道是什么原因态度也差。高中我最敬佩的两个老师，教书教的好一直教书，而那些整天醉醺醺脸红晚上打牌的老师，一个个往上升，什么教务处主任副校长乃至教育局。这个现象说明了很多问题，这个社会病了，而且病得不轻。

掌握权力的人没有被好好监督的话，他们就是“人上人”，容不得质疑，这才是最可怕的地方。

这两天和附近健身房的小哥来回battle了几轮，终于下定决心办了个卡。1399两年，不知道这次能去多久，总之最近没事（不上班都可以认为没事）就每天出去游泳，争取先把本挣回来，多的就是赚了。

游泳是个好运动，即是有氧，又可全身塑形，更重要的是对于肩颈非常的友好。平时坐办公室或者宅家里玩手机，肩颈是重灾区，这次肩膀也转起来了，头也抬了，一举多得。

今天是游泳的第一晚，感觉水有点脏，期待一段时间以后的改变。

“逆进化”的小龙虾

发表于 2023-10-18 分类于 纪录 Changyan： 本文字数： 343 阅读时长 ≈ 1 分钟

最近回家和同学及亲戚吃了几顿夜宵，几乎次次都有小龙虾这个东西。其实我个人在很多年还是会吃一吃的，毕竟没吃过，贪个新鲜。大学毕业后，几乎就很少吃了。用一句流行的话来说，这种做法，把鞋底扔进去都好吃。让我感到疑惑的是一点，当年小龙虾身体还算匀称，头不大，身体细长，而且身体是主要的食用部分，为什么这么多年的养殖培育，反而变成了现在这个德行：头越来越大，身体比重越来越小，钳子里也没肉。一斤下来能有2两肉吃就不错了，加之普遍都会加水称、缺斤少两各种手段一算，一斤虾下来也就算1两多食用部分吧，点个5-6斤下来估计几个人能吃开胃，垃圾桶能装满两桶。实在是不能理解，是什么把它变成了这个样子，就等同于，果农家里种桃子，经过细心的培育，桃核越来越大，皮越来越厚，同样吃一个桃，利润率翻一倍？

别人点了我可能会因为不想浪费吃几口，但是我自己肯定是不会再点了。为什么，因为我看不起它。

蔡康永与黄渤

发表于 2023-10-18 分类于 纪录 Changyan： 本文字数： 420 阅读时长 ≈ 2 分钟

看到某一年金马奖颁奖典礼时某一段“火药味”很重的片段，暂且不说是不是剧本（娱乐圈节目常有的事），但是这个片段也堪称经典了。起因是女嘉宾与黄渤互相调侃，女嘉宾说自己盛装出席而黄渤穿的像睡衣，后者解释到，你可能好久没来了而我最近年年都来，把金马奖当成自己家一样，在自己家穿睡衣也无可厚非对吧？谁知旁边的蔡康永来了一句，“这里是我家，不是你家”。这一句触到了在场所有人的敏感神经，台下的刘德华也是尴尬而紧张的双手不自主的握紧。这个时候黄渤机智的拿蔡康永的造型（肩膀上有一只马）调侃到“我之前只看过人骑马，没见过马骑人”博得满堂喝彩，不明真相的观众可能还不理解这个梗，其实是有一个传言蔡康永的“另一半”名字叫马启仁，一语双关，跳过了这个敏感话题反而将其一军，确实挺漂亮！

看历史或者看八卦有个共同有意思的地方就是，人很容易自觉地把自己带入场景角色。我在想，如果当时是我，我会怎么回答。脑子里冒出来了一个，也许我会回答“你说的没错，但是我们两个说的‘家’范围不一样，你说的是你的小”家“，我说的是大”家“，小家也是大家的一部分嘛，分什么你我呢，大家说对不对？”

看了余华的书和记者对其的采访，很难不联想到周星驰。余华书里写的那么多苦难，让人悲悯唏嘘，但是自己却是整天乐乐呵呵；周星驰在荧幕前逗乐了无数观众，私下里却是一个孤独而忧郁的人。也许每个人的正负能量终究都是要走向平衡，苦自己吃了，快乐就给了别人；而把苦给别人，自己倒是可以释然。所以，人啊，要珍惜那些整天乐乐呵呵或者有意逗你笑的人，他们的苦可能只有自己知道。

35“危机”

发表于 2023-10-16 更新于 2023-10-17 分类于 纪录 Changyan： 本文字数： 1.3k 阅读时长 ≈ 5 分钟

不记得是从哪年开始，貌似是从阿里开始，其他公司也纷纷效仿，陆续地或”裁“或”优化“掉许多35岁以上的员工，公关水平低一点的叫”优化组织结构“，水平高一点的就自称”为社会输送人才“。把这个动作还说出了一点社会责任感，很是佩服。

回到自己身上，目前也在经历这个所谓的”35危机“，其中一点很明显就是，当你重新开始找工作的时候，你会感受到这个社会的”恶意“，你会收到很多”已读不回“，你会收到很多的”不匹配“，你会收到各式各样的拒绝你的理由，但他们不会说你年龄大。

对此我也表示理解，因为之前听到一个说法”当你工作到30几岁的时候，找工作如果还需要自己去投简历，说明你确实挺失败“。虽然很残酷，但想想好像也是那么回事。年轻的时候尽量去折腾了，缺少对自己未来的规划，这边搞搞那边尝试一下，晃晃几年过去了，没有积累到足够的“人脉”，也没有成为某方面的“专家”，还是那个“兴趣广泛，无一精湛”，且还多了房贷和娃的中年男人。如果这个时候还不好好反省，那确实应该受点“毒打”。

但是平心而论，经历的多，我个人觉得收获还是挺大。从开始的懵懵懂懂去搞数据分析，到后面义无反顾地从分析角色头铁跑去做产品经理，在阿里好好的又头脑发热去创业，再到后面做一点杂七杂八的事情，再到后来被忽悠去做自己不喜欢不擅长的售前客户工作，再到后来觉醒决定一门心思的去好好钻研深耕一个岗位一个行业，其实都是在不断摸索的过程。我认为道路虽然曲折，但是前途未必不光明。因为做过的角色多，更清楚在不同的视角看问题，知道从执行层到管理层再到上下游再到投资人是怎么看待公司怎么看待业务的，知道什么才是关键的问题，知道怎么去做团队和产品的管理。目前这个阶段，起码我自己感觉，应该是最可以在岗位上发光发热的时候，也是可以价值最大化的时候。

首先声明下，G就是我自己，看起来好像是最没着落的一个，在外漂泊，想好好看看这个世界，在刚毕业那几年，他们都很羡慕我，有胆量，敢闯，有见识，做公务员的一直抱怨每天无聊透顶，电脑不能装qq微信，每天喝茶看报纸见不到几个活人，然后每天就是看看窗外今天进出了几辆车，作为今天的工作内容……

而现在不一样了，这次聚会我感觉到明显的不一样的就是几点：

  • 原来大家不喝酒的，现在这些公务员还有烟草人员，基本烟酒都来，还喜欢劝酒
  • 他们变得“自信”了，觉得自己已经“高人一等”了，是个“官”
  • 我拿出尿酸报告说不能喝酒，对方劝酒失败，我感觉出来他们有一种“官威”被冒犯到感觉
  • 除了公务员和烟草局的香单位，其他的各有各的苦
  • 越“努力”的人，好像就越苦；被安排的，越走越顺（上一辈还是老练）

单从工作稳定与收入角度来说，确实挺羡慕他们。当年毕业时作为学校最好的学生，年少轻狂誓要靠自己闯出一片天空，拒绝各种好意与帮助，回头想想真的是挺“蠢”。到现在十几年后来看，慢慢懂得了一些社会运行的法则，开始寻求一些改变，但是这个“悟”的过程，相比之下代价就有点大。

不过过去的已经过去了，我们一直都是乐观向前的那个在路上的少年，从未停歇。前面的路很长，许多人注定会渐行渐远，但是同时也会有新的伙伴进入你的世界，专心做好自己的事情，就不会孤单。

学校不教的东西

很可悲，学校的教育到了大学，很多还是填鸭式的方式。不集中精力搞学术，非要去评比，搞综合性大学，搞一堆自己从来没有的专业，做不擅长的事情，教材用别人的，老师也没有行业相关经验与资源，大家心照不宣玩了四年，出去什么都不会，处处面壁。学校也不会教你社会各类企业的特点，各个工种需要什么能力，有哪些权利可以使用。是的，没有，都没有，我只能说，这些学校远离社会，身在象牙塔高层不知人间几何。每个高校都在喊“产学研结合”，结合了吗，除了增加一点对外露脸的渠道，多了些赚钱的方式，其他都是停留在表面而已。

回首自身，环顾四周。大家在经历某些事情的时候，都是闷着头到处找教程，自己摸索。比如什么生育津贴、比如失业金领取、比如异地就医、公积金转移等等，对于体制内的人来说如数家珍，但是对于大部分社会企业的员工，一头雾水。之前很多时候大家的印象就是，公务员钱少很累、国企都是马屁精没意思、外企资本主义剥削……导致很多人就出去“闯”，这样的结果，有少数确实闯出来了。但是大多数经不起各类风险，因为给到的保障太少了，反观之，在做公务员的，每天很清闲，可以和同事“白头到老”，国企的表面工资不高，但是福利奖金各方面给你顶到满，企业每年几百几千个亿营收没有对手，结果为了数万员工的福祉，楞是能做成亏损。没错，这就是社会主义的好处。而某些外企的就更惨了，生娃奖励10W，假期中外的都休，什么公积金比例很多也给你按最高的交，生怕惹怒了中国的员工。

像我们民企多好，为了社会经济的进步，以顾劳动法，已加班为荣，不光PUA自己，还PUA别人，卷得飞起。员工什么金都按最低标准交，多出来的还得自己凑发票，账上资金有问题的话，资金断裂融不到钱的，首先也是这些企业。但是他们还是很坚强的在生存，起码还提供一些工作岗位让一些人能活下来。疫情过后，本以为是万物复苏，结果却是后遗症发作期。亲身经历的某些行业，发现仅剩国企还活着，民企暴雷的暴雷，裁员的裁员，苦不堪言，经济危机的同时还要面对所谓的35岁危机“为社会输送人才”，不得不说，这些人都是我国的栋梁。

  • 什么样的公司福利比较好

  • 什么坑尽量不要踩

  • 什么样的公司有背景的可以去，普通人别想

  • ……

如此观之，对于张雪峰，只有敬佩。只恨当年他还没有出来，我们就草草毕业。
