现在还戴手表的人通常都是有点讲究的，而且不是手环，大部分都是有钱或有权或者家庭条件很好的那种。很久没见了，所以今天见了觉得稀奇。同时我还得感谢这位隔壁来的领导，在大家都想结束某个会不好意思打断某个领导的时候，这位领导身先士卒，在某位仍沉浸在自己世界涛涛不觉的人面前撸起袖子，连着看了两遍时间，然后做出一个意味深长的表情。另外这位情商再低也应该懂了，终于在大家的期待中散了会。

爱猫人士

发表于 2025-06-03 分类于 记录 Waline： 本文字数： 385 阅读时长 ≈ 1 分钟

在我大学毕业后，不管是同学或者朋友，周围有挺多养猫的人，大家对于猫的态度也完全不一样。

我的好朋友徐爷，当年在楼下小巷子里捡到一只野生狸花猫，养了很多年，直到他们夫妻俩出国也把它带出去，直到现在。

有一个女同事，也很喜欢猫，家里猫的数量日渐增长，看她伺候猫感觉比伺候娃还要累，但是乐此不疲，我相信她一定是很爱猫的。

在上海的静安公园，常年有特别多的猫出没，许多人都会经常带食物过去喂，导致那里的猫慢慢变得有点泛滥，有时甚至会影响到公园里的人们；同样的，在我现在的城市，有一些地方会有野猫出没，但是大家也只是路过去喂点东西然后撸一阵，撸完就走。

我有一个同事也喜欢干这事，有一天我问他，你这么喜欢猫，怎么不自己养一只呢？他说，太麻烦了，我不想养，我只想需要的时候撸两下。

这时我开了一个不合时宜的玩笑，我说你这就跟去找失足女一样，你同情她的遭遇，你喜欢她的样貌、手法和身体，但是你却不愿意和她建立一段稳定的关系，只想爽完就走，是不是有点渣？

他笑了笑，那猫和人能一样吗？

经常碰到某些事情的时候，脑子里时不时会回想出来《小王子》里的场景，当小狐狸知道小王子下午4点要来，1-3点时候它已经开始幸福了。期待，或者说希望，会给人以力量或者动力，人应该学会给自己找到并持续输入这种力量。

尚未统一的度量衡，秦始皇看了都流泪

发表于 2025-05-27 分类于 记录 Waline： 本文字数： 440 阅读时长 ≈ 2 分钟

秦始皇早在两千多年前就统一了度量衡，车同轨书同文，然而到了今天，我们还有这种情况出现：

  • 同样的尺码号，买到的衣服大小就是不同，有的偏大有的偏小
  • 鞋子也是，我正常情况都穿42，但有的鞋必须大一码，有的必须小一码
  • 菜市场，鬼秤遍地，缺斤少两成为潜规则，8两称你都得夸对方有良心

抛开菜市场这种主动没良心的，其他已经有行业的标准的东西，非要特立独行搞个性，就不明白他们脑子里想的是什么。就比如充电器，USB、Type-C、苹果的Lightening，都是使用非常广适用性强的标准。但是有些东西就偏要设计一些奇形怪状的充电口，最常见的就是剃须刀，还有某一些小家电，导致如果充电线坏了，除了重新买一根一个类型的，其他身边的线基本派不上用场，代替不了。除了恶心用户外，没什么别的作用。有些人还以为这样可以促进销量，但苹果只有一个，你没有它的命，却得了它的病，实在是认不清自己。

某些产品经理也是一样，明明同行已经给你培养好的习惯，你在某些方面照着做就可以，在别的领域去竞争去想办法超越同行，但是他却非要整点不一样的，试图强行改变用户的操作习惯，还以为自己很聪明很有创意，建议如果有老板遇到这样的员工，赶紧裁掉，不然后患无穷。

没有人会喜欢你的负能量

发表于 2025-05-27 分类于 记录 Waline： 本文字数： 375 阅读时长 ≈ 1 分钟

和周围的人沟通多了会有一个明显的感受，当你绘声绘色描述你的兴趣、展示你的闪光点时，别人也会不自觉地被你影响甚至吸引；反之，如果你一身负能量，各种抱怨，大倒苦水，除了真的关心你的人，或者和你有共同经历或者感受的人之外，大部分人是难以与你共情的。而且，这类苦水倒多了以后，很可能会引起对方的反感，毕竟除了最亲密的人，没有谁有义务来分担你的情绪。

所以，如果有不好的情绪，作为男人，其实更主要的是要学会自己排解。告诉亲密的人，他们会担心；告诉朋友，也许会间接影响对方；告诉不熟的人，别人权当吃瓜，甚至还可能会给你添油加醋扩散出去；还有的，看到你扩散的负能量，会觉得你是一个懦弱的人。

对于我来说，我平时是一个比较乐观且随性的人。当然，也会有情绪有压力。我排解的方式很简单：打游戏、弹吉他，这两种方式可以让我得到内心的平静。一个人要保持心理状态的正常，是一定要有适合自己的排解渠道。身体问题医生可以帮你治，心理上情绪上的问题，其实更多还是要靠自己。

做抖音号的一点总结

发表于 2025-05-14 分类于 记录 Waline： 本文字数： 1.1k 阅读时长 ≈ 4 分钟

去年待业在家期间，找工作的同时给自己安排了挺多事情来做的，包括学习当时比较热的AI，炒股，用代码实现一下每日的自动复盘等等，还有一个就是，做一个抖音号。因为对我来说，这类事情不用跟人沟通不用看人脸色，而且也不用过多投入，自己一个人就能干，属于最舒服的一类事情了，做得好的话还能赚钱，一举多得。

做视频其实我还是有点基础的，大学的时候虽然主课没学好各种挂科，但是选修课我可基本都是高分，当你学PS、AI、Pr，其中部分设计作品参赛还被老师打印出来展览，当时的选修设计课程我得了99，也算是大学里学到的一项有用的技能吧。那个时候也学会了剪视频，自己玩游戏还录屏发到youku上获得了一定的赞，那时不知道流量变现为何物，只觉得好像被人认可是一件很爽的事情。那时的剪辑门槛高，不像现在，剪映这类软件已经把剪辑的门槛拉到了一个极低的位置，正常人只要智力不存在缺陷，除了像蒙版一类稍微有些门槛的用法之外，2-3天怎么也能学会正常的剪辑手法了。

这里又得夸一下当时我的眼光了，当年我刚来广州第一任老板，当时做视频项目，瞄准的是一条二更这类精品视频路线，而我当时看到抖音迅速崛起，力荐老板赶紧调整方向赶紧发力短视频，之前的中长视频，一堆人要到处跑，还要出差，要写脚本文案，拍摄剪辑下来，一周也才出一条片，真的少得可怜。而短视频，当时平台有流量激励，一类30s-1min的视频，一天随便出5-6条，拍摄剪辑都能公式化，标准的流水线作业。而且试错成本也很低，这个方向不行就换一个，想不到就先抄，总会淌出来一条路的。可惜的是当时最后老板决定用团队投票的方式，我们团队里的文案、摄影、剪辑等同事又都是文艺青年，有“情怀”，觉得做那类搞笑搞怪的短视频太low，显得他们很掉价。于是在民主的投票下，最终老板决定走向那个方向…后来的后来，也算有些成绩，但是也是苦苦支撑，再经历了一段时间后终于坚持不下去，团队解散。

去年趁着有时间，我也打算自己起一个号，练练手，顺便自己来摸索一下抖音这个平台里的门道，熟悉一下规则。做什么其实没那么重要，可以做的很多，但关键是开始做起来。于是在经历了短暂的思想斗争后，我决定做英文脱口秀视频的搬运，自行翻译顺便也可以练练英语。中间的过程不细说了，总之结果就是随着我几个月200多条视频的产出，积累了1.4W的关注，对这个成绩其实我是不太满意的，但是多少也积累了一些经验。简单分享一下：

1、前5s很关键，好的开头可以让你视频获得更多的流量

2、完播率也很重要，看完视频的人越多，得到的推荐也越多（讨巧的做法，前期尽量要短，比如15S以内）

3、互动很重要，如何引导用户更多的收藏分享与讨论，也能让你的视频持续保持热度

4、原创性很重要，二次剪辑也要讲究技巧，审核通过率高

5、量不需要太大，一天一更就可以了，多了反而分散流量

6、防同行（直接盗视频顺便反手举报你，恶人先告状），加水印，没底线的人多

最近遭遇创作瓶颈，在思考原创视频的方向，如何能把号走的长远

为什么要把博客放在Github

发表于 2025-05-14 分类于 记录 Waline： 本文字数： 726 阅读时长 ≈ 3 分钟

朋友问我，为什么要把博客放在Github，让我思绪一下回到了10几年前

当年刚毕业的时候在广告公司做广告与网站流量分析，在那个时候受同事影响，决定学习自己搭建网站，以更好地去完成手上的工作。最开始是用wordpress搭建，有一撇没一撇的随便更新点内容，顺便自学一下html、css、js等在自己手上练手，并且在网站上布上监测代码以验证自己的某些猜想，那个时候还没想着自己写点什么，毕竟我从小到大最恨的就是语文

后来一段时间心血来潮，买了域名，买了服务器，决定好好地更新一下，奈何肚子里没有墨水，不知道写点啥，中间荒废了一段时间。但是成本还在，域名和服务器的钱一样要交，导致后来想重新拾起来的时候，因为续费不及时问题差点所有内容都被清除

再后来，写了一段时间公众号，但我这个人不太会拿捏文字的尺度，导致经常性地，写了半天的东西抨击一下社会不良现象，针砭时弊的内容，无一例外全部被下架，审核都过不了。一气之下直接放弃了公众号，这种被平台肆意践踏、不自由的感觉让我感觉极其恶心，于是我决定，一定要找一个“自由”的平台，不要各种乱七八糟的备案，没有平台各种主观且无理由的拒绝，于是找到了Github+Hexo+Dropbox这个组合，爱不释手。Dropbox多端同步保留我的原始文件，hexo用于生成静态网站并提供高质量的主题，github用于存放网站内容，还有什么比这个更贴心的呢。

  • 自由，不受各种管控，无需看人脸色，想说什么说什么

另外一位同事，是前司的一位博士，学历高专业能力强，这点毋庸置疑。为什么单独拿她出来说呢，因为当时我们做的项目，因为她学历最高，领导先让她来主导，这个时候弊端就出现了，她选择了一条非常学术、非常理想化的路线来做这个事情，并且把做这个事情的人员门槛都拉得非常高。导致的结果就是，花了大量的时间做一些“无意义”的事情。聪明的学会了，对业务没具体帮助；笨一点的直接学不会，进退两难。

这两位都是非常典型的“学生思维”，论做题，他们绝对是佼佼者，单打独斗绝对没有问题。但是如果是做一个商业化的产品，这种思维会成为一种枷锁，使得人过度沉溺于细节而忘却初心。

因此奉劝一下各位看官们，尤其是工作年限尚短的朋友，尽早摒弃掉这种“学生思维”，多想想公司和客户的业务目标、想想领导关心的指标，你的工作如何与这些指标挂上钩，你在职场上一定会比别的同龄人走的更快。

再到后来，发现很多时候，自己努力了，但是结果往往不受自己控制，但是还能昂首挺胸地继续前进，这就是因为有一个好的目标。目标明确了，你会想尽办法去达成，会有一种不到最后不罢休的勇气与毅力，而往往，有了这个目标以后，这件事情的成功概率是最大的！

1、高中考上大学时，没有目标，随便填了个大学与专业，惶惶不可终日，耽误了最好的青春年华

2、大学毕业时没有目标，只是被动地乱投简历，随便找了个不知道是否喜欢是否可以坚持的工作，虽然很努力学习，有收获，投入产出比太低

4、直到最近这两年，受老板提点，整日给我们灌输“目标”二字，而很多事情在围绕这两个字的基础上，虽然历经波折，但是往往都有比预期更好的效果

比如最近工作上的事情让我很烦躁，具体表现在：能者多劳，多做多错，加量不加价

因为之前在求职的时候无意中暴露了自己这个也会那个也会，导致老板一有什么不知道该谁去做的时候，总会想到我，让我去，这无形中让我多了不少工作

因为做的多，对我本职工作的进度造成了影响，同时因为某些分外的事情不太熟悉，导致前期难免会出一些问题，这就导致了，老板就会看到你出了一些问题，甚至会觉得“你本职工作做得也不咋地，让你干点别的也不会”，纯吃力不讨好

总结原因：

1、不太会拒绝

性格有些软弱，碰到强势的领导，虽然有时会跟TA硬刚，但是发现每次硬刚除了挨更多的骂之外对事情并没有什么卵用，于是乎我很多时候就用“哦”“嗯”代替。有些事情无形中就扔给你了

时不时展露一些自己这也懂那也懂（都是自己的积累，但不是为你这家公司特意准备的），显得总比其他的同事干得好，这样变相把对方的活给“抢“了，结果呢，挨了对方挨的骂，接了对方的活，还拿不到对方的一毛钱工资

有时感觉自己吃亏了就吃亏了，但是这样不行，没有人会因此同情你，反而会继续在你身上踩一脚，世界就是这么现实!

接下来，我要去多争取一些东西，不然，持续性地让自己不开心，这个工作也铁定长久不下去。

监督学习-Supervised learning 机器学习的类别，监督学习就是训练的数据有题目也有答案，机器从问题和答案中找规律，下次新的问题来了，根据规律或者从中学习到的经验/公式，通过“套公式”得出对应的结果

主流监督学习算法
算法 类型 简介
朴素贝叶斯 分类 贝叶斯分类法是基于贝叶斯定定理的统计学分类方法。它通过预测一个给定的元组属于一个特定类的概率，来进行分类。朴素贝叶斯分类法假定一个属性值在给定类的影响独立于其他属性的 —— 类条件独立性。
决策树 分类 决策树是一种简单但广泛使用的分类器，它通过训练数据构建决策树，对未知的数据进行分类。
随机森林 分类 随机森林是由很多决策树构成的，不同决策树之间没有关联。当我们进行分类任务时，新的输入样本进入，就让森林中的每一棵决策树分别进行判断和分类，每个决策树会得到一个自己的分类结果，决策树的分类结果中哪一个分类最多，那么随机森林就会把这个结果当做最终的结果。
SVM 分类 支持向量机把分类问题转化为寻找分类平面的问题，并通过最大化分类边界点距离分类平面的距离来实现分类。
逻辑回归 分类 逻辑回归是用于处理因变量为分类变量的回归问题，常见的是二分类或二项分布问题，也可以处理多分类问题，它实际上是属于一种分类方法。
线性回归 回归 线性回归是处理回归任务最常用的算法之一。该算法的形式十分简单，它期望使用一个超平面拟合数据集（只有两个变量的时候就是一条直线）。主要解决二分类问题，用来表示某件事情发生的可能性。
回归树 回归 回归树（决策树的一种）通过将数据集重复分割为不同的分支而实现分层学习，分割的标准是最大化每一次分离的信息增益。这种分支结构让回归树很自然地学习到非线性关系。
K邻近 分类+回归 通过搜索K个最相似的实例（邻居）的整个训练集并总结那些K个实例的输出变量，对新数据点进行预测。
Adaboosting 分类+回归 Adaboost目的就是从训练数据中学习一系列的弱分类器或基本分类器，然后将这些弱分类器组合成一个强分类器。
神经网络 分类+回归 它从信息处理角度对人脑神经元网络进行抽象， 建立某种简单模型，按不同的连接方式组成不同的网络。
