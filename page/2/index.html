多问问自己为什么

发表于 2025-03-21 分类于 记录 Waline： 本文字数： 912 阅读时长 ≈ 3 分钟

一直以来我都认为我是个情绪很稳定的人，有时稳定到我都觉得自己很薄情寡义。有的人觉得我很理性，碰到事情总会不慌不忙的去处理，去想办法解决，而不是肆意发泄情绪。我不知道我是性格如此还是说因为经历地多了对很多事情变得麻木。但是最近，我感觉情绪开始变得不稳定了，我告诉我自己，说时候要做些改变了。

我的经验告诉我，如果对现在的生活不是很满意，如果你不开心，一定不要闷着生气，抱怨，因为抱怨是没有任何用的。一定要想想，原因是什么？如果要改变，需要做什么？

比如最近工作上的事情让我很烦躁，具体表现在：能者多劳，多做多错，加量不加价

因为之前在求职的时候无意中暴露了自己这个也会那个也会，导致老板一有什么不知道该谁去做的时候，总会想到我，让我去，这无形中让我多了不少工作

因为做的多，对我本职工作的进度造成了影响，同时因为某些分外的事情不太熟悉，导致前期难免会出一些问题，这就导致了，老板就会看到你出了一些问题，甚至会觉得“你本职工作做得也不咋地，让你干点别的也不会”，纯吃力不讨好

因为某些其他的活给别的同事干不好，老板一怒之下把他裁掉了，然后一甩手就说全部交接给我。看起来好像是对我的赏赐？现在我一个人又多干了一个人的活，还TM不给我加工资

总结原因：

1、不太会拒绝

性格有些软弱，碰到强势的领导，虽然有时会跟TA硬刚，但是发现每次硬刚除了挨更多的骂之外对事情并没有什么卵用，于是乎我很多时候就用“哦”“嗯”代替。有些事情无形中就扔给你了

2、喜欢卖弄小聪明

时不时展露一些自己这也懂那也懂（都是自己的积累，但不是为你这家公司特意准备的），显得总比其他的同事干得好，这样变相把对方的活给“抢“了，结果呢，挨了对方挨的骂，接了对方的活，还拿不到对方的一毛钱工资

3、不太主动去争取

有时感觉自己吃亏了就吃亏了，但是这样不行，没有人会因此同情你，反而会继续在你身上踩一脚，世界就是这么现实!

举个简单的例子，现在我们同事都是理工男，性格比较腼腆的那种，之前吃了很多哑巴亏，而越是这样，老板就越肆无忌惮。我来之前，他们都是要自己带电脑的，连纸笔包括餐巾纸都要自己买，我在想，凭什么？虽然没多少钱，但是我率先开口，于是乎我的电脑是老板给的，办公室里有了公费的纸币卫生纸。

所以，权利一定要去争取！你要敢去跟人要东西，去提要求，别不好意思！宁愿让别人讨厌你，别不要让别人觉得你的便宜好占！

接下来，我要去多争取一些东西，不然，持续性地让自己不开心，这个工作也铁定长久不下去。

科技进步“造福全人类？”

发表于 2025-03-19 分类于 记录 Waline： 本文字数： 418 阅读时长 ≈ 2 分钟

ChatGTP横空出世，到前段时间的Deepseek以及炒的火热的机器人，以及现在的各种无人工厂，我们可以看到科技进步确实带来了生产力的明显提升。然而历史的经验告诉我们，几次工业革命带来的生产力的飞跃，造就的其实是更大的贫富差距，工厂主效率高成本低了，更加赚钱；而原本在产线上的工人则面临着失业与生存危机。

我们应该清晰地认识到这一点，而不是一味地相信科技进步能带给你个人一个美好的未来。就如同前段时间上海的那些二代们，听说许多韩国人来上海旅游，扎堆地把豪车开出来炫耀，旁边的打工人和外卖小哥看着那些韩国人一脸没见过世面的样子，心里居然还能生出来优越感，“看你们这些棒子，没见识，我们中国比你们想象的有钱！”。但是实际上，那帮二代的豪车跟你一毛钱关系没有，改变不了你任何命运，甚至造成的拥堵还会影响你这单外卖能不能准时送达。

认清自己，很重要；知道什么和自己有关很重要。当然，面对这些新鲜事物的出现，不要盲目沉溺在对人类社会这么大维度的乐观中，而是更应该关心，对自己有何影响，如何调整自己，或抓住机会，或未雨绸缪。毕竟，企业家们可是苦打工人久矣。

做事的方法论

发表于 2025-03-18 分类于 记录 Waline： 本文字数： 381 阅读时长 ≈ 1 分钟

最些年有一个很明显的感受，就是万事要做好，都得有一个“方法论”。以前我对有些方法论是嗤之以鼻的，觉得那都是一些所谓的专家胡诌出来的几个名词用来骗钱的。但是实际上，真正到使用的时候，你会发现有了理论的加持，实践起来会更加得心应手。

比如营销领域的4P模型、比如PMP项目管理的理论，再到最近拿项目我们内部的“三板斧”，以及给企业做业务评估的“价值流分析”，让你在做一类事情的时候都能有条理地去按一个方法来实施，效果立竿见影，而不是向无头苍蝇一样，这一下那一下的，碰了一头灰最终还把事情做了个四不像。

以前初高中的时候，我的数学物理很好，现在回想起来是当时数学老师的一句话点醒了我。他说“做题啊，其实都是有套路的，你要知道，所有的条件给你都是有原因的，一个条件推出一个结果，所有条件的结果汇总起来，就是这个题的答案“，所以当时做起数学物理题来非常的顺畅。其实生活上或者工作上的很多事情，也是一个道理，如何以现有的条件，达到最终的目的，就是我们要解开的一道道题。

机器学习经典算法

发表于 2024-03-07 更新于 2024-03-25 分类于 人工智能 Waline： 本文字数： 2.1k 阅读时长 ≈ 8 分钟
主流监督学习算法
算法 类型 简介
朴素贝叶斯 分类 贝叶斯分类法是基于贝叶斯定定理的统计学分类方法。它通过预测一个给定的元组属于一个特定类的概率，来进行分类。朴素贝叶斯分类法假定一个属性值在给定类的影响独立于其他属性的 —— 类条件独立性。
决策树 分类 决策树是一种简单但广泛使用的分类器，它通过训练数据构建决策树，对未知的数据进行分类。
随机森林 分类 随机森林是由很多决策树构成的，不同决策树之间没有关联。当我们进行分类任务时，新的输入样本进入，就让森林中的每一棵决策树分别进行判断和分类，每个决策树会得到一个自己的分类结果，决策树的分类结果中哪一个分类最多，那么随机森林就会把这个结果当做最终的结果。
SVM 分类 支持向量机把分类问题转化为寻找分类平面的问题，并通过最大化分类边界点距离分类平面的距离来实现分类。
逻辑回归 分类 逻辑回归是用于处理因变量为分类变量的回归问题，常见的是二分类或二项分布问题，也可以处理多分类问题，它实际上是属于一种分类方法。
线性回归 回归 线性回归是处理回归任务最常用的算法之一。该算法的形式十分简单，它期望使用一个超平面拟合数据集（只有两个变量的时候就是一条直线）。主要解决二分类问题，用来表示某件事情发生的可能性。
回归树 回归 回归树（决策树的一种）通过将数据集重复分割为不同的分支而实现分层学习，分割的标准是最大化每一次分离的信息增益。这种分支结构让回归树很自然地学习到非线性关系。

很多年前，我们获得信息的方式顶多是电视、报纸、杂志以及身边的人，信息差非常明显。web1.0四大门户和各类垂直网站起来的时候，我们通过他们单方面的来获得信息，虽说是来自媒体，其实背后也是个人，他们说啥是啥，但是有一点好处就是，信息量是有限的，大家还看得过来，这个时候的信息差在减小，但是依然存在，因为信息源也很有限。但是自从进入web2.0以后，开始互动了，开始由个人发表不同的内容的时候，大家就已经看不过来了，都是选择性的去看自己愿意相信的内容。这个时候，信息差就不再是“我知道，你不知道”了，而是“好多人都知道，但是我不知道该信谁”的问题。网络上充斥着谣言以及营销性质的内容，铺天盖地，更可恶的是，这些营销性质的内容往往有着更大的流量，他们甚至舍得花钱去刷，导致真正需要信息的人被蒙蔽了双眼。

大众点评初期，很多人还是认真去点评每个店铺的服务、口味等等，还具有参考性。但是现在，你去了某一个不熟悉的地方想找一个好吃的店，如果你看点评/小红书/抖音等平台等热门或者高分推荐，八成是要踩坑。因为你搜到的热门内容，许多都是有团队来运作的，内容质量“高”而且有煽动性，没经验的人一看觉得不错可能就信了，结果过去了发现网络和现实的差距实在太大……

天下熙熙，皆为利来；天下攘攘，皆为利往。工具或者技术的进步可以帮助人去减少一定的信息差，但是在利益面前，人为的“信息差”却又无处不在。他们充斥在各大平台上，误导着一群又一群过来的人。原本向善的平台，希望能为大家服务的平台，“屠龙者终成恶龙”，变成了滋养这些内容的润土。

当然，还是存在许多的优秀内容的，并没有说一棒子打死。但是劣币驱逐良币在各个角落都存在，需要个人自己的经验和判断力，以及平台的共同努力，当大家对一个平台不信任时，它差不多也凉凉了。

开始只觉得周围好像多了很多咳嗽的小朋友，没想到这么快也轮到了我1岁半的儿子。当他开始咳嗽发烧时，我们并没有多在意，就把家里原来屯着新冠用的药吃了点，烧是容易退，但是会反复，而且咳嗽也不见任何好转。没办法带去医院检查一下，结果看到现场的场景惊呆了，相比于其他科室，儿科+急诊基本上全是小朋友，咳嗽的咳嗽，挂水的挂水。很多年没有见到过这种场面了，应该说对于我印象中的儿科来讲，这是头一次。

治疗的经历比较坎坷，庆幸的是我这段时间不用上班，医院跑了不知道多少回。而这次经历，让我对部分医生产生了质疑：

  • 第一次验血，医生a（副主任医师）说支原体弱阳性，开了阿奇霉素吃3停4，毫无用处
  • 第二次验血，医生b（副主任医师）推翻第一个医生的结论，说验支原体毫无用处，改打三天头孢试试（真的是试试），期间问了问题，很不耐烦不愿解答
  • 打完针三天，症状无好转，回医生b复诊，说继续再加针加药再验血，还要宝宝住院七天观察，尝试询问又被怼回去，认为我们质疑他专业性。我把单子拿了，没交钱直接出了医院
  • 老婆从邻居宝妈群得知医生c（主治医师）口碑不错，运气好挂到了号再去看，医生很有耐心看了之前的病例，知道部分药有剩，开了点新的药以及中药，一共花了40
  • 吃了一天药后，症状有所好转，状态也好了起来，准备过几天再挂他的号复查

看到这个医生，我就想起了我高中的老师们。这个医生态度好有耐心开的药也便宜效果也好，结果一直是主治医师，没有副主任更没有主任，挂号费也便宜，但是那几个让我花了上千还不知道病因的，挂号费贵搞半天也不知道是什么原因态度也差。高中我最敬佩的两个老师，教书教的好一直教书，而那些整天醉醺醺脸红晚上打牌的老师，一个个往上升，什么教务处主任副校长乃至教育局。这个现象说明了很多问题，这个社会病了，而且病得不轻。

掌握权力的人没有被好好监督的话，他们就是“人上人”，容不得质疑，这才是最可怕的地方。
