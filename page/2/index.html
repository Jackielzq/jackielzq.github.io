关于破坏规则

发表于 2025-05-08 分类于 记录 Waline： 本文字数： 621 阅读时长 ≈ 2 分钟

规则是文明社会的产物，是其得以稳定发展极为重要的因素。作为一个社会中的个体，当然默认也要去遵守各种规则，至少，在我们接受的教育，不管是家庭教育还是学校教育，都教我们要“守规则”。当然有些时候直接变味了，变成了“听话”，某些掌握权力者把自己的话当“规则”，因而从小“听话”也被认为是素质优良的一种表现。

但是到了社会这座大学，你会发现，很多时候守规则变成了一种“束缚“。虽然很多时候我们都告诉自己，一定要恪守底线，自己认为不对的事情坚决不做，但是当你看到许多不守规则的人，反而走上了捷径的时候，心里的那种不平衡感是很强烈的。尤其是见得越来越多的时候，你会开始怀疑自己，是不是也要走一下这种“捷径”。

堵车的时候，你老老实实排队，发现旁边应急车道蹭蹭的车流从耳边掠过，你对这些不守规则的人嗤之以鼻，硬生生在排队大军中比这些人多排了许久，这个时候你会想，赶紧把这些人拍下来举报，6分200块罚死他们（好像他们也不在乎，反正一年也堵不了几次）。

打游戏的时候，你恪守底线，坚持用自己的手法来与对方竞技，发现对面一堆开挂的，你拼尽全力也动不得对方分毫，这个时候有几种选择：1、离开这个游戏，远离这种环境 2、举报，发现永远举报不完，惩罚力度太小，重来太简单 3、打不过就加入。试问你会选哪种？

我相信很多人一开始也是这样的，觉得应该做正确的事，然而由于破坏规则人因此享受到了便利，影响了他人的同时却得不到应有的惩罚，慢慢地也变成这样的人，试问谁一开始就喜欢走应急车道，谁一玩游戏就想着开挂？

就和现在某些荒谬的案件一样，如果好人得不到保护，坏人得不到制裁，那么，坏人只会越来越多，而手握规则的人则都是整个风气变坏的帮凶。

同质化的景区

发表于 2025-05-05 分类于 记录 Waline： 本文字数： 654 阅读时长 ≈ 2 分钟

这次的五一假期没有出去玩，选择宅在家里睡觉打游戏，然后和附近的朋友简单聚一下就可以，前面比较忙而且睡眠质量不行，所以这次的重点在于休息，相比于那些去景区游玩，尤其是去热门景区的人，我显得轻松惬意很多。

年轻一点的时候，也喜欢到处走走玩一下，虽然去的地方不多吧，但国内还是有相当一部分地区都留下过足迹。我个人比较喜欢自然风光，那个是许多地方的特别之处。相比之下我尤其不喜欢国内大多数的人造景区，充斥着铜臭味。去了几个地方以后，你会发现，商业街区卖的东西，大部分的都是一样的，只有少数是属于当地的“特色”。打个比方，卖文玩扇子、帽子、丝巾、饰品等等这些东西，你会感觉周庄、丽江、黄山、阳朔里卖的似乎都是一样的，可能都是made in 义乌；你会发现人气景点里所谓的“美食街”也都是那些东西，柠檬茶、臭豆腐、烤串、炸串、章鱼小丸子……去了这些地方给我的感觉就是，好像走出去了，又好像完全没有走出去…

现在国内的大多数景点商业街，基本上我怎么进去的就怎么出来，几乎不买吃的和用的。也许这个地方会赋予这里的物品某些“特殊的含义”，但是对我来说，也许在网上也能买到一模一样的，何必去挨这刀的宰。但是同时，秉持着中国优秀的传统“来都来了”，还是会尝试一下当地的特色小吃，虽然踩雷的概率也挺大，毕竟好的东西终归还是少数。

要出去看看，但是同类的”当”，就尽量别上第二次了。对于我个人来说，目前除了西北、东北，其他大部分地区都去过，留下深刻印象的并不多。现在我更希望去的两个地方，一个是我国的大美新疆，有足够的时间的话，花两周时间去好好地转一转；另外一个是新西兰，从高中时期看过相关的照片就已心驰神往，这几年一定要去一趟。

现在没有了当年说走就走的勇气，但是话说回来，人一定要有念想，要有目标，这样你才会有方向

“学生思维”

发表于 2025-05-04 更新于 2025-05-05 分类于 记录 Waline： 本文字数： 758 阅读时长 ≈ 3 分钟

前段时间这边一位同事的一个状态，让我突然有点恍神，那一瞬间让我想到了前公司的另外一位同事。首先这位同事是这样的，通过自己的数学知识和编程手段，成功解决了一个会影响我们产品功能的问题，那一刻他非常开心，就像高考时解对了最后一道那题那样开心。但那时我高兴不起来，因为它产生的额外问题会影响客户的正常使用，我得去考虑怎么把这个影响降到最低，后面得做什么样的调整。但是对于他来说很简单，我解开了这道题，证明我很厉害，很有成就感，就行了。

刚毕业的时候我其实也是那样，先不管公司的业务，也不管客户的业务和目标，也不管某个问题是否有什么坑，这个问题给到我了，我针对这个“题目”解出来了，就很有成就感，觉得自己应该会得到表扬。但实际上并不是那样，领导和客户看中的都是结果，你把一个螺丝钉拧得再好看，它也不会让这个产品卖出多贵的价格，多赚多少钱。这个同事也是我当时的想法，于是我给他泼了一盆冷水，告诉他，解决这个问题仅仅是开始，而且领导也会认为，解决这个问题是你必须要做的，并不是“加分项“。做的事情最终都能转化为客户在业务上的价值，这才是这个事情最大的意义。

另外一位同事，是前司的一位博士，学历高专业能力强，这点毋庸置疑。为什么单独拿她出来说呢，因为当时我们做的项目，因为她学历最高，领导先让她来主导，这个时候弊端就出现了，她选择了一条非常学术、非常理想化的路线来做这个事情，并且把做这个事情的人员门槛都拉得非常高。导致的结果就是，花了大量的时间做一些“无意义”的事情。聪明的学会了，对业务没具体帮助；笨一点的直接学不会，进退两难。

这两位都是非常典型的“学生思维”，论做题，他们绝对是佼佼者，单打独斗绝对没有问题。但是如果是做一个商业化的产品，这种思维会成为一种枷锁，使得人过度沉溺于细节而忘却初心。

因此奉劝一下各位看官们，尤其是工作年限尚短的朋友，尽早摒弃掉这种“学生思维”，多想想公司和客户的业务目标、想想领导关心的指标，你的工作如何与这些指标挂上钩，你在职场上一定会比别的同龄人走的更快。

京东与美团

这几天京东来了大动作，宣布要进军外卖行业，并且要给所有的外卖员上社保。国内我比较有好感的两个企业家，一个是雷军，一个就是刘强东。他们两个都算是草根出身，两个特点很鲜明：一个重视用户，为自己的客户提供性价比高的产品；一个重视员工，尤其是底层员工，让他们更有尊严更有安全感地活着。

以前我一直比较喜欢去京东上面购物，主要就是服务好，速度快，今天下单一般第二天就能到，而且能上门就肯定会上门，而不是随便往驿站或哪儿一扔就完事。目前进军外卖，也许确实是遇到挑战了，美团几百万的骑手，不光送外卖，也进军3C要吃掉京东的蛋糕，那京东只能来个换家战术，你动我主业，那我也来干外卖！你不给骑手上社保是吧，我给上！你压榨商家，大量抽成，导致劣币驱逐良币，充斥着各种预制菜或者幽灵厨房的问题，我来给用户上一层保险，而且我还不多赚！看谁玩得起！

之前我一直比较厌恶的两个平台，一个是美团、一个是滴滴。这两个都是一样的成长套路，前期疯狂烧钱补贴占领市场卷死同行，等自己一家独大以后，暴露本性疯狂吸血。外卖一旦抽30%以上，商家想要竞争地过同行，一方面要投钱占排名，另一方面还得给平台补贴，为了不亏钱，只能降低成本呢，降低品质，大量使用预制菜或者廉价的食材，平台倒是赚钱，结果用户健康出现问题，商家还不挣钱，对社会造成的影响非常恶劣。而滴滴也是一个德行，抽成甚至达到了50%，司机玩命的跑最后还是得给平台打工，一旦出现纠纷了，不管是谁的错，先罚司机。

不可否认他们都提供了一定的社会价值，让某些事情变得方便，但是当缺乏了竞争，一家独大之后，他们就都不装了，人性的恶体现地淋漓尽致。还是得有竞争，各凭本事，才能互相监督促进行业的良性发展。没有竞争的地方，只会滋生邪恶。

最北与最南

发现一个现象，刷到海南的短视频，基本上周围的群众一溜全是东北口音。我身在珠海，但是周围听到东北口音的概率也远远大于广州。给我的感觉就是，东北人一大票的都在往南边，尤其是靠海的地方转移，其中年纪偏大的比例更多。好家伙，一下从一年一半冬天直接到了一年一半夏天。

很多年前就有个说法“投资不过山海关”，东北的公务员们在全国人民的口碑跟黑社会没什么区别了已经，各种吃拿卡要把想去投资的人都整得够够的。而那些通过权力交换财富的人呢，一部分把自己的子女和财产送往国外，还有一部分去了海南，买豪宅养老，好不自在。同时过去的还有一批过去开烧烤店的，拉帮结伙，宰客，缺斤少两的事情也没少干，把原来海南本地淳朴的民风都带出了一点“匪气“。把这里变成了东北人的“第二故乡”。

当然，许多人远离家乡并不都是出去“享福”，更多的是因为自己的家乡容不下自己而被迫外出。在外地见过许多东北人，都是传说中的仗义、靠谱、热心，他们都是因为家乡没有背景关系没有合适自己的岗位才选择出来。而作为一个湖南人，其实大部分外出的湖南人（包括其他江西四川等省份）也一样，湖南教育资源很好，人才足够，但是没有一个好的土壤，没有好的公司提供足够多的岗位让他们留下来，所以才选择外出。网上流行的梗“湖南人的毕业礼物，就是一张去往广东的车票”，乍一看挺搞笑，实际道出了大家的心酸与无奈。

停止内耗

发表于 2025-04-08 更新于 2025-04-09 分类于 记录 Waline： 本文字数： 575 阅读时长 ≈ 2 分钟

内耗真的一件非常蠢的事情，当你在一个组织中，不管是公司亦或是家庭，不管是因为某个人还是什么情况，当发生了持续内耗的情况时，这个组织的风险已经很大了。

之前在的某司，本来公司的愿景很好，老板也愿意投入资源，奈何被“骗”得很惨，招了一些不靠谱的高管，高管带了一堆不靠谱的人，在公司业务上毫无建树，然而在搞事情上却到处是这些人的身影。立一些没有意义的项，做出来屎一样的产品，然后被骂，被关闭项目，然后再找个油头再干别的项目，继续浪费公司的资源。其实这样问题也不是特别严重，严重的是，别的组别的部门做的好好的东西，他们再去做一些类似或者一竞争的，再加之在上级领导面前碎嘴，导致领导把做事的这些人的活转给了那些不靠谱的群体。本来是一个团队做好能够非常好对接的东西，硬是分成了两个团队两个项目，接下来的扯皮、对接事情更是层出不穷，耗费了大量的人力物力，最后还把项目整得四不像…

在家庭里，本来各司其职，互相理解包容，一个家庭会是非常和谐。然而如果出现了内耗型成员，那么整个家庭的氛围都会被这一个人搅得鸡犬不宁。有些事情不太了解，为什么有的人就是不喜欢和谐的场面，非要去搞一些事情出来，才显得这个家庭热闹，好像自己才有了存在感，才被大家关注。也许是原生家庭成长环境的问题，而且一个人的性格一旦形成，真的很难改变。不要指望去改变谁，但是真的需要找到一个好的办法和这种人相处，怎么样才能保持一个相对和谐的状态。

内耗真的是太费心力了，把时间拿出来，去做些有意义的事情吧，而不是来折磨人。

过程 - 结果 - 目标

发表于 2025-03-30 分类于 记录 Waline： 本文字数： 668 阅读时长 ≈ 2 分钟

这些年工作的一些感悟，我觉得很重要的一点，就是你在做一件事情关注的是什么，这点也许是最重要的。

刚毕业时，觉得自己什么都不会，埋头苦干，有什么需要就学什么，想把自己本事练会，就能得到器重，到哪都能吃上饭

工作了一些年头后，发现很多事情不在于你有多聪明多努力，而是结果更重要。只要结果是好的，那么这件事情就算做好了，你管这个人到底专业不专业，人品好不好，他能谈下单子，你管他到底是怎么做的？

再到后来，发现很多时候，自己努力了，但是结果往往不受自己控制，但是还能昂首挺胸地继续前进，这就是因为有一个好的目标。目标明确了，你会想尽办法去达成，会有一种不到最后不罢休的勇气与毅力，而往往，有了这个目标以后，这件事情的成功概率是最大的！

回想了下自己的经历：

1、高中考上大学时，没有目标，随便填了个大学与专业，惶惶不可终日，耽误了最好的青春年华

2、大学毕业时没有目标，只是被动地乱投简历，随便找了个不知道是否喜欢是否可以坚持的工作，虽然很努力学习，有收获，投入产出比太低

3、工作了一段时间后，不知道自己想做什么，但是起码知道了自己不想做什么，在这个过程中跳来跳去，换了很多工作，导致没有在一个领域深耕，错过了许多更好的机会

4、直到最近这两年，受老板提点，整日给我们灌输“目标”二字，而很多事情在围绕这两个字的基础上，虽然历经波折，但是往往都有比预期更好的效果

所以啊，人一定要有目标，不管是事业、金钱、爱情甚至是玩，都需要有清晰的目标，哪怕看起来有些难以实现或者荒诞，但是当你分解成若干个小目标一个个实现离最终目标靠近时，好像一切都不是那么遥不可及了，难的事情也没那么难了。

突然想到初中的一句课文“天下事有难易乎？为之，则难者亦易矣；不为，则易者亦难矣”。关键是行动起来，这个道理老生常谈，但真正做到的人太少。

一直以来我都认为我是个情绪很稳定的人，有时稳定到我都觉得自己很薄情寡义。有的人觉得我很理性，碰到事情总会不慌不忙的去处理，去想办法解决，而不是肆意发泄情绪。我不知道我是性格如此还是说因为经历地多了对很多事情变得麻木。但是最近，我感觉情绪开始变得不稳定了，我告诉我自己，说时候要做些改变了。

我的经验告诉我，如果对现在的生活不是很满意，如果你不开心，一定不要闷着生气，抱怨，因为抱怨是没有任何用的。一定要想想，原因是什么？如果要改变，需要做什么？

比如最近工作上的事情让我很烦躁，具体表现在：能者多劳，多做多错，加量不加价

因为之前在求职的时候无意中暴露了自己这个也会那个也会，导致老板一有什么不知道该谁去做的时候，总会想到我，让我去，这无形中让我多了不少工作

因为做的多，对我本职工作的进度造成了影响，同时因为某些分外的事情不太熟悉，导致前期难免会出一些问题，这就导致了，老板就会看到你出了一些问题，甚至会觉得“你本职工作做得也不咋地，让你干点别的也不会”，纯吃力不讨好

因为某些其他的活给别的同事干不好，老板一怒之下把他裁掉了，然后一甩手就说全部交接给我。看起来好像是对我的赏赐？现在我一个人又多干了一个人的活，还TM不给我加工资

总结原因：

1、不太会拒绝

性格有些软弱，碰到强势的领导，虽然有时会跟TA硬刚，但是发现每次硬刚除了挨更多的骂之外对事情并没有什么卵用，于是乎我很多时候就用“哦”“嗯”代替。有些事情无形中就扔给你了

2、喜欢卖弄小聪明

时不时展露一些自己这也懂那也懂（都是自己的积累，但不是为你这家公司特意准备的），显得总比其他的同事干得好，这样变相把对方的活给“抢“了，结果呢，挨了对方挨的骂，接了对方的活，还拿不到对方的一毛钱工资

决策树 分类 决策树是一种简单但广泛使用的分类器，它通过训练数据构建决策树，对未知的数据进行分类。
随机森林 分类 随机森林是由很多决策树构成的，不同决策树之间没有关联。当我们进行分类任务时，新的输入样本进入，就让森林中的每一棵决策树分别进行判断和分类，每个决策树会得到一个自己的分类结果，决策树的分类结果中哪一个分类最多，那么随机森林就会把这个结果当做最终的结果。

新时代的信息不对称

都说互联网出现极大提升了信息传输的效率，让人不出门便知天下事。这个年代如果和年纪小的朋友说我们父辈当年从隔壁县进一顿钢材转手利用差价赚一两年的工资钱，他们的确是很难会相信的。毕竟现在到处都在比“最低价”，销售渠道和品牌都卷的厉害，除了拥有绝对话语权的王者，其他的都在你争我夺剩下的市场。

很多年前，我们获得信息的方式顶多是电视、报纸、杂志以及身边的人，信息差非常明显。web1.0四大门户和各类垂直网站起来的时候，我们通过他们单方面的来获得信息，虽说是来自媒体，其实背后也是个人，他们说啥是啥，但是有一点好处就是，信息量是有限的，大家还看得过来，这个时候的信息差在减小，但是依然存在，因为信息源也很有限。但是自从进入web2.0以后，开始互动了，开始由个人发表不同的内容的时候，大家就已经看不过来了，都是选择性的去看自己愿意相信的内容。这个时候，信息差就不再是“我知道，你不知道”了，而是“好多人都知道，但是我不知道该信谁”的问题。网络上充斥着谣言以及营销性质的内容，铺天盖地，更可恶的是，这些营销性质的内容往往有着更大的流量，他们甚至舍得花钱去刷，导致真正需要信息的人被蒙蔽了双眼。

大众点评初期，很多人还是认真去点评每个店铺的服务、口味等等，还具有参考性。但是现在，你去了某一个不熟悉的地方想找一个好吃的店，如果你看点评/小红书/抖音等平台等热门或者高分推荐，八成是要踩坑。因为你搜到的热门内容，许多都是有团队来运作的，内容质量“高”而且有煽动性，没经验的人一看觉得不错可能就信了，结果过去了发现网络和现实的差距实在太大……

天下熙熙，皆为利来；天下攘攘，皆为利往。工具或者技术的进步可以帮助人去减少一定的信息差，但是在利益面前，人为的“信息差”却又无处不在。他们充斥在各大平台上，误导着一群又一群过来的人。原本向善的平台，希望能为大家服务的平台，“屠龙者终成恶龙”，变成了滋养这些内容的润土。

当然，还是存在许多的优秀内容的，并没有说一棒子打死。但是劣币驱逐良币在各个角落都存在，需要个人自己的经验和判断力，以及平台的共同努力，当大家对一个平台不信任时，它差不多也凉凉了。

最近回家和同学及亲戚吃了几顿夜宵，几乎次次都有小龙虾这个东西。其实我个人在很多年还是会吃一吃的，毕竟没吃过，贪个新鲜。大学毕业后，几乎就很少吃了。用一句流行的话来说，这种做法，把鞋底扔进去都好吃。让我感到疑惑的是一点，当年小龙虾身体还算匀称，头不大，身体细长，而且身体是主要的食用部分，为什么这么多年的养殖培育，反而变成了现在这个德行：头越来越大，身体比重越来越小，钳子里也没肉。一斤下来能有2两肉吃就不错了，加之普遍都会加水称、缺斤少两各种手段一算，一斤虾下来也就算1两多食用部分吧，点个5-6斤下来估计几个人能吃开胃，垃圾桶能装满两桶。实在是不能理解，是什么把它变成了这个样子，就等同于，果农家里种桃子，经过细心的培育，桃核越来越大，皮越来越厚，同样吃一个桃，利润率翻一倍？

别人点了我可能会因为不想浪费吃几口，但是我自己肯定是不会再点了。为什么，因为我看不起它。

蔡康永与黄渤

发表于 2023-10-18 分类于 纪录 Waline： 本文字数： 420 阅读时长 ≈ 2 分钟

看到某一年金马奖颁奖典礼时某一段“火药味”很重的片段，暂且不说是不是剧本（娱乐圈节目常有的事），但是这个片段也堪称经典了。起因是女嘉宾与黄渤互相调侃，女嘉宾说自己盛装出席而黄渤穿的像睡衣，后者解释到，你可能好久没来了而我最近年年都来，把金马奖当成自己家一样，在自己家穿睡衣也无可厚非对吧？谁知旁边的蔡康永来了一句，“这里是我家，不是你家”。这一句触到了在场所有人的敏感神经，台下的刘德华也是尴尬而紧张的双手不自主的握紧。这个时候黄渤机智的拿蔡康永的造型（肩膀上有一只马）调侃到“我之前只看过人骑马，没见过马骑人”博得满堂喝彩，不明真相的观众可能还不理解这个梗，其实是有一个传言蔡康永的“另一半”名字叫马启仁，一语双关，跳过了这个敏感话题反而将其一军，确实挺漂亮！

看历史或者看八卦有个共同有意思的地方就是，人很容易自觉地把自己带入场景角色。我在想，如果当时是我，我会怎么回答。脑子里冒出来了一个，也许我会回答“你说的没错，但是我们两个说的‘家’范围不一样，你说的是你的小”家“，我说的是大”家“，小家也是大家的一部分嘛，分什么你我呢，大家说对不对？”

周星驰与余华

发表于 2023-10-17 分类于 纪录 Waline： 本文字数： 161 阅读时长 ≈ 1 分钟

看了余华的书和记者对其的采访，很难不联想到周星驰。余华书里写的那么多苦难，让人悲悯唏嘘，但是自己却是整天乐乐呵呵；周星驰在荧幕前逗乐了无数观众，私下里却是一个孤独而忧郁的人。也许每个人的正负能量终究都是要走向平衡，苦自己吃了，快乐就给了别人；而把苦给别人，自己倒是可以释然。所以，人啊，要珍惜那些整天乐乐呵呵或者有意逗你笑的人，他们的苦可能只有自己知道。

35“危机”

发表于 2023-10-16 更新于 2023-10-17 分类于 纪录 Waline： 本文字数： 1.3k 阅读时长 ≈ 5 分钟

不记得是从哪年开始，貌似是从阿里开始，其他公司也纷纷效仿，陆续地或”裁“或”优化“掉许多35岁以上的员工，公关水平低一点的叫”优化组织结构“，水平高一点的就自称”为社会输送人才“。把这个动作还说出了一点社会责任感，很是佩服。

回到自己身上，目前也在经历这个所谓的”35危机“，其中一点很明显就是，当你重新开始找工作的时候，你会感受到这个社会的”恶意“，你会收到很多”已读不回“，你会收到很多的”不匹配“，你会收到各式各样的拒绝你的理由，但他们不会说你年龄大。

对此我也表示理解，因为之前听到一个说法”当你工作到30几岁的时候，找工作如果还需要自己去投简历，说明你确实挺失败“。虽然很残酷，但想想好像也是那么回事。年轻的时候尽量去折腾了，缺少对自己未来的规划，这边搞搞那边尝试一下，晃晃几年过去了，没有积累到足够的“人脉”，也没有成为某方面的“专家”，还是那个“兴趣广泛，无一精湛”，且还多了房贷和娃的中年男人。如果这个时候还不好好反省，那确实应该受点“毒打”。

但是平心而论，经历的多，我个人觉得收获还是挺大。从开始的懵懵懂懂去搞数据分析，到后面义无反顾地从分析角色头铁跑去做产品经理，在阿里好好的又头脑发热去创业，再到后面做一点杂七杂八的事情，再到后来被忽悠去做自己不喜欢不擅长的售前客户工作，再到后来觉醒决定一门心思的去好好钻研深耕一个岗位一个行业，其实都是在不断摸索的过程。我认为道路虽然曲折，但是前途未必不光明。因为做过的角色多，更清楚在不同的视角看问题，知道从执行层到管理层再到上下游再到投资人是怎么看待公司怎么看待业务的，知道什么才是关键的问题，知道怎么去做团队和产品的管理。目前这个阶段，起码我自己感觉，应该是最可以在岗位上发光发热的时候，也是可以价值最大化的时候。

国庆和一批同学聚了一下，越来越觉得，随着时间的推进，大家的距离真的越来越远了。这些同学或朋友，都是初高中就认识了，我简单做个对比

G 父亲曾在乡政府，后政府变卖国有资产，失业被迫创业，母亲全职家庭主妇 国内各大城市四处奔波，各种不服，誓要自己掌握命运。放弃过湖南卫视与国税局的安排，换过许多工作，目前刚失业，待业在家 外地城市 一线城市小户型，还欠几十年100多w贷款 一子

首先声明下，G就是我自己，看起来好像是最没着落的一个，在外漂泊，想好好看看这个世界，在刚毕业那几年，他们都很羡慕我，有胆量，敢闯，有见识，做公务员的一直抱怨每天无聊透顶，电脑不能装qq微信，每天喝茶看报纸见不到几个活人，然后每天就是看看窗外今天进出了几辆车，作为今天的工作内容……

而现在不一样了，这次聚会我感觉到明显的不一样的就是几点：

  • 原来大家不喝酒的，现在这些公务员还有烟草人员，基本烟酒都来，还喜欢劝酒
  • 他们变得“自信”了，觉得自己已经“高人一等”了，是个“官”
  • 我拿出尿酸报告说不能喝酒，对方劝酒失败，我感觉出来他们有一种“官威”被冒犯到感觉
  • 除了公务员和烟草局的香单位，其他的各有各的苦
  • 越“努力”的人，好像就越苦；被安排的，越走越顺（上一辈还是老练）

单从工作稳定与收入角度来说，确实挺羡慕他们。当年毕业时作为学校最好的学生，年少轻狂誓要靠自己闯出一片天空，拒绝各种好意与帮助，回头想想真的是挺“蠢”。到现在十几年后来看，慢慢懂得了一些社会运行的法则，开始寻求一些改变，但是这个“悟”的过程，相比之下代价就有点大。

不过过去的已经过去了，我们一直都是乐观向前的那个在路上的少年，从未停歇。前面的路很长，许多人注定会渐行渐远，但是同时也会有新的伙伴进入你的世界，专心做好自己的事情，就不会孤单。
