别让自己的文件白白丢失—云盘的存储与同步

发表于 2013-03-31 更新于 2016-09-21 分类于 工具 Changyan： 本文字数： 1.2k 阅读时长 ≈ 4 分钟

两天前，跟我了五年的台式机硬盘突然崩溃，本来以为不可能会发生在我身上的事情确确实实地就发生了：硬盘损坏，进不了系统，数据无法恢复。有许多文件可能都不会用到但是却又对我意义非凡，丢了着实可惜。懒人就是这样，拥有的时候不会未雨绸缪，失去以后才会懂得珍惜。这让我开始反省自己之前对于个人文件存储的习惯，同时我也在寻找通过养成一种新的习惯来避免这种事情的再度发生，这时候我想到了云盘。

云这个概念是近几年从国外炒起来的，国内在这方面跟进得特别快。以前没用过的时候我是这么认为的：云盘得用网，现在这网速，把文件什么东西丢到网上存起来还不如用U盘或者移动硬盘来得实在。这在相当长的一段时间里都左右着我对云盘的看法。但是这次跟修电脑的小哥聊聊天以后总结了一下，硬件问题就和突发性的大病一样，没问题可以用的很好，一出问题就没办法挽回了。用他的原话说“几年前的硬盘能4年不出问题，现在的2年不出问题就不错了”。比如说我这次有好多以前的照片或者文档，只有一部分备份到了移动硬盘，而在移动硬盘后面却没有下一步的保障了。那么如果有一天我的移动硬盘又坏了，我的文件从哪里去找呢（相信只有高度强迫症才会每有更新都会手动去同步一次吧）？

上一节我们讲到GA收集数据的原理，现在我们聊一聊GA呈现给我们的报表中的各种数字的含义。做过表格的都有这样的概念，表格中的行标签和列标签共同决定了这个表格中数字的归属。但是维度和指标却有时傻傻分不清楚，在这里简单得跟大家解释一下。通常来说，不能量化的叫维度，可以量化的叫指标。举个例子，一个访问的来源，他的浏览器名称、版本等，这些都是维度（Dimension）；而他访问了多少次，看过多少页面，停留了多长时间，这些都是用数字来展示的，我们称之为指标（metrics）。在GA的界面中有一个比较人性化的设计：所有维度底色都是绿色的，而指标都是蓝色的，挺好区分吧。

由于GA的强大，它所提供的维度和指标数量是比较大的，为了节约时间我就不一一介绍了。维度的话认识中文字就不会有什么障碍，指标的话可能有一些计算的地方会不太清楚，这些直接问google就好（一个小提示，用英文搜的结果会好很多，英文老外本身这方面比我们早比我们先进，我们遇到的好多问题他们早已经PO上来了）。下面介绍一下基础的维度和指标吧：

互联网广告的“衰”与“兴”

发表于 2013-03-21 更新于 2016-09-21 分类于 观点 Changyan： 本文字数： 2.1k 阅读时长 ≈ 8 分钟

今天在知乎上看到有人问这样的问题：“现在的互联网广告有用吗？是在浪费钱吗？”。在这之前我也有想过这个问题，据我所看到的数据来说，从有互联网广告开始到现在，效果可以说是在逐渐下降的，但是最近这两年却又有回暖之势。

为 什么这么说呢？我们从杨致远创立的雅虎开始讲，虽然雅虎目前除了雅虎日本，其他国家或都已经在没落了，但却丝毫不能否认他在互联网历史中举足轻重的作用： 创立了门户网站的模式，内容免费，利用广告获得收入，同时也最先推出了搜索引擎。那个时候真可谓是风生水起，广告的效果也因为互联网的兴起而得到了很好的 效果，展示广告普遍都有1%的点击率。然而回到现在，普通的展示广告大部分都只有0.1%的水平（当然，部分位置醒目尺寸大的除外）。因为现在的网站太多 了，各类门户网站也太多了，广告也是眼花缭乱，让人下意识地对广告产生了抵触情绪，对于熟悉的网站甚至能够做到非常熟练的避开各种各样的广告，所以这两年 不免让少数广告人喊出“互联网广告已死”这种口号。
