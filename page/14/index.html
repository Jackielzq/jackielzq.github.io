视频网站的贴片广告

发表于 2014-03-06 更新于 2016-09-21 分类于 观点 本文字数： 1.3k 阅读时长 ≈ 5 分钟

最近看到知乎上好多人讨论视频网站广告的问题，我逐个看了下帖子和评论，基本90%以上都是在说视频网站多么没节操，广告多么烦人。前段时间傲游浏览器的“支持快进”功能快速跳过视频广告掀起了一小段波澜，大多数人都在力挺傲游，对于这个现象我并不感到意外，但是并不代表这90%是正确的。

我顺手查了下最近的相关数据，截至2013年12月，中国网民规模达6.18亿，互联网普及率为45.8%。其中，手机网民规模达5亿，继续保持稳定增长。而视频网站覆盖人数为4.6亿，也就是说74%的网民都会浏览视频网站，网络视频已经成为绝大多数网民不可或缺的一部分。而支持视频广告存活的衣食父母，基本上只有广告主，最近优酷土豆的财报显示，其90%的收入都来自于广告，而内容成本、带宽成本一直居高不下，要想增加营收最简单粗暴的方法就是增加贴片广告时长，所以优酷土豆实现里程碑式的首次盈利，和这有着莫大的关联。

发表于 2014-03-06 更新于 2016-09-21 分类于 纪录 本文字数： 1.2k 阅读时长 ≈ 4 分钟

两天前我终于看了期待已久的艾薇儿演唱会，一直以来我从不掩饰对她与其歌曲的喜爱，有时会和朋友开玩笑说“我是VV的脑残粉”。但是这次以后我发现，我错了，比起真正的“脑残粉”，我充其量算个“伪粉丝”。 为什么这么说呢，主要是因为此次杭州演唱会，快到尾声时音响出错，主办方束手无策。艾薇儿一脸尴尬勉强继续唱完了那一首歌便匆匆离场，一个简单的道别仪式也没有。主办方直接广播“本次演出完美结束，请大家有序离场”，惹怒了在场大半的观众，我也不例外。主办方傲慢的态度让人难以忍受，始终不承认是他们的失职，也不承认这是演出事故。现场的观众普遍素质都很高（以大学生为主，有许多浙大同学），没有肢体冲突也很少有污言秽语。跟工作人员交涉半天没有结果，全场两三千人齐喊“退票！退票……”这种场面我还只是从网上看到过，某个足球比赛全场齐唱“裁判是SB,咚咚 咚 咚咚，裁判是SB……”没想到我平生看的第一场演唱会就撞上了，不知道是运气好呢还是点背。

2013总结-写给自己

发表于 2014-01-01 更新于 2016-09-21 分类于 纪录 本文字数： 2.6k 阅读时长 ≈ 10 分钟

2013已经过去了，时间总在我们不经意间匆匆溜走，但幸好仍会留下痕迹。昨晚在好朋友家里一起看跨年晚会，才发觉张含韵已经出道10年了，而在我印象中她还是那个唱唱跳跳“酸酸甜甜就是我”那个大舌头小姑娘。大学毕业以后心态慢慢变得不一样了，但我在努力做那个最真实、最好的自己。今天写下这篇总结，是对自己的一个交代，也是对下一年自己的鞭策吧。

这一年，我做了什么

  • 今年一月8日，我从我任职的第一个公司离职，一月九日开始在新公司上班。从上个公司离职的原因，一个是我觉得我对当时我需要做的东西已经比较熟练，导致上班时有较多空余时间学习新的东西，但是在学习的过程中没有项目或者机会来实践，在这个公司可以做到；二是原公司很少有机会接触客户，属于闷头干活但不知道对方要什么或者有时候抓不住重点；三是比起广告投放，我更想知道用户在网站内的行为，更能看到真实的数据，那是我的方向；第四是职(qian)业(tai)发(shao)展(le),上(wo)升(yao)空(chi)间(fan)大(a)。

  • 3月19日，我放弃更新之前jackieluo.info那个博客，从Godaddy淘到了一个中意的域名，购买了一个香港空间,从新开始了博客之旅。花钱的通常都比免费的要好，这样也是督促自己勤更新，不要浪费自己花的这个钱。现在回头看看，其实更新的挺不勤快的，质量也都不好。一个是因为懒，还有是因为肚子里东西少，再一个就是有点私心，有些东西觉得舍不得一下拿出来分享。我还是比较崇尚自由分享的黑客精神，本身许多东西也是在别人的博客里学的，所以我今后会放开手脚更新的。写一篇好点的博客其实是很花费时间的，有时我会想拿这个时间不如学学coding，但是现在想起来，写东西一是最自己知识的总结，二是一次重复学习的机会，因此我今后尽量平衡这个时间。

  • 五月-六月，那段时间加班比较多，为了准备职业生涯的第一个Presentation，一开始让我有些束手无策，一是本身没有讲过，不知道从何下手；二是对方level比较高，害怕自己hold不住场面。开讲时比较紧张，中途声音还有微微颤抖，但总算还是顺利完成了这个任务。我很感谢有这种锻炼机会，过了这一关之后我内心在迅速地强大，到现在为止经历了各种大大小小的会议，和客户，和第三方公司，已经非常从容了。* 9月，参加公司组织的outing，今年是去的厦门，虽说比去年的巴厘岛要降两个档次，但是在某些方面，我感觉这次旅行带给我的东西更多。巴厘岛确实是梦幻般的天堂，阳光、沙滩、游艇、椰林树影，但时间太赶，没有时间好好品味。而在厦门是自由行，我果断“抛弃”了其他的同事，好好享受属于自己的旅行。我从不指望旅行能给心灵带给什么洗涤，也不齿那些所谓的“朝圣之旅”(骑车去西藏…)，我不是不同意他们的做法，也许因为我本身没有信仰，中国一直以来的佛学？我以为他只是再西游记出现过，仅当神话故事看。一个人朝圣归来，日子仍然是一样的过。改变？兴许会有吧。旅行之于我来说很简单，看到不同的风景，了解不同的生活方式，品味不同的美食。也许是与生俱来的好奇心吧，作为一个双子座，对新鲜事物的确有很强的好奇心，恨不得自己马上就弄明白，恨不得自己很快掌握。所以在厦门，我环着海岸线骑从下午一直骑到天黑，晚上在各种美食街窜东窜西，跑到海边一边孤独地玩沙子，一边羡慕那些在海浪冲击下其乐融融的家庭与情侣。* 上个月，我提出了离职，不是因为我在个公司不满意，相反，我对此充满感激。但是有些东西比工作更为重要，我的女朋友毕业了，和我不在一个城市工作，女孩子不同男孩，孤独和寂寞感会被她们的多愁善感所放大。我想如果我在身边，她应该会安心很多。虽然我也曾经说过自己想去的城市，想带她一起去，最后我做出了妥协。家里的朋友们表示不太理解，因为在我们老家，基本都是女随男。虽然我非常羡慕，但是人生苦短，这有什么关系呢？ 最近，入手了Macbook Pro，今后可能不怎么玩游戏了。”倾家荡产”买这个，主要是见识到了它功能的强大和对效率的提升，做出这个投资我觉得是很必要的。人生苦短，我用Mac，不解释。

这一年，我学会了什么

  1. 对专业知识更加深入 因为单独负责了许多项目，对监测的部署，数据的采集和计算方面，自己做了许多尝试，并取得了一定的成效，这种成就感堪比解出高中的变态物理题。2. 内心变得强大 遇到事情不要慌乱，理清思路有序执行.勿做老好人学会拒绝 做事主动，该问就问，该催就催，不要坐等

  2. 思维更加有逻辑 主要是开始写PPT和写博客，当你的东西展示给别人看时，得让别人能顺理成章的看下去并让他能看懂。讲故事而不是列数据、讲报告，这种能力是还需要修炼的，但是我可以感觉到自己和之前的不同，比如我以前写文章喜欢balalbala一段归一段，想到哪写到哪，现在更多的用1234。比起高手来我还有距离，但是起码起步了，对吧。

  3. 鼓捣了许多有趣的东西

  • Mac(Alfred、Homebrew、其他发掘中)* Sublime Text(性感的编辑器，写Markdown、python、html)* HEXO(在Github上快速搭建博客)* Wordpress(扫了一遍Wordpress各种文件结构，方便做实验)* 马克飞象(用evernote的福利)* python(做了点练习，水平仍然初级，继续努力)* XAMPP(凑个数) * 等等……(欢迎推荐)

有兴趣玩的，你知道怎么去找

  1. 认识了许多新同事、朋友 “内心强大”的附带效应,之前那个内向的开始向”自来熟”转变了，和同事之间更有聊更能扯了。在这个压力较大而普遍”苦逼”的行业，同事之间”何以解忧，唯有扯淡”。和之前的同事关系也还不错，听说我要换工作，好多人都来给我推荐，对此我感激涕零。有一些机会接触过没有成功，是自己表现的不好，生怕败坏他们的名声。以前校招参加笔试的时候基本没怎么做以前的题目准备，都是直接上阵，觉得那才是自己的真实水平，现在想想真是太幼稚了，你是真实水平不做题，别人题目都做烂几遍了，Too Young Too Simple!很多时候决定结果的是态度，而不是第一眼看到的能力，许多我们看到的真实往往都不是事实，这就是现实。最近自己在整理一些面试上的问题，好好准备，不给你们丢脸！

想对朋友说

我是一个比较随意的人，应该也是一个有趣的人，但同时也是比较”没心没肺”的人。为什么这么说，因为我很少主动给老同学老朋友打电话，短信也懒得发，但是这不代表我们关系疏远，也不代表我没有想你们。和我关系好的人，就算几年没联系了，一回到家见面，我们还是能跟当年一样无话不谈，也许这就是我们的方式。我不善于主动表达这些情绪，以前甚至会想，有些话说出去会不会显得太娘、太掉价，直到现在还是这样，多年的惯性吧。但是我相信，你们会懂的。 自认为比普通人还是多一些幽默感的，也很喜欢损人，这是从初中以来就有的”病”，也许是受了星爷电影的部分影响吧。但是我想说，如果我损你，那么我已经当你是自己人了，二者程度成反比。

最后，如果你还在上大学，这些话送给你

  • 要么学好你的专业 要么发现并坚持自己的兴趣,不然你毕业以后会不开心，当然，如果你是土豪，这(wo)句(men)话(zuo)当(peng)我(you)没(hao)说(ma)* 好好谈一场恋爱 学会怎么和对方相处，怎么解决矛盾，怎么培养默契，和他/她一起计划将来* 好好思考自己今后的方向 今后要做什么工作，还是看别人投什么我投什么，决定权不在你，但选择权在你* 自我节制 玩游戏，当然要玩！确实能让大脑更灵活 但是不能耽误正事(我就是一个活生生的例子，差点毕不了业) 现在事后诸葛亮，事前猪一样，我希望你们能比我更早醒悟* 真实，不要伪装 按字面意思自己理解

不知不觉写了将近三个小时，我个人是不喜欢熬心灵鸡汤的，但这些也是我最真实的体会，如果正好对你胃口，那么干了这碗鸡汤，开始努力吧！

重新认识自己

发表于 2013-12-27 更新于 2016-09-21 分类于 纪录 本文字数： 498 阅读时长 ≈ 2 分钟

在这边工作将近一年，从一开始的各种纠结头疼到现在的游刃有余，个人觉得进步还是比较大的。工作上可以独当一面，人也有些膨胀起来，这种时候被泼泼冷水最好不过了。

上午接到一个电话面试，虽然我一直说个没停，但同时也感觉答的不好，条理不够清晰，重点不够突出，也许觉得说个不停才能掩饰自己的紧张或无知？觉得自己准备好了，但实际上根本没有。让我说我之前做的项目、流程、遇到的问题、如何解决这些执行层面的我都可以滔滔不绝，因为自己是一步一步紧跟着做过来的。但是遇到这样的问题：“你在整个项目中的角色是什么，你的关键作用是什么，你是否无可取代，你在哪方面有独特的优势？”，突然不知道怎么回答了。还好有良师益友从旁指导，才发现自己对自己的认识，对自己在工作中的地位或者作用，想的完全不够。重要的不好好想，不好好总结，不重要的东西却喜欢瞎折腾，玩花样，确实有些误入歧途。

趁自己现在清醒一些，好好思考下面这几个问题：

  1. 如果执行一个项目，你在其中担任的角色是什么？
  2. 你做出了哪些贡献？
  3. 如果换了别人，是否能做的和你一样或者更好？
  4. 你相比别人有什么独特的优势？
  5. 你觉得自己在哪些地方需要改进？
  6. 你的专业技能在现在是否够用？如果不够你会如何解决？
  7. 未来你努力的方向是什么？你能做到什么高度？
    带着这些问题睡觉，下次过来回答自己的问题

我们眼中的“他”和他们眼中的“我”

不知不觉毕业已经两年，一直觉得离校园生活其实还不是太远，仿佛就在昨天。也许是因为住得也离学校近吧，加上还有几个同学在学校读研，说话吹牛还是以前的调调。但是最近觉得变化比较明显的还是收到一个消息：我的堂妹要结婚了。我的堂妹，在我眼里一直还是那个初高中的样子，还没长大的感觉，怎么就结婚了呢？这同时让我想起上次去上海玩，和我表哥和嫂子一起吃饭的时候，我表哥第一眼见到我，看我和女朋友一起，说话谈吐都不在是他眼中的那个我了，显得非常“吃惊”，就如同看到我堂妹的感觉一样，也许他心里也在想“这小伙子高中还没毕业怎么就谈女朋友了？”。因为大家不是经常见面，所以他们眼中的我也许一直还保留在几年前的那个阶段，同样，正如我眼中的他们。

自从上大学以后，远离家乡，和之前的亲戚朋友沟通都甚少，原来的好朋友还是挺多，虽然都不怎么见面，偶尔在往上聊天扯淡，但是感觉却从未生疏。每年过年回家，还是一样的人员配置，还是一样的活动，热情感觉稍减但是情谊却依然还在。我还是比较相信那个理论：“真正的好朋友，就算长时间不联系，当再次见面时，还是能好若当初”。每个阶段都会有新的好朋友，到了下一个阶段再有新的朋友时，再见到原来的朋友，感觉应该也不外如是。到最后最了解自己的，还是自己。只有自己知道自己经历了什么，有什么变化。突然有个想法，想知道初中同学，高中同学，大学同学，前同事，现同事对我是什么样的印象，是否和我感觉的一个样，改天去调查一下好对自己做一个重新认识。至今还记得以前的同学录上的部分内容，小学的同学录写的是“祝你考上好的初中”，“学习进步”这类非常励志的话；初中则是“你人很聪明，但是你为什么这么黑啊，祝你早日变成黑煤球”，“祝你成为魔兽之王，称霸CS界”（这两段话背景：1、初中时候在家里待不住，成天出去玩，因此晒得特别黑；2、打游戏还比较厉害，在学校还小有点名气，没少被老师罚俯卧撑写检讨）。去年年间同学组织了一个小学同学聚会，我见到了时隔十多年的小学同学，让我感到震惊的是：我觉得除了我自己，其他人都没变，还是原来的样子，该胖的胖，该瘦的瘦，只是变成了放大版，不知道其他人会不会有同样的感觉，因为自己的变化还是自己最清楚。

一个什么样的人

发表于 2013-08-25 更新于 2016-09-21 分类于 纪录 本文字数： 1.1k 阅读时长 ≈ 4 分钟

今天偶然跟朋友聊到一点生物的话题，讲到基因杂交的问题时，我想起一道让我非常难忘的生物题目，大概意思是有几只某某显性隐形基因的狗，如何最后得到某某性状的后代。乍一看都觉得这是一道普通的生物题目，但是做这道题时让我感到不快：从基因学角度来讲，有比较简单的方法来得到最后结果，但是这中间会有一个问题：子代与母带杂交。从实际或者伦理方面，我实在是无法接受这样的答案。因为这不是见得最多的豌豆，而是动物，当时考虑了很久，最后在答卷上写了一个非常复杂的过程，这个杂交过程中只会出现同辈，虽然也是有近亲的部分。想想当时一直有这么固执，坚持己见，不会去想这会对我的分数造成什么样的影响。

最后考试结果出来了，班里绝大多数同学这道题都很轻松地得了满分，而我，虽然版面写得很满，也得出了最后的结果，但是因为过程比标准答案的要多出了好几步，因此还扣了好几分。当时胆子比较小，不会去争论分数上的这些问题。事后老师和几个同学都私底下问过我，为什么写得这么复杂，这么简单的分都不要。我跟他们说了一下原因，他们无一例外的哈哈大笑，跟我说”你想多了”。现在回来想想，也许真是我想多了，没有单纯得把他当一道学术题目来做。但是不知道为什么这件事情在我脑海里的印象一直非常深刻，到现在许多年了依然十分清晰。也许出题老师们真的应该多下点功夫，以免遇到像我这样的人，而且这个题目还能对他产生这样的影响，因为这对于他来说可能就不只是一道题目，更像是一个等待被解决的问题。
