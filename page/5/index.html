早起

发表于 2015-05-19 分类于 生活 Waline： 本文字数： 1.3k 阅读时长 ≈ 5 分钟

自大学毕业，稀里糊涂入广告一行，几乎没有早睡过，更不谈早起。上班比别人晚1小时，代价下班比别人晚2小时+。上班不打卡，加班也是应得的，不用提加班费这个词。这个行业虽然钱不多，但是成长地很快。果不其然，年方26的我，看起来已经像30左右的人了，挣的钱虽然每一次跳槽都翻了一倍左右，但是基数小，泪多不谈。也许真的是时间让人成长，好多以前觉得无所谓的东西，现在开始重视了：比如说外表，比如说健康，比如说兴趣。

看看自己以前的照片再照照镜子里看现在的自己，好像“天生丽质”这个词跟我没什么关系。虽说以前长的还可以，没有脱我们家里的后腿(我爸妈当年用俊男靓女来形容绝对无法反驳)，我姐当年在学校不说TOP3,也得在TOP5之列，追她的男生一大把，马屁拍到我头上也让我得了不少小恩小惠。虽说“身体发肤，受之父母”，但是我认为这句话的有效期间仅仅在于出生到大学毕业。父母在这段时间供你吃穿，使你的基因得到了正常的施展，长成了你本身该有的样子。但是过了这段期间，你的外貌就主要是自己来控制了，不管是表面的保养还是内心的修炼。一年多以前我才开始正式用洗面奶，犹记得第一次用时那真切的感受，感觉在这之前好多年，我的脸上一直套着一层由油脂构成的“天然面膜”，洗掉以后，才算真正触摸到了自己的皮肤。然而一年下来，发现光用洗面奶好像并没什么卵用，于是我想到了调理。

其实说调理不太全面，这包括身体的锻炼和饮食起居的调节。网上查了好多相关的资料，也阅读过一些书籍，幡然醒悟，其实大道至简，许多简单的东西古人已经说了上千年了。“早睡早起，方能养生”，玩过三国杀online的就知道，华佗也确实有这么说过，只是人们在听的时候，也仅当台词去听。所以为什么说“我听过很多道理，但还是过不好这一生”，知易行难。这段时间开启了早睡早起计划，感觉像换了一种人生在过：以前1点睡，即使睡到9点还整天犯困，现在11点半睡，7点起一天精神抖擞，貌似把加班“亏”到的时间给“挣回来了”。以前老觉得晚上9，10点回到家，时间才开始属于自己，不好好上网干干别的，怎么对得起自己。而现在早上7点起来，运动+洗澡+吃早点，也才8点出头，上班以前还可以干好多事，比如洗个衣服，比如像我这样，一边消化刚入胃的早餐，一边端着笔记本胡思乱想……

最近刚看完了这本比较热门的书，可以用一个词来形容，那就是-脑洞大开。这本书讲述的是作者和数位精神病人的访谈记录以及自己的一些看法，不管其内容真实性如何，但可以肯定的是，它让我对周围，对这个世界有了不一样的看法。对一些事物的理解仅停留在之前受过的教育范畴内，以为一些“常识”就应该“理所当然”，没有用其他的角度来看这些问题，甚至是一些看似荒谬的理论的可能性。这本书让我感觉到了思维的乐趣：

1、通常我们认为蚂蚁是群居动物，一堆工蚁供养着蚁后，维系着这个氏族的延续。但从一个“精神病”的眼里，一个群族才是一个完整的个体，每个工蚁就类似于动物体内的红细胞，他们在外界运送着养分，而蚁后则是大脑，支配着他们的行为，蚂蚁的队伍就像是血液在血管内流动，维持着这个整体的生命，而脱离了这个群体，任何的工蚁都不能独自存活。一种即视的画面感，我突然间被这个画面带过去了，如果说，这是真的呢？

生命的意义

发表于 2015-02-28 分类于 生活 Waline： 本文字数： 1.9k 阅读时长 ≈ 7 分钟

就在年前，我的爷爷彻底离开了。刚得到消息时心情很复杂，同时也很平静。近两年他的身体差了不少，几次打电话连说话都喘不过气来。而这个情况家里的亲戚都基本上有了心理准备，虽然事情发生的仍然比较突然，但是大家依然是默默接受了这个事实。

而现在，我更多的是怀念与惋惜。小的时候，我和爷爷都有一个共同的爱好——喝白粥。当时我和爸妈住在一栋房子，爷爷住在隔壁的一栋房子，每天早上在鸡打鸣之前，我都会自觉地起来去敲爷爷的家门，催他生火，熬粥。如果他和奶奶还没起床，那么我就自己去灶头点火，劈柴，加水热锅。用那种大铁锅熬出来的白粥，香气浓郁，口感极佳，加上几勺白糖，再配上一点锅沿上烫出来的锅巴，吃起来更有味道。那个时候最温馨的画面就是爷孙俩坐在餐桌上，伴随着公鸡的打鸣享用美味的早餐。说句实话，到后来这么多年我喝过好多种粥，但是没有一种能和这个味道相比，是的，没有一种。随着我年龄的增长，我和父母搬到了县城，每年回家的时间越来越少：小学初中，每个寒暑假在爷爷那住上一大半的日子，夏天爬梯田、放火熏田鼠、挖泥鳅捡田螺，到山里摘杨梅，冬天放陷阱抓野鸡兔子、放鞭炮、挖竹笋；到了高中，每个假期能住上一周也算多了，每天看看电视，打打牌；然后上了大学，每年回家待的时间就不多，回去看他们老人家的时间也更少了，也许一年也就那么几天，有时觉得待着特别无聊，新号不好玩不了手机，没有电脑不能玩游戏；工作以后，这个时间也只会少，不会增，在家基本上活动就是和亲戚朋友打牌，然后坐等吃饭。与他们之间不管是沟通还是感情上的依赖，都越来越少。

那个时候起我明显开始感觉到了他们的孤独感，所以大学以后每年回家，我都会刻意抽出一些时间来陪爷爷聊天。聊的内容基本都一样：火车、铁路、知青，想到这里不禁唏嘘，他的一生唯一的“光辉岁月”也许就是他当兵的那几年。从小便独自一个人守着一栋大房子，16岁响应国家号召去当兵，在广州军区成为了一名侦察兵，由于表现优秀很快当了班长，那几年不愁吃穿还能领粮票布票(年纪小一点的应该对这两种东西没概念，等同于买米和步的专用货币)，坐火车可以优先，每年数次免费乘坐……直到后来他退伍还乡，回到了我们这个小山村，一切都变了：每天周而复始的农活，娶了奶奶以后还要养5个孩子，每天都在愁明天，愁他们吃穿教育，还要愁他们婚娶……这么多年就一直在生存，没有生活。最近这些年我爸和几位叔叔都过上了像模像样的日子，但是他依然保持着这么多年生活的“惯性”，依然下田，砍树，干农活，而不愿意到城里来“享清福”。唯一看他开心的时候就是有人陪他说起过去的故事，我真心地感觉到他的开心，因为那个时候，我从他的眼里看到了光芒。也许是其他叔叔婶婶听的太多，每次他聊到这个话题他们都会有意或无意地慢慢离开现场。本来回家就少的我因为心生愧疚，就负责静静地听他重复当年的故事，尽管每年听的内容都大同小异，但是比起他一个人默默黯然神伤，我更愿意看到他神采奕奕地夸夸其谈。

锻炼与健康

发表于 2014-12-26 分类于 生活 Waline： 本文字数： 969 阅读时长 ≈ 4 分钟

一直觉得自己抵抗力还是比较强的，最近周围同事一个个被感冒病毒侵蚀后，我也终于败下阵来。虽然难以幸免，但是较之以前的的感冒经历，还是觉得比以前的抵抗力还是要号很多。回想一下，上一次注意锻炼身体还是在大一的时候，加入了学校的跆拳道社，每周两次高强度的练习，最后给我留下了三个印象：一是大一我体育拿了老师所称的“理论上不存在”的满分；二是当时的腹肌捏着跟砖头一样硬；三是那一年没有病过。但是跆拳道练习必须得常年保持高强度的练习，否则退化地会非常明显。而大二下学期以后由于自己沉醉于魔兽世界、真三等游戏，最后直接放弃锻炼了。从那以后，我的体重以10斤/年的速率匀速增长，到毕业的时候，体重已经保持在了150。虽然有180的个子，看着不胖，但是小腹凸显的那块肥肉明显已经占据了有利地势，仿佛之后的体重增长全部负担在了它的身上。

毕业之后，现在已经经历了三家公司，在互联网广告这行，加班是一个惯例。而就我的经历来看，这三家的加班是一家比一家狠。之前老妈偷偷跑去找神婆给我算命，知道我比较反感这个，就用试探性的语气跟我说“你的相啊，30岁前都是劳碌命，得去南方，30以后会有作为”，之所以她比较相信这位“神婆”，因为她成功“预测”了我高考会失利，复读一定能上一本，这么看来目前来看还挺准的呵，但是这位“神婆”怎么也没算到自己最后竟然被自己老公拿刀砍死了，听起来确实有些讽刺。其实累一点倒无所谓，关键是对自己有收获，但是底线是自己的健康得有保障。

这几年观察过来，虽然周围大部分人都知道“身体是革命的本钱”，但是真正注意的却寥寥无几，大部分都是在透支自己的身体(当然，除了公务员或者国企的同学)。在北京的时候，偶尔早一点下班，一个小时的地铁回到住处，换好跑鞋骑20分钟单车道最近的学校操场，还得呼吸着着雾霾跑步，感觉也是在自己作死，最后也放弃了这种锻炼的方式。现在虽然加班也多，但是我始终保持了每周3-4次的健身房锻炼。也许这也是成长的一部分吧，变得对生活更加认真，对自己更加负责。虽然到这个年纪再来说“成长”这个词，似乎显得有些不合时宜。但是当我越来越感觉锻炼对我的身心带来的好处时，我反而越来越对之前这么多年的懒惰而感到自责。“随心所欲”、“随便”这种生活态度已不能承受生命之重，因为到头来影响的是自己，后悔的也是自己。

不敢登录的搜狗拼音

今天偶然发现，平时用惯的搜狗拼音竟然是可以登录了伴随着好奇心我通过QQ账号授权成功来到了个人界面，心情是比较忐忑的。首先它记录了你在这台电脑上输入的所有信息，我的台式机一看也由几十万字了，然后balabala给了你一堆勋章，给了你好友中的排名。换做是以前，我也许还会有一些欣喜，心想，我还真NB，都打这么多字了。但是接触得多了，想的也就多了。像搜狗说的，根据你历史的输入情况，找到你的习惯，最快时间匹配你想要的词。乍这么一说还挺有道理，但是接触下来我发现对我本身的输入其实是没有多大帮助的，相反我非常怀念当年的智能ABC，虽然说都是固定的位置，但是很多字我都清晰地记得那个字在第几页的第几位，真正实现了盲打。通过初中偷偷跑网吧玩网游的成果，高一打字速度直接秒杀了计算机老师。有一个简单的打字游戏，他刚秀出他的记录让我们挑战一下，我几分钟内就创造了新记录，他一直没办法破掉。而后来它的没落让我换上了搜狗拼音，老实说一开始真的很难适应，这一次我打zc，第一个出现的可能是“正常”，第二次可能第一位是“这次”，再下次可能就是“支持”，让我每次必须在结果中挑选，也就是说我每输入几个字母都必须去瞄一眼，然后再做选择。整体来看应该对不少的用户来说都是更加方便的，也许我是属于比较奇葩的那一类吧。

话说回来，它记录了你历史输入的内容，这部分是非常宝贵的“金矿”，我想除了QQ以外，也就搜狗能拿到这么丰富的用户数据了。可以想象在不久的将来，当我和朋友聊天说，“我想去滑雪，咱们组队把”，然后几秒之后，我随便打开了一个网页，上面醒目的位置出现了“北京南山滑雪场”的banner，仿佛是为我量身定做的一般。想到这种场景，作为业内人士当然会觉得这种技术的进步带动了精准广告的发展，是一件好事。根据我输入过的几十万字，可以分析出我的各种属性和偏好，根据我的需求来推送相应的广告，这个现在的广告技术公司都是可以做到的，算法会比较类似，关键的差别在于数据源。BAT有你的聊天内容、搜索行为、购物行为数据，其余广告或媒体公司能监测你在互联网上的内容访问行为，每一家都能对你有一个独特的认识，但是输入法工具可以挖到更深的级别，如果用现实和虚拟做区分的话，输入法就相当于现实中在你身上安装的窃听器。所以作为个人来说当然觉得有些后怕，想想我几秒前刚聊到滑雪马上就能看到滑雪广告，仿佛无时无刻不被后面的一双眼睛盯着。他知道我想要什么，讨论什么，我在人前人后说的话是不是风格迥异，我是不是有打字说过某人坏话……可能我打完字过后就忘了，然而它都能帮我记得。这些内容可比我在QQ聊过什么，在百度搜过什么，在淘宝买过什么更加隐秘。如果什么时候我脑抽地说了一些不合时宜的话，搜狗把这些数据作为呈堂证供，我真是百口莫辩。

想到这里我马上退出了登录并清掉了相关的记录，不是说我背地里是多么阴暗，因为每个人都有阴暗的一面，这个不可否认，但只要不影响到别人，总归还是可以接受的。但是宁愿我把阴暗留给自己，也不要给他被曝光的可能。

对我影响最深的老师

发表于 2013-09-18 分类于 生活 Waline： 本文字数： 1.2k 阅读时长 ≈ 5 分钟

不知不觉毕业已经两年，一直觉得离校园生活其实还不是太远，仿佛就在昨天。也许是因为住得也离学校近吧，加上还有几个同学在学校读研，说话吹牛还是以前的调调。但是最近觉得变化比较明显的还是收到一个消息：我的堂妹要结婚了。我的堂妹，在我眼里一直还是那个初高中的样子，还没长大的感觉，怎么就结婚了呢？这同时让我想起上次去上海玩，和我表哥和嫂子一起吃饭的时候，我表哥第一眼见到我，看我和女朋友一起，说话谈吐都不在是他眼中的那个我了，显得非常“吃惊”，就如同看到我堂妹的感觉一样，也许他心里也在想“这小伙子高中还没毕业怎么就谈女朋友了？”。因为大家不是经常见面，所以他们眼中的我也许一直还保留在几年前的那个阶段，同样，正如我眼中的他们。

自从上大学以后，远离家乡，和之前的亲戚朋友沟通都甚少，原来的好朋友还是挺多，虽然都不怎么见面，偶尔在往上聊天扯淡，但是感觉却从未生疏。每年过年回家，还是一样的人员配置，还是一样的活动，热情感觉稍减但是情谊却依然还在。我还是比较相信那个理论：“真正的好朋友，就算长时间不联系，当再次见面时，还是能好若当初”。每个阶段都会有新的好朋友，到了下一个阶段再有新的朋友时，再见到原来的朋友，感觉应该也不外如是。到最后最了解自己的，还是自己。只有自己知道自己经历了什么，有什么变化。突然有个想法，想知道初中同学，高中同学，大学同学，前同事，现同事对我是什么样的印象，是否和我感觉的一个样，改天去调查一下好对自己做一个重新认识。至今还记得以前的同学录上的部分内容，小学的同学录写的是“祝你考上好的初中”，“学习进步”这类非常励志的话；初中则是“你人很聪明，但是你为什么这么黑啊，祝你早日变成黑煤球”，“祝你成为魔兽之王，称霸CS界”（这两段话背景：1、初中时候在家里待不住，成天出去玩，因此晒得特别黑；2、打游戏还比较厉害，在学校还小有点名气，没少被老师罚俯卧撑写检讨）。去年年间同学组织了一个小学同学聚会，我见到了时隔十多年的小学同学，让我感到震惊的是：我觉得除了我自己，其他人都没变，还是原来的样子，该胖的胖，该瘦的瘦，只是变成了放大版，不知道其他人会不会有同样的感觉，因为自己的变化还是自己最清楚。

一个什么样的人

发表于 2013-08-25 分类于 观点 Waline： 本文字数： 1.1k 阅读时长 ≈ 4 分钟

今天偶然跟朋友聊到一点生物的话题，讲到基因杂交的问题时，我想起一道让我非常难忘的生物题目，大概意思是有几只某某显性隐形基因的狗，如何最后得到某某性状的后代。乍一看都觉得这是一道普通的生物题目，但是做这道题时让我感到不快：从基因学角度来讲，有比较简单的方法来得到最后结果，但是这中间会有一个问题：子代与母带杂交。从实际或者伦理方面，我实在是无法接受这样的答案。因为这不是见得最多的豌豆，而是动物，当时考虑了很久，最后在答卷上写了一个非常复杂的过程，这个杂交过程中只会出现同辈，虽然也是有近亲的部分。想想当时一直有这么固执，坚持己见，不会去想这会对我的分数造成什么样的影响。

最后考试结果出来了，班里绝大多数同学这道题都很轻松地得了满分，而我，虽然版面写得很满，也得出了最后的结果，但是因为过程比标准答案的要多出了好几步，因此还扣了好几分。当时胆子比较小，不会去争论分数上的这些问题。事后老师和几个同学都私底下问过我，为什么写得这么复杂，这么简单的分都不要。我跟他们说了一下原因，他们无一例外的哈哈大笑，跟我说”你想多了”。现在回来想想，也许真是我想多了，没有单纯得把他当一道学术题目来做。但是不知道为什么这件事情在我脑海里的印象一直非常深刻，到现在许多年了依然十分清晰。也许出题老师们真的应该多下点功夫，以免遇到像我这样的人，而且这个题目还能对他产生这样的影响，因为这对于他来说可能就不只是一道题目，更像是一个等待被解决的问题。

数据的态度

发表于 2013-08-20 分类于 观点 Waline： 本文字数： 1.4k 阅读时长 ≈ 5 分钟

好久没吐槽了，虽然最近一直比较忙，而且现在也已是凌晨0点30分。但是想起博客好久没更新了，而且正好这段时间一直有在纠结这个问题，而且思考到我们这个工作本身的意义，咬咬牙写出来舒服一点，反正晚睡也不是两三天了。

近些年“数据”这个词越来越热，“大数据”这个概念也被炒得比栗子还熟，但是在这个浮躁的年代，新生事物总是被过度得炒作和夸大从而偏离了其本身的意义。我的理解是，做数据分析，最重要的目的是以目标为导向，发现数据中各种现象的原因和规律，并给出相应的建议以达到更佳的效果。数据再多再复杂，如果你只是看着爽了，玩的开心了，但是用不到实处，那所谓的“大数据”也只能沦为摆设。同样，另一种相同的情况是“数据可视化”，目前相当流行的信息图，着实让看惯了PPT的人眼前一亮，内容也比较多彩绚丽，但是同时也发现许多人数据关没把好，导致了一部分乌龙情况的发生。好的内容用好的方式呈现那叫锦上添花，数据本身不够重视而花大部分时间在展示效果上，这叫舍本逐末。
