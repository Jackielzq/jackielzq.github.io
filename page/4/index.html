“你每天最好相同时间来。”狐狸说，

小王子问：“为什么？”

“比如，你下午四点来，那么从三点起，我就开始感到幸福。时间越临近，我就越感到幸福。我就发现了幸福的价值……所以应当有一定仪式。”

“仪式是什么？”小王子问。

狐狸说，“（仪式）它就是使某一天与其他日子不同，使某一时刻与其他时刻不同。”

对文本文件的操作：输入、输出、查找、替换

命令 说明 备注
echo “abc” 输出 “abc”
echo < “abc” 输出 “abc” 标准输出
echo “abc” > abc.txt 在abc.txt文件中写入“abc”字符
cat abc.txt 输出文件abc.txt的内容
cat abc.txt > xyz.txt 把abc的内容完全写入到xyz（覆盖）
cat abc.txt >> xyz.txt 把abc的内容添加到xyz中（叠加，在abc原始内容后）
cat abc.txt I wc 输出行数、字(单词)数、字符数
cat abc.txt I wc I cat xyz.txt 竖线”I”作为管道，左边为输入，右边为输出
sort abc.txt 按字母顺序输出（行）
uniq abc.txt 文件内去重
grep aa abc.txt 在abc.txt中查找含有aa的行 global regular expression print
grep -i aa abc.txt 在abc.txt中查找含有aa的行，不区分大小写 case insensitive
grep -R aa /user/Documents/abc 在指定目录下查找相应内容aa，显示详细内容 R：recursive递归
grep -Rl aa /user/Documents/abc 在指定目录下查找相应内容aa，只显示文字不显示详情 files with matchs
sed ‘s/abc/xyz’ xyz.txt 在xyz.txt中查找第一个abc并将其替换为xyz ‘s’为sed固定参数
sed ‘s/abc/xyz/g’ xyz.txt 在xyz.txt中查找所有abc并将其替换为xyz ‘g’：global

穷

发表于 2015-05-27 分类于 生活 Waline： 本文字数： 1.1k 阅读时长 ≈ 4 分钟

这两天知乎上有一个比较火的帖子中国真的有很多穷人吗？ ， 重新勾起了之前的一些所见所闻所想。并不是惊于其中挣扎在温饱边缘的社会底层的境地，更多的是觉得这个社会的不公平、信息的不对称，这是一个常态，也必将一直持续下去。但说到归根结底的原因，我认为还是教育。“再穷不能穷教育”这个口号好像接触过好多次，但一直记不起来是在哪见过还是听过，所以有时候觉得，口号这东西，只有在真正真刀真枪干的时候喊，才有意义。

之前我写了一篇关于我高中老师的文章，现在想想，更觉其甚。现在距高一已经将近10年，而这位老师目前还待在他原来的岗位上，矜矜业业地教育一代又一代高中生，但是据我所知，他的薪水也不过4k上下。10年前是1k多，10年以后，工资还没跑的过物价。相比于其他我认为“完全在瞎讲课”的老师，那些人如今多已身居高位，大权在握。平时饭局不断，讲课时红光满面，对了，他们现在已经不用讲课了，而是坐在办公室里吹着空调，偶尔到教室后面看其他老师讲课。他们是一个个的矛盾体：他们本身讲课不行，但职位却升得很快；他们不断地劝说别人的优秀小孩留在本校，而把自己的子女送往外地的知名高中就读，尽管学费可能贵得离谱；他们一直教育学生品德是第一位，却多只关心成绩最好的那几个。穷的真的是教育。因为钱不到位，学校质量不行、老师水平不行，穷的地方教育更加落后，发达的地方老师挤破头，两级分化持续加剧。

相比之下社会方面更加残酷，想象一个，一个富人，如果他有1000万，只放到余额宝，年薪就有40-50万，更不用谈其他投资渠道了，而穷人生活保障都有问题，好不容易存点钱，仅仅能享受定期才5%的年化利率，活期的话那点钱更可以忽略，贫富差距的趋势可见一斑。
