穷

发表于 2015-05-27 分类于 纪录 Waline： 本文字数： 1.1k 阅读时长 ≈ 4 分钟

这两天知乎上有一个比较火的帖子中国真的有很多穷人吗？ ， 重新勾起了之前的一些所见所闻所想。并不是惊于其中挣扎在温饱边缘的社会底层的境地，更多的是觉得这个社会的不公平、信息的不对称，这是一个常态，也必将一直持续下去。但说到归根结底的原因，我认为还是教育。“再穷不能穷教育”这个口号好像接触过好多次，但一直记不起来是在哪见过还是听过，所以有时候觉得，口号这东西，只有在真正真刀真枪干的时候喊，才有意义。

之前我写了一篇关于我高中老师的文章，现在想想，更觉其甚。现在距高一已经将近10年，而这位老师目前还待在他原来的岗位上，矜矜业业地教育一代又一代高中生，但是据我所知，他的薪水也不过4k上下。10年前是1k多，10年以后，工资还没跑的过物价。相比于其他我认为“完全在瞎讲课”的老师，那些人如今多已身居高位，大权在握。平时饭局不断，讲课时红光满面，对了，他们现在已经不用讲课了，而是坐在办公室里吹着空调，偶尔到教室后面看其他老师讲课。他们是一个个的矛盾体：他们本身讲课不行，但职位却升得很快；他们不断地劝说别人的优秀小孩留在本校，而把自己的子女送往外地的知名高中就读，尽管学费可能贵得离谱；他们一直教育学生品德是第一位，却多只关心成绩最好的那几个。穷的真的是教育。因为钱不到位，学校质量不行、老师水平不行，穷的地方教育更加落后，发达的地方老师挤破头，两级分化持续加剧。

相比之下社会方面更加残酷，想象一个，一个富人，如果他有1000万，只放到余额宝，年薪就有40-50万，更不用谈其他投资渠道了，而穷人生活保障都有问题，好不容易存点钱，仅仅能享受定期才5%的年化利率，活期的话那点钱更可以忽略，贫富差距的趋势可见一斑。

这里不是说仇富，也不是抱怨社会的不公，因为它本来就不是平等的，竞争是进步的源泉与动力。我在这里想说的是，在这个物欲横流、金钱至上的年代，我们不妨就直接地谈钱，一旦扯上感情、责任感、使命感、靠谱、踏实…..等乱七八糟的因素，你的雇主就可以站在一个道德的高点俯视你，并以此“正当理由”来压榨你的报酬。所以网上一些人扯的什么“现在的年轻人太浮躁，动不动就跳槽”，“只想着钱，不想着我们在做一个伟大的事情….”，甚至”改变世界“都来了。我觉得现在的90后一代比80后要强，一代比一代要强。信息的发达使他们比我们要成熟地更早，而且更加大胆。在我接触到的不少90后，绝大多数都是又聪明又靠谱的。他们之所以要走，那是因为在这里的价值被低估了，而别的公司却发现了他的价值，也给了他对应的酬劳。你说他现实，你自己不是也一样吗？说这种话的人，如果另外一家公司给你跑出橄榄枝，薪水翻倍，我相信九成以上的人都会蹦跶蹦跶高高兴兴过去上班了，同时心里还会默念着“总算有赏识我这匹千里马的伯乐了”。

在这个很多东西国家都给不了你的情况下，钱能解决很多问题，能给你安全感。所以现在社会的浮躁和金钱至上的观念，是基于这种内心的不安导致的。生个大病怎么办，小孩要读书怎么办，交不起房租怎么办，谈不起女朋友怎么办……理想还是要有的，但是，咱们还是得先把钱挣了。

 

 

 

早起

发表于 2015-05-19 分类于 纪录 Waline： 本文字数： 1.3k 阅读时长 ≈ 5 分钟

自大学毕业，稀里糊涂入广告一行，几乎没有早睡过，更不谈早起。上班比别人晚1小时，代价下班比别人晚2小时+。上班不打卡，加班也是应得的，不用提加班费这个词。这个行业虽然钱不多，但是成长地很快。果不其然，年方26的我，看起来已经像30左右的人了，挣的钱虽然每一次跳槽都翻了一倍左右，但是基数小，泪多不谈。也许真的是时间让人成长，好多以前觉得无所谓的东西，现在开始重视了：比如说外表，比如说健康，比如说兴趣。

看看自己以前的照片再照照镜子里看现在的自己，好像“天生丽质”这个词跟我没什么关系。虽说以前长的还可以，没有脱我们家里的后腿(我爸妈当年用俊男靓女来形容绝对无法反驳)，我姐当年在学校不说TOP3,也得在TOP5之列，追她的男生一大把，马屁拍到我头上也让我得了不少小恩小惠。虽说“身体发肤，受之父母”，但是我认为这句话的有效期间仅仅在于出生到大学毕业。父母在这段时间供你吃穿，使你的基因得到了正常的施展，长成了你本身该有的样子。但是过了这段期间，你的外貌就主要是自己来控制了，不管是表面的保养还是内心的修炼。一年多以前我才开始正式用洗面奶，犹记得第一次用时那真切的感受，感觉在这之前好多年，我的脸上一直套着一层由油脂构成的“天然面膜”，洗掉以后，才算真正触摸到了自己的皮肤。然而一年下来，发现光用洗面奶好像并没什么卵用，于是我想到了调理。

其实说调理不太全面，这包括身体的锻炼和饮食起居的调节。网上查了好多相关的资料，也阅读过一些书籍，幡然醒悟，其实大道至简，许多简单的东西古人已经说了上千年了。“早睡早起，方能养生”，玩过三国杀online的就知道，华佗也确实有这么说过，只是人们在听的时候，也仅当台词去听。所以为什么说“我听过很多道理，但还是过不好这一生”，知易行难。这段时间开启了早睡早起计划，感觉像换了一种人生在过：以前1点睡，即使睡到9点还整天犯困，现在11点半睡，7点起一天精神抖擞，貌似把加班“亏”到的时间给“挣回来了”。以前老觉得晚上9，10点回到家，时间才开始属于自己，不好好上网干干别的，怎么对得起自己。而现在早上7点起来，运动+洗澡+吃早点，也才8点出头，上班以前还可以干好多事，比如洗个衣服，比如像我这样，一边消化刚入胃的早餐，一边端着笔记本胡思乱想……

关于兴趣，可以用我以前提的两个词来形容——“兴趣广泛、无一精湛”。要一个双子座静下来真的很难，总想弄点不一样的东西出来娱乐消遣。有时候不一定是热爱，仅仅是好奇，需要一个体验。没有亲自感受过，怎么知道自己是不是真正的喜欢。从小到大我最大的爱好应该就是玩游戏了吧，其实之所以一直这么喜欢玩，其中一个原因是对于各种游戏我上手都比较快，而且玩的都还不错，这其中带来的满足感有时就像烟瘾一样。虽然我不抽烟，但是我能感受到，过了好久不抽那种心痒痒的感觉。其次是音乐，这个范围比较大，但我如果套用那句经典太台词“我有一个音乐梦想”，估计正在吃面条的人，面条吃进去的一端要从鼻孔出来。对我来说音乐是一种语言，一种纯情感类的语言，就像人群中那一个独一无二的眼神。听到的时候，整个人的心情都不一样。就像眼神，时而坚毅，时而温柔，时而委婉，时而哀怨，不需要多余的辞藻，仅凭内心来感受。如果不是音乐科班出版，静静欣赏就好，听到不喜欢的直接跳到下一个，最好不要多做评论。就如同看电影一样，如果都以文艺片的标准来评价商业片，那样真的太累。

9点多了，隔壁应该起床了，可以去练会儿吉他再去上班。可恶的大横按！

用传统的方式来思考互联网与广告

发表于 2015-04-26 分类于 观点 Waline： 本文字数： 1.6k 阅读时长 ≈ 6 分钟

现在是互联网的时代，全世界都在谈大数据、O2O，但现在看来，绝大多数人都以“纯互联网思维”在看待它，完全把他当成一个新的事物。而我认为，互联网只是传统的一种扩展，是一种介质的延伸，正如以前大家都看报纸知天下，然后电视出现，大行其道，再到后来的互联网，一统天下。从本质上区别这三种方式，其实就是效率的改变：报纸须写稿、审核、排版、印刷，电视需要拍片、剪辑、按时播放，而互联网真正做到了即时，在多种方式的选择上，由于人天生的惰性，毋庸置疑会选择最便捷的哪一种，仅此而已。

因此，有的东西是趋势、是必然，是挡不住的，之前传统的行为方式必然会在互联网上得到印证。我们可以看近些年崛起的互联网巨头，他们分别对应了哪些线下的行为：

之前我们打114、去图书馆查阅资料、或者求助他人答案、带着地图指南针旅行……为的就是查找一个结果，而这些方式都非常不便捷，甚至花了很多的时间也得不到结果，因此出现了google、百度……

之前做生意要自己跑外地看货、选货，下单，再跟着或者运货，因为信息的不对称，也许隔壁市的同一物品就比本市的便宜50%；想买一些特定的东西但求购无门，因此有了亚马逊、阿里巴巴、淘宝……

之前我们写信、明信片，与外地朋友交笔友，与陌生人交流或者分享信息，由于物理条件的限制，效率极其之低下，因此facebook、twitter把全世界人民联系到了一起，当然，被围墙围起来的不算，呵呵

上面提到的获取信息、交易、沟通，是基本的三类行为，在互联网出现之前，他们一样存在，只是形式不一样而已。说的正经一点叫“刚需”，奇怪，每次提起“刚需”这个词，总会想到那类满足人生理需求的网站，这里就不加赘述了，扯一扯接下来要说的，广告。

我觉得现在这个年代广告人急需做的一件事情，就是让大家去了解广告本身：

1、为什么车站里有免费发报纸杂志的，外面卖都1块一张，几块一本？

因为上面印有广告，给大家简单算一下，比方说印一张报纸可能成本2毛钱，我印100万份需要20万，但是我上面印了10个广告，每个广告10万，那么发行这一期我能赚80万，如果今天的报纸没人买、甚至说没人看，那这些广告就相当于白投了，就算我卖了10万份，那么对于报纸本身和广告主，是双输的局面：报纸我印用了20万，卖出去只有10万，那么我亏了10万，而90%的广告没有被人看到，那么广告主亏了90万，而且，下次谁来再来给我的报纸投广告打水漂玩呢？

2、为什么电视老放广告，越好看的节目广告越多？

首先，电视节目是需要成本的，再跟大家一起做个小学数学题：拍一部《还珠》，我投资2000万，收视率超高，广告主都想观众在看《还珠》的时候能看到我的广告，记住我的品牌，买洗发水就买我的blabla，电视剧我相当于免费给你看的，那我拍完就要去睡大街了不是？答案当然是否定的。所以我就在电视剧中间放广告，一个15S的，接收8条，一直持续到整部《还珠》放完，每条收800W，那么我这一个暑假下来，就可以转6400 - 2000 = 4400万，至于其他没多少人看的节目，我就卖便宜些，免得浪费了国家规定可以播放广告的时间。

3、为什么视频网站这么讨厌，放15秒广告也就算了，现在都90秒了？

这就是视频网站最尴尬的地方，不仅内容成本高，而且较之电视，带宽成本也高的离谱，广告呢也卖的更便宜。看看新浪土豆的财报就知道了，就算我从45S延长到90S，顶多也只能从亏损几亿美金到亏损几千万美金，因为广告费仍然支持不了高额的成本。但是我也不能显得我好像不行了的样子，那么好吧，我延长时间少亏点，你们骂我我还是要放的。因为只有把广告放上来，我才能活下去，而我活下去了，你们才能继续享受我免费的视频服务，这一点绝大多数用户都是不知道的。圈内人士很多都不明白，这么浅显的道理大家怎么就不懂了，还老骂我？这个时候你需要跳出行业本身这个圈子，问问你的家乡父老，你就不会奇怪他们为什么会觉得视频网站这么“无赖”了。

其实这上面三个例子对应了三个媒体渠道：报纸、电视、互联网，其本质上都是一样的，广告主为内容买单，而消费者通过接受广告的信息，再去为广告主买单，中间的各个环节都是可以互相对应上的。有个说法“有人在的地方就有江湖”，在这个信息急剧爆炸的年代，“有人在的地方，就有广告”。互联网跟他们一比有安徒生童话里三弟的感觉，老大老二都吃的多，活又干得少，三弟则勤勤恳恳，又要替大哥二哥挨骂又要辛苦干活。但是之后谁更有出息呢？我觉得必然是三弟无疑。

对应上面说的一句话，有些东西的出现是必然的。有时候我也在思考，哪些我们线下的行为方式，可以继续被数字化，被搬移到线上？那是否会是下一个无法阻挡的产品？我在寻找，同时也在期待。

《天才在左，疯子在右》

发表于 2015-03-03 分类于 纪录 Waline： 本文字数： 1.4k 阅读时长 ≈ 5 分钟

最近刚看完了这本比较热门的书，可以用一个词来形容，那就是-脑洞大开。这本书讲述的是作者和数位精神病人的访谈记录以及自己的一些看法，不管其内容真实性如何，但可以肯定的是，它让我对周围，对这个世界有了不一样的看法。对一些事物的理解仅停留在之前受过的教育范畴内，以为一些“常识”就应该“理所当然”，没有用其他的角度来看这些问题，甚至是一些看似荒谬的理论的可能性。这本书让我感觉到了思维的乐趣：

1、通常我们认为蚂蚁是群居动物，一堆工蚁供养着蚁后，维系着这个氏族的延续。但从一个“精神病”的眼里，一个群族才是一个完整的个体，每个工蚁就类似于动物体内的红细胞，他们在外界运送着养分，而蚁后则是大脑，支配着他们的行为，蚂蚁的队伍就像是血液在血管内流动，维持着这个整体的生命，而脱离了这个群体，任何的工蚁都不能独自存活。一种即视的画面感，我突然间被这个画面带过去了，如果说，这是真的呢？

2、植物是否能交流？动物植物都有自己的生命，他们身上也有大大小小的“血管”，就像动物一样，遗憾的是他们不能自由移动。如果他们也能交流，会是用什么方式？如果有，会是怎样的？食人花是一种什么样的存在，既像动物又像植物，如果他可以“吃”昆虫，那么其他的植物是否都具有这种潜力？

3、时间是否可以穿越？宇宙到底有几个？如果时间可以被计算，可以被改变，那世界是个什么样子？有个“疯子”说，世界上有无数个宇宙，每个宇宙每一秒都有无限种可能，在当前的宇宙下，我可能正在打字，但在别的宇宙，我可能正在洗脚，可能正在发呆，可能在玩手机。这让我想到一个电影，叫《预见未来》，主人公能遇见2分钟后发生的事情，在最后有一个镜头，类似于黑客帝国慢动作躲避子弹的场景，对方向其射击，但所有子弹都被他躲过去了，因为其他所有死亡的可能性都被他分配在了别的“平行宇宙”中，那个时间段，其他宇宙的他在经历着各种死法，有的被爆头，有的中一枪，有的中两枪……

4、如果有“平行宇宙”，他们中的同一个个体，有没有相通的可能？如果另外一个宇宙的我可以把他脑子里接受的东西同步到我现在的脑子里，类似于“附体”，那会是一种什么样的景象？我们时不时看到“似曾相识”的画面，所有的人，所有的场景，所有的情节，都和某个时候看到过的一模一样，关键是还能知道下一秒是什么人出现，什么车过来。这种情况不少人都经历过，但是知乎上相关问题完全不能解释我的疑惑，特别是预见到下一秒或者几秒发生的场景，不能用现有的科学来解释，到底实际是怎么样的呢？也许我们还没理解到那个层面吧

4、宇宙到底有多大？是无限扩大还是有边界，如果有，是像《楚门的世界》里那种，打开一扇门外面还有更大的世界，还是说，宇宙就像是一个分子或者原子，里面的星球只是他其中的一些微粒，而他的边界就是他的“外壳”？如果宇宙经历过大爆炸，那么大爆炸之前是什么样子，它是否有边界？他处在的空间是有限的还是无限的？

5、是否有生死轮回？一些人能说出自己前面几世的情况，姓甚名谁，家人名字，年代，还有自己是怎么死的。如果真的有呢？我还真想知道我上辈子是谁。

6、是否有“鬼”？有些人可以通灵，可以看到“鬼”，如果他们真的能看到，我还真想看一看，他们的世界是长什么样子的。

7、玛雅，一个神秘而古老的文明：他们交通发达，但是没有轮子，不是造不出来，相反在玩具中出现了轮子，他们的交通工具，是否真的有“飞行器”；文字中充满了艺术，沟通上却原始而简单；对献祭的崇拜，对天文的精通，是否预示着他们真的附属于另外一个更加强大的文明？所以他们只需要搞搞艺术搞搞献祭就行了，连说话都不用说太多？

这些问题我都没想明白，但是我觉得，这个世界真是神奇而美妙，有好多未知的事情等大家来探索，想想就觉得兴奋。这些“精神病”对事物独特的看法和严谨的思维逻辑，也确实给了我们一个不同的角度来看待问题。至于上面的问题到底是如何？也许以后会有答案，但是对于最后一个问题，我觉得还是多看看书先。
